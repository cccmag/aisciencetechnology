# Sony Discman：隨身音樂的新境界

西元 1984 年，[索尼公司](https://zh.wikipedia.org/wiki/索尼)推出第一代 [Discman](https://zh.wikipedia.org/wiki/Discman)（D-50），這款世界第一台 CD 隨身聽改變了人類聆聽音樂的方式。

## 產品的誕生

Discman 的開發基於索尼在 CD 技術方面的領先地位：

### CD 標準
1980 年，索尼與飛利蒲共同制定了 CD 標準（紅皮書）。

### 技術轉化
索尼工程師將 CD 播放技術微型化，創造出了可隨身攜帶的設備。

### 設計理念
輕、薄、小，適合隨身攜帶；同時保持高品質音頻輸出。

## 技術特點

第一代 Discman（D-50）的規格：

| 項目 | 規格 |
|------|------|
| 尺寸 | 127 × 135 × 25 mm |
| 重量 | 約 500 克 |
| 音頻 | 16 位元，44.1 kHz |
| 播放時間 | 約 5 小時（AA 電池） |
| 防震 | 無（第一代） |

## 技術創新

Discman 的核心技術創新包括：

### 數位音頻
CD 使用數位方式存儲音頻，確保了極高的聲音品質。

### 隨身化
通過精心設計的機械結構和積體電路，實現了 CD 播放器的微型化。

### 抗干擾
數位信號傳輸减少了雜音和失真。

## 市場影響

Discman 的推出帶來了深遠的影響：

### 高品質音頻
消費者首次能夠在移動中享受 CD 品質的音樂。

### 音頻載體轉變
CD 和 Discman 加速了音頻從類比（錄音帶）向數位（CD）的轉變。

### 便攜音頻市場
為日後的 MP3 播放器和智慧手機音頻功能奠定了基礎。

## 歷史意義

Discman 的發明標誌著數位音頻時代的來臨。從 CD 到 MP3，再到串流媒體，人們獲取和聆聽音樂的方式經歷了革命性的變化，而不變的是「隨身音樂」這一核心概念。
