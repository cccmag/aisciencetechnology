# Commodore 64 個人電腦：遊戲與程式設計的結合

西元 1982 年，[Commodore 公司](https://zh.wikipedia.org/wiki/Commodore_International)推出了 [Commodore 64](https://zh.wikipedia.org/wiki/Commodore_64)（C64），這款配備 64KB 記憶體、Advanced Video Chip 的 8 位元個人電腦迅速成為歷史上暢銷的電腦之一，總銷量超過 1700 萬台。

## 產品的誕生

[Commodore 64](https://zh.wikipedia.org/wiki/Commodore_64)的開發始於 1981 年，當時 Commodore 收購了 MOS Technology 公司，獲得了 6510 處理器的使用權。結合自行開發的圖形和聲音晶片，Commodore 於 1982 年推出了這款劃時代的產品。

## 技術規格

Commodore 64 的配置：

| 組件 | 規格 |
|------|------|
| 處理器 | MOS 6510，1 MHz |
| 記憶體 | 64 KB RAM |
| 圖形 | VIC-II，320×200，16色 |
| 聲音 | SID 6581，3 聲道合成 |
| 儲存 | 磁帶/磁碟 |
| 作業系統 | Commodore BASIC 2.0 |

## 技術特點

Commodore 64 的硬體設計極具創新：

### 先進的圖形能力
VIC-II 晶片提供：
- 多種圖形模式（字符模式、點陣模式、滾動模式）
- 硬體精靈（sprites）
- 彈出式視窗效果
- 256 種自定義顏色

### 強大的聲音功能
SID（Sound Interface Device）晶片被認為是 8 位元時代最好的聲音合成器：
- 3 聲道獨立振盪器
- 多種波形（方波、鋸齒、三角、正弦）
- 濾波器效果
- 聲音包絡生成

## 市場成功

Commodore 64 取得了空前的市場成功：

### 銷售數據
- 1982-1993 年間，生產超過 1700 萬台
- 連續多年蟬聯最暢銷個人電腦
- 在某些國家（如英國）市場佔有率超過 40%

### 價格策略
C64 的成功部分歸功於其親民價格。隨著產量增加，價格持續下降，最低曾達到 199 美元。

## 遊戲與教育

Commodore 64 成為遊戲和程式設計的首選平台：

### 遊戲產業
C64 上誕生了大量經典遊戲，如《Fallout》《The Last Ninja》《Turrican》等。其優秀的圖形和聲音表現使其成為遊戲開發的標準平台。

### 程式設計
C64 配備 Commodore BASIC，解釋型 Basic 語言易於學習。許多程式設計師的生涯起步於 C64。

### 黑客與破解
C64 的流行催生了龐大的軟體盜版和破解文化。

## 對後世的影響

Commodore 64 的遺產深遠：

### 遊戲開發
眾多知名遊戲開發者從 C64 起家，將經驗帶入後續的遊戲產業。

### 數位音樂
SID 晶片的聲音合成技術影響了後續的數位音樂創作。

### 價格親民化
C64 展示了如何通過整合硬體和降低成本來普及個人電腦。

## 歷史意義

Commodore 64 不僅是一款成功的產品，更是一個時代的象徵。它代表了 1980 年代個人電腦文化的黃金歲月，影響了整整一代程式設計師和遊戲開發者。
