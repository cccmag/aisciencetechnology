# 萊特飛機：人類首款實用動力飛機

1903 年 12 月 17 日，[萊特兄弟](https://zh.wikipedia.org/wiki/萊特兄弟)的「飛行者一號」完成了人類史上首次有動力、可操控、持續滯空的飛行。這款飛機雖然簡陋，但卻開創了人類航空的新時代。

## 飛行者一號的結構

| 項目 | 規格 |
|------|------|
| 翼展 | 12.3 公尺 |
| 機翼面積 | 47.4 平方公尺 |
| 機身重量 | 約 274 公斤 |
| 最大起飛重量 | 約 340 公斤 |
| 發動機功率 | 12 馬力 |
| 最大飛行時間 | 59 秒 |
| 最大飛行距離 | 260 公尺 |

## 設計特點

### 機翼設計
萊特兄弟使用風洞測試，優化了機翼的升力特性。機翼呈弧形，產生更大的升力。

### 控制系統
採用機翼扭曲設計，解決了飛機滾轉控制的問題。這是萊特兄弟最重要的創新之一。

### 螺旋槳
採用效率更高的雙葉螺旋槳，減少了動力損失。

## 歷史的意義

飛行者一號的成功試飛標誌著：
- 人類動力飛行時代的開始
- 現 代航空工業的起點
- 交通革命的新篇章

此後，萊特兄弟繼續改進設計，陸續推出了飛行者二號、三號，載人飛行時間不斷延長。
