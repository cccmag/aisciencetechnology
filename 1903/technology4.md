# 電照明技術普及：城市夜晚的光明

[愛迪生](https://zh.wikipedia.org/wiki/托馬斯·愛迪生)發明白熾燈泡以來，[電照明](https://zh.wikipedia.org/wiki/電燈)技術持續改進。1903 年，電燈照明在更多城市普及，改變了人類的夜間生活。

## 電照明技術的進展

### 燈泡技術改進
- **燈絲材料**：使用更耐高溫的鎢絲
- **燈泡壽命**：平均壽命延長到 1000 小時以上
- **光效提升**：每瓦特產生更多流明

### 電網建設
- **發電廠**：更大規模的中心發電站
- **輸電線路**：更高電壓的輸電線路
- **配電網**：更完善的配電網絡

## 對城市生活的影響

電照明技術的普及深刻改變了城市面貌：

### 夜晚的延長
- 工廠可以夜間生產
- 商店營業時間延長
- 城市夜生活興起

### 安全改善
- 街道照明減少犯罪
- 交通事故減少
- 夜間出行更加安全

### 經濟發展
- 服務業營業時間延長
- 休閒娛樂業發展
- 經濟活動時間延長

## 1903 年的應用

1903 年，電照明在以下領域得到推廣：

| 領域 | 應用 |
|------|------|
| 家庭 | 室內電燈逐漸普及 |
| 商業 | 商店、工廠使用電燈 |
| 公共 | 街道、路燈使用電燈 |
| 工業 | 工廠車間照明改善 |

## 歷史的連結

從 1903 年的[電燈](https://zh.wikipedia.org/wiki/電燈)到今天的[LED 照明](https://zh.wikipedia.org/wiki/發光二極體)：

- 1879：愛迪生白熾燈泡
- 1903：鎢絲燈泡普及
- 1938：螢光燈發明
- 1962：發光二極體（LED）
- 2010：智慧照明系統

電照明技術的進步讓我們的夜晚更加明亮，也更加節能環保。
