# 歷史穿梭：1903 年的科學、技術與未來

**科學技術年鑑　1903 年號**

編輯：[陳鍾誠](https://csie.nqu.edu.tw/p/404-1038-2377.php?Lang=zh-tw)

---

西元 1903 年，是科學與技術史上的一個重要里程碑。在這一年，我們看到了人類首次實現有動力、可操控的飛行，這一成就的影響延續至今，並繼續塑造著人類的未來。

## 一、1903 年的科學連結

### 1.1 萊特兄弟首次飛行 → 航空時代 → 太空探索

1903 年 12 月 17 日，[萊特兄弟](https://zh.wikipedia.org/wiki/萊特兄弟)在北卡羅來納州基蒂霍克實現了人類史上首次有動力、可操控、持續滯空的飛行。這短短 12 秒的飛行，開創了人類交通的新紀元。

**歷史的連結：**

| 年份 | 事件 | 意義 |
|------|------|------|
| 1903 | 萊特兄弟首次飛行 | 人類航空時代開始 |
| 1914 | 商業航班首航 | 航空客運服務 |
| 1947 | 突破音障 | 超音速飛行 |
| 1969 | 阿波羅 11 號登月 | 人類登陸月球 |
| 2020 | SpaceX 載人飛行 | 商業太空旅遊 |

從 12 秒到載人登月，再到未來的商業太空旅遊，這是人類飛行夢想的持續實現。

### 1.2 貝克勒爾放射性研究 → 核物理學 → 核能時代

[貝克勒爾](https://zh.wikipedia.org/wiki/亨利·貝克勒爾)在 1896 年發現的[放射性](https://zh.wikipedia.org/wiki/放射性)現象，為人類打開了原子核物理學的大門。

**歷史的連結：**

- **1896**：貝克勒爾發現放射性
- **1938**：哈恩和斯特拉斯曼發現核分裂
- **1942**：費米建成第一座核反應堆
- **1945**：原子彈在廣島和長崎使用
- **1954**：第一座商業核電廠建成
- **2020**：全球約 400 座核反應堆運行

### 1.3 阿瑞尼烏斯電離理論 → 電化學 → 鋰離子電池

[阿瑞尼烏斯](https://zh.wikipedia.org/wiki/斯凡特·奧古斯特·阿瑞尼烏斯)的[電離理論](https://zh.wikipedia.org/wiki/電離)為現代電化學奠定了基礎。

**歷史的連結：**

- **1884**：阿瑞尼烏斯提出電離理論
- **1890**：首個實用蓄電池發明
- **1991**：鋰離子電池商業化
- **2020**：電動汽車和行動裝置廣泛使用鋰離子電池

### 1.4 芬森光療 → 光動力療法 → 癌症治療

[芬森](https://zh.wikipedia.org/wiki/尼爾斯·呂伯格·芬森)發明的[光療](https://zh.wikipedia.org/wiki/光療)方法開創了光醫學的先河。

**歷史的連結：**

- **1903**：芬森發明光療方法
- **1960**：雷射用於醫療
- **1990**：光動力療法開始用於癌症治療
- **2020**：光動力療法在多種癌症治療中應用

## 二、1903 年的技術連結

### 2.1 萊特飛機 → 商業航空 → 太空運輸

萊特兄弟的[飛行者一號](https://zh.wikipedia.org/wiki/飛行者一號)開創了飛機時代。

**技術演進：**

| 階段 | 特點 | 典型應用 |
|------|------|----------|
| 1903 | 動力飛行 | 萊特飛機 |
| 1920 | 商業客運 | 早期航班 |
| 1970 | 噴射客機 | 波音 747 |
| 2020 | 電動飛機 | 實驗性項目 |

### 2.2 收音機技術 → 廣播電視 → 網際網路

1903 年的[收音機技術](https://zh.wikipedia.org/wiki/收音機)為日後的廣播和通信奠定了基礎。

**技術演進：**
- 1903：收音機開始普及
- 1920：商業廣播電台
- 1950：電視機普及
- 1990：網際網路興起
- 2020：5G 和物聯網

### 2.3 內燃機改進 → 汽車工業 → 電動汽車

1903 年的[內燃機](https://zh.wikipedia.org/wiki/內燃機)技術進步為汽車工業的發展奠定了基礎。

**技術演進：**
- 1903：內燃機改進
- 1908：福特 T 型車
- 1913：流水線生產
- 1990：混合動力車
- 2020：純電動汽車

### 2.4 電照明普及 → 智慧照明 → 節能環保

1903 年的[電照明](https://zh.wikipedia.org/wiki/電燈)技術改變了人類的夜間生活。

**技術演進：**
- 1879：愛迪生白熾燈
- 1903：鎢絲燈普及
- 1938：螢光燈
- 1962：LED
- 2020：智慧照明系統

## 三、1903 年的產品連結

### 3.1 萊特飛機 → 商用客機 → 私人飛機

從[萊特飛機](https://zh.wikipedia.org/wiki/萊特兄弟)到現代客機：

- 1903：飛行者一號
- 1920：Ford Trimotor 商業客機
- 1970：波音 747 大型客機
- 2020：波音 787 夢想客機

### 3.2 收音機 → 電視機 → 智慧手機

從[收音機](https://zh.wikipedia.org/wiki/收音機)到[智慧手機](https://zh.wikipedia.org/wiki/智慧手機)：

- 1903：收音機商業化
- 1927：電視機發明
- 1973：行動電話
- 2007：iPhone 智慧手機

### 3.3 汽車 → 電動汽車 → 自動駕駛

從[汽車](https://zh.wikipedia.org/wiki/汽車)到[自動駕駛汽車](https://zh.wikipedia.org/wiki/自動駕駛汽車)：

- 1903：福特等汽車公司成立
- 1908：福特 T 型車大眾化
- 1990：混合動力車
- 2020：特斯拉等電動汽車

## 四、結論：創新的長河

1903 年的科學發現和技術發明，猶如歷史長河中的重要支流，最終匯入了今日科技革命的汪洋大海。

從萊特兄弟的 12 秒飛行，到今天的洲際噴射客機和商業太空飛行；從貝克勒爾發現的放射性，到今天的核能發電和醫學成像；從阿瑞尼烏斯的電離理論，到今天的鋰離子電池——這些科學和技術的進步，构成了一条貫穿一百二十餘年的創新鏈條。

這條創新鏈條告訴我們：
1. 基礎科學是技術創新的源泉
2. 技術進步是累積的
3. 應用會反哺科學
4. 跨領域連結是創新的關鍵

讓我們銘記 1903 年的這些成就，並繼續在科學和技術的道路上前進。
