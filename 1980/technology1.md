# IBM PC發布：個人電腦標準

1981 年，[IBM](https://zh.wikipedia.org/wiki/IBM)發布了[IBM PC](https://zh.wikipedia.org/wiki/IBM_PC)（IBM個人電腦），但在 1980 年，IBM 已經秘密開發這款產品，並在 1980 年做出了開放架構的關鍵決定，這一決定在未來徹底改變了個人電腦產業。

## IBM PC的開發背景

### 開發歷程

[IBM](https://zh.wikipedia.org/wiki/IBM)在 1980 年組建了「臭鼬工廠」團隊來開發個人電腦。這個團隊在短短一年內就開發出了 IBM PC。

### 開放架構的決定

1980 年，IBM 做出了歷史性的決定——採用開放架構：
- **開放的匯流排設計**
- **標準的元件供應商**
- **公開的技術文檔**
- **協力廠商相容性**

這與當時其他電腦公司封閉的做法截然不同。

## 技術規格

### IBM PC的硬體配置

[IBM PC](https://zh.wikipedia.org/wiki/IBM_PC)的主要規格：

| 項目 | 規格 |
|------|------|
| 處理器 | 英特爾 8088，4.77 MHz |
| 記憶體 | 16 KB - 256 KB |
| 顯示器 | MDA 或 CGA |
| 軟碟 | 5.25 英吋軟碟 |
| 作業系統 | PC DOS 1.0 |

### 開放式設計

IBM PC 的設計特點：
1. **ISA匯流排**：允許第三方擴充卡
2. **標準電源**：ATX電源標準的前身
3. **BIOS設計**：簡單的開機程式

## 軟體生態

### 作業系統

[IBM PC](https://zh.wikipedia.org/wiki/IBM_PC)最初採用[PC DOS](https://zh.wikipedia.org/wiki/MS-DOS)（後來稱為 MS-DOS）。

### 應用軟體

圍繞 IBM PC 形成了龐大的軟體生態：
- **Lotus 1-2-3**：電子試算表
- **WordStar**：文字處理
- **dBASE**：資料庫
- **MIDI軟體**：電腦音樂

## 對產業的影響

### 個人電腦革命

[IBM PC](https://zh.wikipedia.org/wiki/IBM_PC)的發布標誌著[個人電腦革命](https://zh.wikipedia.org/wiki/個人電腦)的真正開始。

### 產業結構變化

| 變化 | 意義 |
|------|------|
| 開放標準 | 良性競爭 |
| 協力廠商硬體 | 產業規模化 |
| 軟體產業 | 新的商業模式 |
| 相容機 | 市場多元化 |

## 歷史的意義

### 開放戰略的成功

1980 年的開放架構決定，改變了整個電腦產業的命運。

### 技術發展長鏈

從 1980 年的開放架構到今天的開放平台：

| 年份 | 發展 | 意義 |
|------|------|------|
| 1980 | IBM PC開放架構 | 產業變革 |
| 1983 | IBM PC XT | 標準確立 |
| 1987 | IBM PS/2 | 最後的輝煌 |
| 1990 | IBM相容機主流 | Wintel聯盟 |
| 2020 | 開放硬體 | RISC-V生態 |

[IBM PC](https://zh.wikipedia.org/wiki/IBM_PC)的開放架構決定是二十世紀最重要的商業決策之一，它催生了整個個人電腦產業，讓電腦從少數人的工具變成了千家萬戶的日常設備。
