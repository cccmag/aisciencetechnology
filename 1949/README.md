## 本期編輯語

1949 年是冷戰加劇的年份。這一年，蘇聯成功試爆了第一顆原子彈，打破了美國的核壟斷。這個事件預示著核軍備競賽的開始，也加劇了東西方之間的緊張關係。

同年，湯川秀樹成為首位獲得諾貝爾物理學獎的日本人，標誌著日本科學開始走向世界舞台。這是科學無國界的最好證明。

本期雜誌將帶您回顧 1949 年的這些重要事件，以及它們如何影響了未來的科技發展。
