# 宇宙微波背景輻射的精確測量：宇宙年齡的確定

2002 年，美國 NASA 的[威爾金森微波各向異性探測器](https://zh.wikipedia.org/wiki/威爾金森微波各向異性探測器)（WMAP）發布了首批宇宙微波背景輻射的精確觀測數據，確定宇宙年齡約為 137 億年，開創了[宇宙學](https://zh.wikipedia.org/wiki/宇宙學)的精確時代。

## 宇宙微波背景輻射

### 發現歷史

[宇宙微波背景輻射](https://zh.wikipedia.org/wiki/宇宙微波背景輻射)（CMB）是1965 年被發現的：
- 1965 年：[彭齊亞斯](https://zh.wikipedia.org/wiki/阿諾·彭齊亞斯)和[威爾遜](https://zh.wikipedia.org/wiki/羅伯特·威爾遜)發現 CMB
- 1992 年：[COBE](https://zh.wikipedia.org/wiki/宇宙背景探測者)衛星測量了 CMB 的溫度漲落
- 2001 年：WMAP 衛星發射
- 2003 年：WMAP 發布首批精確數據（部分早期數據在 2002 年可得）

### 理論意義

CMB 的意義：
- 是大爆炸理論的最有力證據
- 包含了宇宙早期的信息
- 決定了宇宙的基本參數

## WMAP 探測器

### 任務目標

[威爾金森微波各向異性探測器](https://zh.wikipedia.org/wiki/威爾金森微波各向異性探測器)（WMAP）：
- 測量 CMB 的溫度漲落
- 確定宇宙的幾何形狀
- 測量宇宙的組成
- 確定宇宙年齡

### 觀測波段

WMAP 的觀測：
- 觀測波段：23-94 GHz
- 測量精度比 COBE 高 45 倍
- 全天巡天觀測
- 持續觀測至 2010 年

## 宇宙參數的測定

### 2003 年的重要發現

WMAP 的首批結果確定了以下參數：

| 參數 | 數值 |
|------|------|
| 宇宙年齡 | 137 ± 1 億年 |
| 宇宙組成：暗能量 | 73% |
| 宇宙組成：暗物質 | 23% |
| 宇宙組成：普通物質 | 4% |
| 哈伯常數 | 71 ± 4 km/s/Mpc |
| 宇宙幾何 | 接近平坦 |

### 宇宙的組成

WMAP 揭示的宇宙組成：
- **暗能量（73%）**：推動宇宙加速膨脹的神秘力量
- **暗物質（23%）**：看不見但有引力的物質
- **普通物質（4%）**：我們熟悉的一切

## 對宇宙學的影響

### 宇宙學的精確時代

WMAP 的數據標誌著宇宙學從猜測走向精確測量：
- 不再依賴單一證據
- 多個獨立的宇宙學觀測相互印證
- 宇宙學成為一門精密科學

### 確認暗能量

WMAP 為暗能量的存在提供了獨立證據：
- CMB 與超新星觀測一致
- 宇宙的幾何形狀是平坦的
- 這需要暗能量的存在來解釋

## 宇宙年齡的確定

### 測定方法

確定宇宙年齡的方法：
1. 測量哈伯常數（宇宙膨脹速率）
2. 測量宇宙的物質密度
3. 計算宇宙的膨脹歷史
4. 確定宇宙開始膨脹的時間

### 137 億年

確定的宇宙年齡：
- 137 億年（± 1 億年）
- 比之前估計的更精確
- 與地質年齡和恆星年齡一致
- 成為宇宙學的標準數值

## 與其他觀測的比較

### 恆星年齡

宇宙年齡必須比其中恆星更老：
- 最古老的恆星年齡約 130-140 億年
- WMAP 數據與之一致
- 解決了之前宇宙年齡低於恆星年齡的矛盾

### 大爆炸理論

CMB 觀測確認了大爆炸理論：
- CMB 的存在和溫度
- CMB 的各向異性
- 與理論預言高度一致

## 科學意義

### 基礎物理學

WMAP 數據對基礎物理學的影響：
- 支持暗能量的存在
- 為宇宙暴脹理論提供證據
- 挑戰現有的引力理論和量子力學

### 宇宙演化

對宇宙演化的理解：
- 宇宙從大爆炸開始
- 經歷了暴漲時期
- 形成了我們今天看到的結構

## 後續發展

| 年份 | 發展 |
|------|------|
| 2003 | WMAP 第一年數據發布 |
| 2006 | WMAP 三年數據 |
| 2009 | 普朗克衛星發射 |
| 2013 | 普朗克首批數據 |
| 2018 | 普朗克最終數據 |

## 歷史地位

WMAP 觀測的歷史意義：
- **開創精確宇宙學**：從猜測到精確測量
- **確定宇宙基本參數**：宇宙年齡、組成、幾何
- **確認大爆炸理論**：CMB 是最有力的證據
- **為暗能量提供證據**：與超新星觀測一致
