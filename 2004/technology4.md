# RSS 和部落格的流行：Web 2.0 時代的來臨

2004 年，[RSS](https://zh.wikipedia.org/wiki/RSS) 訂閱和[部落格](https://zh.wikipedia.org/wiki/部落格格)（Blog）開始廣泛流行，標誌著 [Web 2.0](https://zh.wikipedia.org/wiki/Web_2.0) 時代的來臨。用戶不再只是內容的消費者，也成為內容的創作者。

## Web 2.0 的概念

### 什麼是 Web 2.0

[Web 2.0](https://zh.wikipedia.org/wiki/Web_2.0) 的核心概念：
- **用戶生成內容**：UGC 的興起
- **社交互動**：用戶之間的連接
- **開放分享**：知識共享
- **平台效應**：網絡效應

### 與 Web 1.0 的比較

| 方面 | Web 1.0 | Web 2.0 |
|------|----------|---------|
| 內容來源 | 少數機構 | 多數用戶 |
| 互動性 | 有限 | 豐富 |
| 參與方式 | 閱讀為主 | 閱讀+創作 |
| 技術特點 | 靜態網頁 | 動態應用 |

## RSS 技術

### RSS 的用途

[RSS](https://zh.wikipedia.org/wiki/RSS)（Really Simple Syndication）的作用：
- **內容訂閱**：用戶訂閱感興趣的內容
- **信息聚合**：將多個來源整合
- **更新通知**：新內容自動提醒
- **脫機閱讀**：離線也能閱讀

### RSS 閱讀器

 RSS 生態的組成：
- **Google Reader**（已停用）
- **Bloglines**
- **Netvibes**
- **Feedly**（後續崛起）

## 部落格的興起

### 部落格平台

2004 年的主流[部落格](https://zh.wikipedia.org/wiki/部落格)平台：
| 平台 | 特點 |
|------|------|
| Blogger | Google 旗下，易用 |
| WordPress | 開源，可定制 |
| Movable Type | 早期流行 |
| LiveJournal | 社交功能 |

### 部落格類型

 部落格的多樣化：
- **個人日記**：記錄生活
- **專業部落格**：科技、財經評論
- **組織部落格**：企業和非營利組織
- **新聞部落格**：傳統媒體補充

## 對媒體的影響

### 公民新聞

部落格對新聞業的影響：
- **人人都是記者**：任何人都可以發布新聞
- **主流媒體補充**：提供不同角度
- **調查性報導**：揭露醜聞（如爆料）
- **即時性**：比傳統媒體更快

### 媒體轉型

傳統媒體的應對：
- 主流媒體建立自己的部落格
- 吸納部落格作者
- 數位化轉型
- 與社群媒體結合

## 對社會的影響

### 表達自由

部落格對言論的影響：
- **低成本發聲**：任何人都能發布
- **突破審查**：在某些國家
- **社群形成**：同好聚集
- **話語權分散**：挑戰主流媒體

### 知識分享

部落格對知識的影響：
- **技術文檔**：教程和技巧
- **學術交流**：預印本和評論
- **生活經驗**：分享和借鑒
- **百科全書**：維基百科興起

## 歷史意義

部落格和 RSS 流行的歷史意義：
- **內容創作平民化**：從機構到個人
- **信息傳播革命**：更快更廣
- **Web 2.0 的開端**：開創互聯網新時代
- **為社群媒體奠基**：Facebook、Twitter 先驅
