# Microsoft Surface：微軟的硬體嘗試

## 引言

西元 2012 年六月，微軟發布了自有品牌的[Surface 平板電腦](https://zh.wikipedia.org/wiki/Microsoft_Surface)。這是微軟首次以自有品牌推出硬體產品，標誌著這家軟體巨頭戰略轉型的重要一步。

## 產品線

[微軟](https://zh.wikipedia.org/wiki/微軟)推出了兩個版本的 Surface：

### Surface RT
- **作業系統**：Windows RT（專為 ARM 處理器設計的 Windows 8 分支）
- **處理器**：NVIDIA Tegra 3 四核處理器
- **特點**：輕薄、長續航、僅相容 Windows Store 應用
- **價格**：499 美元起

### Surface Pro
- **作業系統**：完整版 Windows 8 Pro
- **處理器**：英特爾 Core i5 處理器
- **特點**：可以運行所有傳統 Windows 應用
- **價格**：899 美元起

## 設計特點

[Surface](https://zh.wikipedia.org/wiki/Microsoft_Surface) 的設計結合了平板電腦和傳統筆電的特點：

### 內建支架
Surface RT 和 Surface Pro 都配備了內建一體式支架，可以將平板電腦立在桌面上，方便觀看影片或進行演示。

### Touch Cover 和 Type Cover
微軟推出了專屬的鍵盤保護套：
- **Touch Cover**：電容式觸控鍵盤，厚度僅 3mm
- **Type Cover**：傳統物理按鍵鍵盤，打字手感更好

### VaporMg 外殼
採用鎂合金一體成型的「VaporMg」外殼，堅固耐用，質感出色。

### 手寫筆支援
Surface Pro 支援 1024 級壓感的手寫筆，適合筆記和繪圖。

## 市場表現

[Surface](https://zh.wikipedia.org/wiki/Microsoft_Surface) 的市場表現並不如預期：

### 銷量低於預期
Surface RT 的銷量低於微軟的內部預期。據報導，截至 2013 年初，Surface RT 的總銷量約為 150-200 萬台。

### 庫存問題
微軟低估了市場需求，一度出現供應短缺；但後來又面臨庫存積壓問題。

### Windows RT 的困境
Windows RT 生態系統發展緩慢，缺乏應用支持，消費者對這個系統的接受度不高。

## 策略意義

雖然[Surface](https://zh.wikipedia.org/wiki/Microsoft_Surface) 的銷量未如預期，但這款產品對微軟具有重要的策略意義：

### 生態系統整合
Surface 展示了 Windows 8 的完整功能，特別是鍵盤+觸控的混合使用體驗。

### 樹立標杆
Surface 為其他 OEM 廠商樹立了 Windows 8 設備的標杆，帶動了整個 Windows 平板市場的發展。

### 硬體能力展示
微軟透過自有硬體，展示了 Windows 平台的可能性，激勵了整個產業鏈的創新。

## 後續發展

[微軟](https://zh.wikipedia.org/wiki/微軟)在 Surface 產品線上持續投入：

### Surface 2 和 Surface Pro 2
2013 年，微軟推出了升級版本，採用更快的處理器和更好的顯示屏。

### Surface Pro 3
2014 年，微軟推出了更大螢幕（12 英吋）、更輕薄的 Surface Pro 3，開始蠶食傳統筆電市場。

### Surface Book
2015 年，微軟進一步推出了首款筆記型電腦 Surface Book，擴大了產品線。

### 持續虧損到盈利
經過幾代的努力，Surface 產品線從持續虧損轉變為盈利，成為微軟重要的營收增長點。
