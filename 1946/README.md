## 本期編輯語

1946 年是人類歷史上重要的轉折點。第二次世界大戰剛剛結束，全世界正在經歷深刻的變革。在科學技術領域，這一年發生了足以改變人類未來走向的重大事件：世界上第一台通用電子計算機 ENIAC 在美國問世，標誌著資訊革命的開始；與此同時，電晶體的發明也在籌備之中，為半導體時代拉開序幕。

本期雜誌將帶您回顧 1946 年的科學發現、技術發明、重要產品，以及諾貝爾獎得主的貢獻。通過「歷史穿梭」專題，我們將探索這些事件如何連接過去與未來，形成今日科技世界的基礎。
