# 生理醫學獎：穆勒與輻射遺傳學

1946 年諾貝爾生理醫學獎授予美國遺傳學家赫爾曼·約瑟夫·穆勒（Hermann Joseph Muller，1890-1967），以表彰他發現基因突變可以由 X 射線誘導的傑出工作。穆勒的發現不僅揭示了遺傳變異的秘密，還為理解輻射對生物的影響提供了科學依據，對後世的放射生物學、輻射防護和基因工程技術都有深遠影響。

## 一、果蠅研究先驅

1890 年，赫爾曼·穆勒出生於美國紐約市。他在哥倫比亞大學學習期間，接觸到了托馬斯·摩根（Thomas Morgan）的果蠅遺傳學研究。果蠅成為穆勒畢生研究的對象。

穆勒的研究方法與眾不同。他設計了一套可以在實驗室裡大量繁殖和觀察果蠅的系統，使得遺傳學研究從「等待偶然」變成了「系統分析」。

1920 年代，穆勒在德克薩斯大學任教期間，開始研究基因突變的機制。當時，科學家已經知道基因是遺傳的單位，但對基因如何變化（即突變）卻知之甚少。

## 二、X 射線誘導突變

突變是進化的原材料。如果沒有突變，生物就不能進化，適者生存的過程也就無法進行。但在穆勒之前，科學家從未能在實驗室裡誘導出突變。

1927 年，穆勒發表了他的突破性發現：用 X 射線照射果蠅，可以大幅增加突變的發生率。這個發現轟動了整個生物學界。

穆勒的實驗設計非常精妙：
- 將一批果蠅暴露在 X 射線下
- 照射後的果蠅與正常果蠅交配
- 觀察後代的性狀變化

實驗結果令人吃驚：經 X 射線照射的果蠅，其後代的突變率比對照組高了數百倍。這意味著，X 射線可以直接損傷遺傳物質，導致基因突變。

## 三、發現的深遠意義

穆勒的發現有幾個層面的重要意義：

遺傳學層面：這個發現首次證明了基因是可以被人為改變的。這為遺傳學研究提供了強大的工具——科學家不再只能等待自然的突變，而可以通過輻射「定向製造」突變。

進化論層面：突變是進化的原材料。穆勒的工作加深了我們對進化機制的理解，表明環境因素（如輻射）可以影響突變率，從而影響進化速度。

醫學層面：穆勒的發現揭示了輻射對生物的遺傳危害。這對於放射科醫生、X 射線技術員和核工業工作者有重要的健康指導意義。

## 四、社會影響與歷史背景

穆勒的發現發表於 1927 年，當時正是遺傳學發展的關鍵時期。這個發現與同一時期其他遺傳學突破一起，奠定了現代遺傳學的基礎。

然而，這個發現的社會影響是雙面的：

一方面，它促進了放射防護意識的建立。人們開始意識到 X 射線和放射性物質對人體的危害，開始制定安全標準保護工人和患者。

另一方面，它也曾被錯誤應用。1940 年代，有些國家進行了不道德的輻射人體實驗。穆勒本人對這些行為強烈反對，成為核時代重要的科學倫理聲音。

## 五、穆勒的其他貢獻

除了 X 射線誘導突變的發現，穆勒在遺傳學領域還有其他重要貢獻：

基因與環境：穆勒強調基因不是命運的完全決定因素，環境和生活方式也會影響性狀的表現。這種「基因與環境交互作用」的思想，對於理解複雜疾病非常重要。

優生學批評：穆勒是一個強烈的優生學批評者。他認為優生學的政策和實踐是危險的，往往被用來為歧視和迫害辯護。他主張通過改善環境和教育來提高人口素質，而非通過強制手段。

冷戰立場：穆勒是一個坚定的和平主義者。1955 年，他參與了「羅素-愛因斯坦宣言」，呼籲各國放棄核武器。

## 六、遺產與傳承

穆勒開創的遺傳學方法，直接影響了後世的基因工程技術。今天科學家使用的定點誘變技術，其原理與穆勒的 X 射線誘導突變一脈相承，只不過更加精確和可控。

1967 年，穆勒在印第安納大學的實驗室裡去世，享年 77 歲。他的一生經歷了遺傳學從默默無聞到成為科學皇冠的歷程，他的工作為這一切做出了重要貢獻。
