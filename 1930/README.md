## 本期內容導覽

1930 年是人類歷史上重要的一年。這一年，[印度](https://zh.wikipedia.org/wiki/印度)物理學家[錢德拉塞卡拉·拉曼](https://zh.wikipedia.org/wiki/錢德拉塞卡拉·拉姆金）因其對光散射的研究，榮獲[諾貝爾物理學獎](https://zh.wikipedia.org/wiki/諾貝爾物理學獎)，這是亞洲科學家首次獲得諾貝爾物理學獎。同時，這也是大蕭條最嚴重的一年，經濟危機深刻影響了全球。
