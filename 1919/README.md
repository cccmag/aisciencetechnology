## 重要歷史背景

1919 年是科學史上的一個重要年份。這一年，英國物理學家愛丁頓率領的觀測隊在日全食期間成功觀測到了星光在太陽引力場中的偏折，證實了愛因斯坦廣義相對論的預言。這一事件轟動了全世界，將愛因斯坦從一位理論物理學家變成了公眾偶像。同時，這也標誌著人類對宇宙的認識進入了一個新的時代。
