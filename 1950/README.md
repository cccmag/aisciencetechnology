## 本期編輯語

1950 年是冷戰正式形成的年份。兩年前的杜魯門主義和馬歇爾計畫已經確立了東西方對抗的基本格局，而這一年爆發的韓戰更是將這種對抗推向了一個新的階段。

在科學技術領域，這一年標誌著電晶體商業化的開始。貝爾實驗室開始向其他公司授權電晶體技術，半導體產業的時代正式拉開序幕。

本期雜誌將帶您回顧 1950 年的科學發現、技術發明和重要產品，以及它們對未來的影響。
