# X射線結晶學的興起：揭示晶體結構

## 概述

1912 年，[馬克斯·馮·勞厄](https://zh.wikipedia.org/wiki/馬克斯·馮·勞厄)在慕尼黑大學發現了 X 射線通過晶體時的衍射現象，開創了[X射線結晶學](https://zh.wikipedia.org/wiki/X射線晶體學)這一新學科。1913 年，這一發現迅速成為物理學和化學研究的前沿領域。

## 一、X射線的發現與性質

### 1.1 倫琴的發現

1895 年，[威廉·康拉德·倫琴](https://zh.wikipedia.org/wiki/威廉·康拉德·倫琴)發現了 X 射線。這種未知性質的射線能夠穿透物質，使包圍在黑紙中的底片感光。

### 1.2 X射線的本質

1905 年，實驗證實 X 射線是一種電磁波，其波長比可見光短得多。這為 X 射線應用於微觀結構研究提供了理論基礎。

### 1.3 X射線的產生

X 射線管是產生 X 射線的主要設備。高速電子撞擊金屬靶時，會產生連續譜 X 射線和特徵 X 射線。

## 二、勞厄的突破性發現

### 2.1 實驗設計

1912 年，勞厄提出一個大膽的假設：如果 X 射線是一種電磁波，而且波長與晶體中原子間距相當，那麼晶體應該能夠作為 X 射線的天然衍射光柵。

### 2.2 實驗結果

在兩位研究生弗里德里希和尼平（Friedrich and Knipping）的協助下，勞厄用硫酸銅晶體進行實驗，成功觀察到了 X 射線的衍射圖案。這一實驗結果發表於 1912 年。

### 2.3 諾貝爾獎的認可

1914 年，勞厄因其 X 射線晶體衍射的發現榮獲諾貝爾物理學獎。

## 三、1913 年的發展

### 3.1 布拉格父子

[威廉·亨利·布拉格](https://zh.wikipedia.org/wiki/威廉·亨利·布拉格)和[威廉·勞倫斯·布拉格](https://zh.wikipedia.org/wiki/威廉·勞倫斯·布拉格)父子在 1912-1913 年發展了 X 射線晶體結構分析的方法，提出了著名的布拉格定律：

$$n\lambda = 2d\sin\theta$$

### 3.2 晶體結構分析

1913 年，布拉格父子用 X 射線衍射方法確定了氯化鈉（食鹽）的晶體結構，證實了離子化合物的存在。

### 3.3 金屬結構研究

X 射線衍射也開始應用於金屬和合金的晶體結構研究，為金屬物理學的發展提供了重要工具。

## 四、科學意義

### 4.1 原子結構的揭示

X 射線結晶學使人科學家能夠「看到」原子在晶體中的排列方式，這是微觀結構研究的一次革命。

### 4.2 化學鍵的新認識

X 射線晶體學提供了化學鍵和分子結構的直接證據，幫助化學家理解化合物的微觀結構。

### 4.3 跨學科應用

X 射線結晶學迅速應用於物理學、化學、礦物學、生物學等多個學科。

## 五、歷史影響

### 5.1 晶體學的建立

X 射線結晶學的建立標誌著現代晶體學的開始。

### 5.2 DNA 結構的揭示

1953 年，[沃森](https://zh.wikipedia.org/wiki/詹姆斯·沃森)和[克里克](https://zh.wikipedia.org/wiki/弗朗西斯·克里克)正是基於 X 射線晶體學數據，才成功確定了 DNA 的雙螺旋結構。

### 5.3 蛋白質結構研究

X 射線晶體學在蛋白質和其他生物大分子的結構研究中發揮了關鍵作用。
