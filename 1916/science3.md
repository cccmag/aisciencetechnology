# 量子理論的新進展

## 舊量子論的巅峰

1916 年，量子理論正在經歷從舊量子論向新量子力學過渡的關鍵時期。馬克斯·普朗克、尼爾斯·玻爾、阿諾德·索莫非等科學家正在努力擴展量子概念，但真正革命性的突破還要等待十年。

## 一、量子概念的本質

量子理論的核心是「量子化」——某些物理量只能取離散的值，而不是連續變化。

**能量的量子化**：
普朗克在 1900 年提出，諧振子的能量只能是 hν 的整數倍。

**角動量的量子化**：
玻爾在 1913 年提出，電子繞原子核運動的角動量只能是 ℏ 的整數倍。

**量子數**：
描述量子化系統需要使用整數或半整數——量子數。

## 二、對應原理

1916-1918 年間，玻爾發展了對應原理（Correspondence Principle）。

**核心思想**：
量子理論在極限情況下應該與經典物理學一致。具體來說，當量子數很大時，量子理論的預言應該與經典理論相同。

**物理意義**：
對應原理幫助物理學家在沒有完整量子力學的情況下，猜測正確的量子規則。它也是檢驗新理論是否正確的重要工具。

**實際應用**：
對應原理在辐射的量子理論、光譜線強度等問題上得到了應用。

## 三、X射線的量子解釋

1916 年，X射線研究取得重要進展。

**X射線的發現**：
1895 年，威廉·康拉德·倫琴（Wilhelm Röntgen）發現了 X 射線。

**X射線的本質**：
X射線是一種電磁波，其波長比可見光短得多。1912 年的 X射線晶體衍射實驗證實了這一點。

**量子理論的解釋**：
1916 年左右，物理學家開始用量子理論解釋 X射線的產生和性質。這為後來的 X射線光譜學奠定了基礎。

## 四、固體比熱的量子理論

**經典理論的困難**：
根據經典統計物理，固體的摩爾熱容應該是 3R（約 24.94 J/(mol·K)），與溫度無關。但實驗發現，在低溫下，固體熱容會顯著下降。

**愛因斯坦模型**：
1907 年，愛因斯坦提出了固體熱容的量子模型。他假設固體中所有原子的振動頻率都相同，根據量子理論，只有頻率 ν 的振動模式能量才能是 hν 的整數倍。

**低溫下的行為**：
溫度降低時，能量量子化效應變得顯著，熱容下降。愛因斯坦的模型雖然比較簡單，但成功解釋了熱容會趨近於零的趨勢。

## 五、光電效應的深入理解

**愛因斯坦的光量子假設**：
1905 年，愛因斯坦提出光電效應的量子解釋，認為光本身是由離散的光量子（光子）組成的。

**實驗驗證**：
1916 年，羅伯特·密立根（Robert Millikan）完成了光電效應的精密實驗，驗證了愛因斯坦的方程式。

```
E_k = hν - φ
```

其中 E_k 是發射電子的最大動能，h 是普朗克常數，ν 是光頻率，φ 是金屬逸出功。

## 六、量子理論的困境與出路

1916 年，舊量子論面臨嚴峻挑戰：

**無法處理的問題**：
- 光譜線強度
- 光譜線的精細結構
- 多電子原子
- 散射問題

**未來的方向**：
這些問題最終在 1925-1926 年的量子力學革命中被解決。海森堡的矩陣力學（1925）和薛定諤的波動力學（1926）開創了新紀元。
