# 馬可尼收音機：走進家庭的通信

**科學技術年鑑　1906 年號**

編輯：[陳鍾誠](https://csie.nqu.edu.tw/p/404-1038-2377.php?Lang=zh-tw)

---

西元 1906 年，[馬可尼](https://zh.wikipedia.org/wiki/馬可尼）的[收音機](https://zh.wikipedia.org/wiki/收音機)技術持續進步，開始慢慢走進家庭。這一年標誌著[大眾通信時代](https://zh.wikipedia.org/wiki/大眾媒體)的到來，為日後[廣播](https://zh.wikipedia.org/wiki/廣播)和[電視](https://zh.wikipedia.org/wiki/電視)的普及奠定了基礎。

## 收音機的原理

### 無線電信號接收

[收音機](https://zh.wikipedia.org/wiki/收音機)的工作原理是接收[無線電信號](https://zh.wikipedia.org/wiki/無限電信號)並将其轉換為聲音：

| 過程 | 說明 |
|------|------|
| [天線接收](https://zh.wikipedia.org/wiki/天線) | 捕捉空中的無缐電信號 |
| [調諧](https://zh.wikipedia.org/wiki/調諧器) | 選擇特定頻率的信號 |
| [檢波](https://zh.wikipedia.org/wiki/檢波器) | 從高頻信號中提取音頻 |
| [放大](https://zh.wikipedia.org/wiki/放大器) | 增強信號強度 |
| [揚聲器](https://zh.wikipedia.org/wiki/揚聲器) | 將電信號轉換為聲音 |

### 關鍵技術

1906 年的[收音機](https://zh.wikipedia.org/wiki/收音機)使用以下關鍵技術：

| 技術 | 組件 |
|------|------|
| [天線](https://zh.wikipedia.org/wiki/天線) | 接收信號 |
| [調諧電路](https://zh.wikipedia.org/wiki/調諧電路) | 選擇電台 |
| [檢波器](https://zh.wikipedia.org/wiki/檢波器) | 提取音頻 |
| [放大器](https://zh.wikipedia.org/wiki/放大器) | 增強信號（使用真空管） |

## 1906年的技術進步

### 接收靈敏度提高

1906 年，[收音機](https://zh.wikipedia.org/wiki/收音機)的接收性能有了顯著提高：

| 指標 | 改進 |
|------|------|
| [靈敏度](https://zh.wikipedia.org/wiki/靈敏度) | 能接收更微弱信號 |
| [選擇性](https://zh.wikipedia.org/wiki/選擇性) | 更好地区分電台 |
| [穩定性](https://zh.wikipedia.org/wiki/穩定性) | 減少信号飄移 |

### 真空管的應用

[真空管](https://zh.wikipedia.org/wiki/真空管)的應用使[收音機](https://zh.wikipedia.org/wiki/收音機)性能大幅提升：

| 型號 | 年份 | 放大能力 |
|------|------|----------|
| [二極管](https://zh.wikipedia.org/wiki/二極管) | 1904 年 | 檢波 |
| [三極管](https://zh.wikipedia.org/wiki/三極管) | 1906 年 | 放大信號 |

## 早期應用場景

### 船舶通信

[收音機](https://zh.wikipedia.org/wiki/收音機)的最早應用之一是[船舶通信](https://zh.wikipedia.org/wiki/船舶通信)：

| 應用 | 說明 |
|------|------|
| [遇難求救](https://zh.wikipedia.org/wiki/SOS) | 船舶遇難時發送信號 |
| [航行信息](https://zh.wikipedia.org/wiki/航行信息) | 接收天氣和导航信息 |
| [港口通信](https://zh.wikipedia.org/wiki/港口通信) | 與岸上保持聯繫 |

### 業餘愛好者

1906 年，[業餘無缐電愛好者](https://zh.wikipedia.org/wiki/業餘無線電)群體開始形成：

| 群體 | 活動 |
|------|------|
| [業餘愛好者](https://zh.wikipedia.org/wiki/業餘無限電) | 進行實驗和通聯 |
| 實驗家 | 改進設備 |
| 先驅者 | 開發新應用 |

### 家庭普及的前奏

1906 年，[收音機](https://zh.wikipedia.org/wiki/收音機)開始緩慢進入少數富裕家庭：

| 用戶類型 | 用途 |
|----------|------|
| 科學愛好者 | 接收摩爾斯電碼 |
| 富裕家庭 | 接收新聞和音樂 |
| 商業用戶 | 接收市場信息 |

## 對大眾媒體的影響

### 廣播時代的到來

[收音機](https://zh.wikipedia.org/wiki/收音機)的發明預示著[大眾媒體](https://zh.wikipedia.org/wiki/大眾媒體)時代的來臨：

| 年份 | 發展 |
|------|------|
| 1906 年 | 技術持續改進 |
| 1920 年代 | 商業廣播電台興起 |
| 1930 年代 | 收音機普及 |
| 1950 年代 | 電視出現 |

### 信息傳播革命

[收音機](https://zh.wikipedia.org/wiki/收音機)帶來了信息傳播的革命：

| 傳統方式 | 收音機方式 |
|----------|------------|
| 紙質媒體 | 即時傳播 |
| 地域限制 | 跨越距離 |
| 緩慢 | 實時 |

## 技術演進

### 從礦石機到真空管機

[收音機](https://zh.wikipedia.org/wiki/收音機)技術的演進：

| 類型 | 年份 | 特點 |
|------|------|------|
| [礦石收音機](https://zh.wikipedia.org/wiki/礦石收音機) | 1900 年代初 | 無需電源，靈敏度低 |
| [真空管收音機](https://zh.wikipedia.org/wiki/真空管收音機) | 1906 年後 | 需要電源，靈敏度高 |

### 從調幅到調頻

[收音機](https://zh.wikipedia.org/wiki/收音機)的調制方式演進：

| 類型 | 年份 | 優勢 |
|------|------|------|
| [調幅](https://zh.wikipedia.org/wiki/調幅)（AM） | 1900 年代 | 傳播距離遠 |
| [調頻](https://zh.wikipedia.org/wiki/調頻)（FM） | 1930 年代 | 音質更好 |

## 對現代通信的影響

### 行動通信的基礎

[收音機](https://zh.wikipedia.org/wiki/收音機)技術是現代[行動通信](https://zh.wikipedia.org/wiki/行動通信)的基礎：

| 技術 | 演進 |
|------|------|
| [收音機](https://zh.wikipedia.org/wiki/收音機) | 點對點通信 |
| [行動電話](https://zh.wikipedia.org/wiki/行動電話) | 移動通信 |
| [蜂窩網絡](https://zh.wikipedia.org/wiki/蜂窩網絡) | 大規模覆蓋 |
| [互聯網](https://zh.wikipedia.org/wiki/互聯網) | 數位融合 |

### 互聯網時代

如今，通過[互聯網](https://zh.wikipedia.org/wiki/網際網路)可以收聽世界各地的電台，這是 1906 年做夢都想不到的事情。

## 結語

1906 年，[馬可尼](https://zh.wikipedia.org/wiki/馬可尼)等先驅者的努力使[收音機](https://zh.wikipedia.org/wiki/收音機)技術持續進步。這項技術的發展預示著[大眾媒體時代](https://zh.wikipedia.org/wiki/大眾媒體)的到來，改變了人類獲取信息和娛樂的方式。

從收音機到電視，從有缐到無線，從類比到數位，通信技術的發展一直在改變人類的生活，而 1906 年正是這個偉大進程中的一個重要節點。
