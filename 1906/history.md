# 歷史穿梭：1906 年的科學、技術與未來

**科學技術年鑑　1906 年號**

編輯：[陳鍾誠](https://csie.nqu.edu.tw/p/404-1038-2377.php?Lang=zh-tw)

---

西元 1906 年，人類科學技術正在經歷深刻的變革。在這一年，我們看到了[科學發現](https://zh.wikipedia.org/wiki/科學發現)、[技術發明](https://zh.wikipedia.org/wiki/技術)和[產品創新](https://zh.wikipedia.org/wiki/產品)如何相互交織，共同塑造人類的未來。

## 從電子到數位時代

### 1906 年的突破

[湯姆森](https://zh.wikipedia.org/wiki/約瑟夫·湯姆森)在 1906 年獲得諾貝爾物理學獎，這是對他 1897 年發現[電子](https://zh.wikipedia.org/wiki/電子)的肯定。這一發現標誌著人類對微觀世界認識的重大突破。

### 連結到未來

[電子](https://zh.wikipedia.org/wiki/電子)的發現開創了[電子學](https://zh.wikipedia.org/wiki/電子學)的新時代：

| 1906 年 | 發展階段 | 對未來的影響 |
|---------|----------|--------------|
| 湯姆森發現電子 | 原子物理學誕生 | 開創微觀時代 |
| 1904 年 | 佛萊明二極管 | 電子設備誕生 |
| 1906 年 | 德福雷斯特三極管 | 放大器發明 |
| 1947 年 | 電晶體發明 | 固態電子學時代 |
| 1958 年 | 積體電路 | 電子設備小型化 |
| 1971 年 | 微處理器 | 個人電腦誕生 |
| 2020 年代 | 量子計算 | 計算新範式 |

### 對現代科技的影響

[電子](https://zh.wikipedia.org/wiki/電子)的發現是現代[半導體技術](https://zh.wikipedia.org/wiki/半導體)的理論基礎：

- [電腦](https://zh.wikipedia.org/wiki/電腦)（1945 年）——ENIAC
- [積體電路](https://zh.wikipedia.org/wiki/積體電路)（1958 年）——電腦的核心
- [微處理器](https://zh.wikipedia.org/wiki/微處理器)（1971 年）——個人電腦的基礎
- [智慧手機](https://zh.wikipedia.org/wiki/智慧手機）（2007 年）——移動互聯網終端

如果沒有 1906 年湯姆森電子的發現，這一切都不可能實現。

## 從熱離子發射到真空管

### 1906 年的研究

1906 年，對[熱離子發射](https://zh.wikipedia.org/wiki/熱離子發射)現象的深入研究為[真空管](https://zh.wikipedia.org/wiki/真空管)的發明提供了理論基礎。

### 連結到未來

[真空管](https://zh.wikipedia.org/wiki/真空管)的發展軌跡：

| 1906 年 | 技術突破 | 對未來的影響 |
|---------|----------|--------------|
| 熱離子發射研究 | 理論基礎 | 電子發射理論 |
| 1904 年 | 二極管發明 | 整流器 |
| 1906 年 | 三極管發明 | 放大器 |
| 1920 年代 | 廣播普及 | 大眾媒體時代 |
| 1940 年代 | 電子計算機 | 電腦時代 |
| 1950 年代 | 電視普及 | 視覺媒體時代 |

### 真空管到電晶體

[真空管](https://zh.wikipedia.org/wiki/真空管)最終被[電晶體](https://zh.wikipedia.org/wiki/電晶體)取代，但熱離子發射原理至今仍在某些應用中使用。

## 從神經元學說到腦科學

### 1906 年的研究

[高爾基](https://zh.wikipedia.org/wiki/卡米洛·高爾基)在 1906 年獲得諾貝爾獎，他的[神經系統研究](https://zh.wikipedia.org/wiki/神經科學)為現代神經科學奠定了基礎。

### 連結到未來

[神經科學](https://zh.wikipedia.org/wiki/神經科學)的發展催生了多個重要領域：

| 1906 年 | 理論基礎 | 對未來的影響 |
|---------|----------|--------------|
| 高爾基研究 | 神經元結構 | 理解神經系統 |
| 1920 年代 | 神經元學說確立 | 神經科學建立 |
| 1950 年代 | 神經生理學 | 腦功能研究 |
| 1990 年代 | 神經成像 | 腦科學革命 |
| 2020 年代 | 腦機接口 | 人機融合 |

### 人工智能與神經科學

[神經科學](https://zh.wikipedia.org/wiki/神經科學)與[人工智能](https://zh.wikipedia.org/wiki/人工智能)的交叉催生了深度學習和神經網絡技術。

## 從氟化學到現代材料

### 1906 年的研究

[莫瓦桑](https://zh.wikipedia.org/wiki/亨利·莫瓦桑)因研究[氟](https://zh.wikipedia.org/wiki/氟)和發明[電弧爐](https://zh.wikipedia.org/wiki/電弧爐)而獲得諾貝爾化學獎。

### 連結到未來

[氟化學](https://zh.wikipedia.org/wiki/氟)和電弧爐技術的發展軌跡：

| 1906 年 | 技術突破 | 對未來的影響 |
|---------|----------|--------------|
| 氟的分離 | 高活性元素 | 氟化學誕生 |
| 電弧爐 | 高溫材料 | 材料科學 |
| 1950 年代 | 特氟龍 | 不沾塗層 |
| 1980 年代 | 鋰離子電池 | 移動電源 |
| 2020 年代 | 固態電池 | 新能源革命 |

### 氟化物的廣泛應用

[氟](https://zh.wikipedia.org/wiki/氟)及氟化物在現代社會有廣泛應用：

- [含氟牙膏](https://zh.wikipedia.org/wiki/含氟牙膏)——預防蛀牙
- [特氟龍](https://zh.wikipedia.org/wiki/聚四氟乙烯)——不沾鍋塗層
- [鋰電池電解質](https://zh.wikipedia.org/wiki/鋰電池)——新能源汽車
- [氫氟烴](https://zh.wikipedia.org/wiki/氫氟烴)——環保製冷劑

## 從馬可尼通信到全球互聯

### 1906 年的技術

[馬可尼](https://zh.wikipedia.org/wiki/馬可尼)在 1906 年持續改進[跨大西洋通信](https://zh.wikipedia.org/wiki/跨大西洋電纜)技術，商用[無線電通信](https://zh.wikipedia.org/wiki/無限電通信)時代開始。

### 連結到未來

[通信技術](https://zh.wikipedia.org/wiki/通信技術)的發展軌跡：

| 1906 年 | 技術突破 | 對未來的影響 |
|---------|----------|--------------|
| 跨大西洋通信 | 商用服務 | 全球通信開始 |
| 1920 年代 | 商業廣播 | 大眾媒體時代 |
| 1960 年代 | 通信衛星 | 全球化通信 |
| 1990 年代 | 網際網路 | 數位時代 |
| 2020 年代 | 5G/6G | 萬物互聯 |

### 互聯網時代

如今，全球[互聯網](https://zh.wikipedia.org/wiki/互聯網)連接了數十億設備，催生了電子商務、社交媒體、雲端運算等新業態。

## 結論

1906 年的[科學發現](https://zh.wikipedia.org/wiki/科學發現)、[技術發明](https://zh.wikipedia.org/wiki/技術)和[產品創新](https://zh.wikipedia.org/wiki/產品)，對後世產生了深遠的影響：

| 1906 年 | 對未來的影響 |
|---------|--------------|
| [電子發現](https://zh.wikipedia.org/wiki/電子) | [量子計算](https://zh.wikipedia.org/wiki/量子計算)、[半導體](https://zh.wikipedia.org/wiki/半導體) |
| [熱離子發射](https://zh.wikipedia.org/wiki/熱離子發射) | [真空管](https://zh.wikipedia.org/wiki/真空管)、[電子學](https://zh.wikipedia.org/wiki/電子學) |
| [高爾基研究](https://zh.wikipedia.org/wiki/卡米洛·高爾基) | [神經科學](https://zh.wikipedia.org/wiki/神經科學)、[腦機接口](https://zh.wikipedia.org/wiki/腦機接口) |
| [氟化學](https://zh.wikipedia.org/wiki/氟) | [新材料](https://zh.wikipedia.org/wiki/材料科學)、[新能源](https://zh.wikipedia.org/wiki/新能源) |
| [X射線應用](https://zh.wikipedia.org/wiki/X射線) | [醫學影像](https://zh.wikipedia.org/wiki/醫學影像)、[精準醫療](https://zh.wikipedia.org/wiki/精準醫療) |
| [跨大西洋通信](https://zh.wikipedia.org/wiki/跨大西洋電纜) | [互聯網](https://zh.wikipedia.org/wiki/互聯網)、[5G](https://zh.wikipedia.org/wiki/5G) |
| [萊特兄弟實驗](https://zh.wikipedia.org/wiki/萊特兄弟) | [商業航空](https://zh.wikipedia.org/wiki/商業航空)、[太空探索](https://zh.wikipedia.org/wiki/太空探索) |
| [福特技術改進](https://zh.wikipedia.org/wiki/福特) | [電動汽車](https://zh.wikipedia.org/wiki/電動汽車)、[自動駕駛](https://zh.wikipedia.org/wiki/自動駕駛) |

這些在 1906 年播下的科技種子，在隨後的一個多世紀裡生根發芽、長成參天大樹，深刻改變了人類社會的面貌。

科技發展的腳步從未停歇。回首一百二十餘年前的這些成就，我們看到了人類智慧的光芒，也看到了未來發展的方向。
