# 化學獎：亨利·莫瓦桑與氟的發現

**科學技術年鑑　1906 年號**

編輯：[陳鍾誠](https://csie.nqu.edu.tw/p/404-1038-2377.php?Lang=zh-tw)

---

西元 1906 年，[法國化學家亨利·莫瓦桑](https://zh.wikipedia.org/wiki/亨利·莫瓦桑)（Henri Moissan，1852-1913）因其研究[氟元素](https://zh.wikipedia.org/wiki/氟)以及[電弧爐](https://zh.wikipedia.org/wiki/電弧爐)的發明，榮獲[諾貝爾化學獎](https://zh.wikipedia.org/wiki/諾貝爾化學獎)。

## 莫瓦桑的科學成就

### 氟元素的分離

[氟](https://zh.wikipedia.org/wiki/氟)是最具反應性的元素之一長期以來，化學家嘗試分離氟元素但都失敗了，因為氟極易與其他元素發生反應。

| 嘗試者 | 年份 | 結果 |
|--------|------|------|
| [戴維](https://zh.wikipedia.org/wiki/漢弗里·戴維) | 1810-1813 年 | 失敗 |
| [夫累米](https://zh.wikipedia.org/wiki/費迪南·弗雷米) | 1886 年 | 失敗 |
| [莫瓦桑](https://zh.wikipedia.org/wiki/亨利·莫瓦桑) | 1886 年 | 成功 |

### 電解法分離氟

[莫瓦桑](https://zh.wikipedia.org/wiki/亨利·莫瓦桑)採用[電解](https://zh.wikipedia.org/wiki/電解)的方法成功分離了氟：

| 條件 | 參數 |
|------|------|
| 電解質 | 氟化氫鉀（KHF₂） |
| 溫度 | 約 -23°C |
| 電壓 | 約 10-12 伏特 |
| 陽極材料 | 鉑銥合金 |

### 電弧爐的發明

[莫瓦桑](https://zh.wikipedia.org/wiki/亨利·莫瓦桑)還發明了[電弧爐](https://zh.wikipedia.org/wiki/電弧爐)，這是一種能夠達到極高溫度的設備：

| 特性 | 溫度 |
|------|------|
| 電弧爐溫度 | 可達 3500°C |
| 高熔點金屬 | 鎢、鉬、鈦 |
| 碳化物的合成 | 碳化矽等 |

## 電弧爐的應用

### 高溫化學

[電弧爐](https://zh.wikipedia.org/wiki/電弧爐)使化學家能夠進行以前不可能的高溫實驗：

| 應用領域 | 具體應用 |
|----------|----------|
| [金屬冶煉](https://zh.wikipedia.org/wiki/冶金學) | 提取高熔點金屬 |
| [化合物合成](https://zh.wikipedia.org/wiki/化學合成) | 合成碳化物、氮化物 |
| [材料研究](https://zh.wikipedia.org/wiki/材料科學) | 研發新材料 |

### 人造寶石

[莫瓦桑](https://zh.wikipedia.org/wiki/亨利·莫瓦桑)利用電弧爐首次合成了[人造寶石](https://zh.wikipedia.org/wiki/人造寶石)：

| 寶石 | 方法 |
|------|------|
| 人造紅寶石 | 氧化鋁摻雜鉻 |
| 人造藍寶石 | 氧化鋁摻雜鈦 |
| 人造鑽石 | 碳的高壓轉變 |

## 對化學的深遠影響

### 鹵素化學

[莫瓦桑](https://zh.wikipedia.org/wiki/亨利·莫瓦桑)的工作推動了[鹵素化學](https://zh.wikipedia.org/wiki/鹵素)的發展：

| 鹵素 | 特性 |
|------|------|
| [氟](https://zh.wikipedia.org/wiki/氟) | 最活潑的非金屬 |
| [氯](https://zh.wikipedia.org/wiki/氯) | 重要的工業原料 |
| [溴](https://zh.wikipedia.org/wiki/溴) | 廣泛用於製藥 |
| [碘](https://zh.wikipedia.org/wiki/碘) | 重要的營養素 |

### 高溫化學

[電弧爐](https://zh.wikipedia.org/wiki/電弧爐)的發明開創了[高溫化學](https://zh.wikipedia.org/wiki/高溫化學)這一新領域。

## 技術應用

### 工業應用

[莫瓦桑](https://zh.wikipedia.org/wiki/亨利·莫瓦桑)的研究成果在工業上有廣泛應用：

| 應用領域 | 例子 |
|----------|------|
| [冶金工業](https://zh.wikipedia.org/wiki/冶金學) | 鋼鐵冶煉 |
| [化工行業](https://zh.wikipedia.org/wiki/化學工業) | 氟化物生產 |
| [材料科學](https://zh.wikipedia.org/wiki/材料科學) | 高級陶瓷 |
| [寶石加工](https://zh.wikipedia.org/wiki/寶石加工) | 人造寶石 |

### 氟化物的應用

[氟](https://zh.wikipedia.org/wiki/氟)及氟化物在現代社會有廣泛應用：

| 應用 | 例子 |
|------|------|
| [含氟牙膏](https://zh.wikipedia.org/wiki/含氟牙膏) | 預防蛀牙 |
| [氟氯化碳](https://zh.wikipedia.org/wiki/氟氯化碳) | 製冷劑（已禁用） |
| [特氟龍](https://zh.wikipedia.org/wiki/聚四氟乙烯) | 不沾塗層 |
| [鋰電池](https://zh.wikipedia.org/wiki/鋰電池) | 電解質材料 |

## 對未來科技的影響

### 材料科學

[莫瓦桑](https://zh.wikipedia.org/wiki/亨利·莫瓦桑)的電弧爐為[材料科學](https://zh.wikipedia.org/wiki/材料科學)的發展提供了重要工具：

| 年份 | 發展 |
|------|------|
| 1906 年 | 電弧爐發明 |
| 1907 年 | 人造紅寶石 |
| 1950 年代 | 區熔法提純半導體 |
| 1980 年代 | 高溫超導材料 |

### 環境問題

氟氯化碳（CFC）等氟化物雖然曾被廣泛使用，但後來被發現會破壞[臭氧層](https://zh.wikipedia.org/wiki/臭氧層)。

## 結語

[莫瓦桑](https://zh.wikipedia.org/wiki/亨利·莫瓦桑)在 1906 年獲得[諾貝爾化學獎](https://zh.wikipedia.org/wiki/諾貝爾化學獎)，是對他分離氟元素和發明電弧爐這兩個偉大成就的肯定。他的工作不僅推動了化學的發展，更為材料科學和工業技術的進步開闢了道路。

從氟化學到高溫冶金，從人造寶石到特氟龍塗層，莫瓦桑的貢獻深刻影響了現代生活和科技發展。
