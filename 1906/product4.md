# 早期電影放映機：動態影像的普及

**科學技術年鑑　1906 年號**

編輯：[陳鍾誠](https://csie.nqu.edu.tw/p/404-1038-2377.php?Lang=zh-tw)

---

西元 1906 年，[電影放映機](https://zh.wikipedia.org/wiki/電影放映機)技術持續發展，[電影](https://zh.wikipedia.org/wiki/電影)作為一種新興的藝術形式和娛樂方式，正在慢慢走向大眾。這一年標誌著電影工業即將爆發的前奏。

## 電影技術的歷史

### 早期發明

[電影](https://zh.wikipedia.org/wiki/電影)的發明基於多項前驅技術：

| 技術 | 發明者 | 年份 |
|------|--------|------|
| [視覺暫留](https://zh.wikipedia.org/wiki/視覺暫留) | 彼得·馬克·羅杰特 | 1824 年 |
| [幻燈機](https://zh.wikipedia.org/wiki/幻燈機) | 克里斯蒂安·惠更斯 | 1659 年 |
| [攝影術](https://zh.wikipedia.org/wiki/攝影術) | 路易·達蓋爾 | 1839 年 |
| [電影攝影機](https://zh.wikipedia.org/wiki/電影攝影機) | 路易·勒埃普拉 | 1895 年 |

### 電影誕生

1895 年，[路易·勒埃普拉](https://zh.wikipedia.org/wiki/路易·勒埃普拉)和[湯馬斯·愛迪生](https://zh.wikipedia.org/wiki/湯馬斯·愛迪生）几乎同時發明了電影攝影和放映設備。

## 1906年的技術進步

### 放映技術改進

1906 年，[電影放映機](https://zh.wikipedia.org/wiki/電影放映機)的技術有了重要改進：

| 技術 | 改進 |
|------|------|
| [光源](https://zh.wikipedia.org/wiki/光源) | 更亮的弧光燈 |
| [膠片](https://zh.wikipedia.org/wiki/電影膠片) | 更耐用的安全膠片 |
| [放映速度](https://zh.wikipedia.org/wiki/放映速度) | 標準化到每秒16-24格 |
| [放映距離](https://zh.wikipedia.org/wiki/放映距離) | 更長的放映距離 |

### 電影膠片的標準化

1906 年，電影膠片的規格開始標準化：

| 規格 | 尺寸 |
|------|------|
| 35毫米膠片 | 寬度 35毫米 |
| 齒孔間距 | 標準化 |
| 畫面比例 | 4:3 |

## 早期電影內容

### 紀錄片

1906 年的[電影](https://zh.wikipedia.org/wiki/電影)內容主要集中在：

| 類型 | 說明 |
|------|------|
| [風景片](https://zh.wikipedia.org/wiki/風景片) | 記錄風光景觀 |
| [新聞片](https://zh.wikipedia.org/wiki/新聞片) | 記錄時事 |
| [舞台表演](https://zh.wikipedia.org/wiki/舞台表演) | 記錄戲劇和舞蹈 |

### 早期電影特點

1906 年的電影具有以下特點：

| 特點 | 說明 |
|------|------|
| 時長 | 較短，通常幾分鐘 |
| 無聲 | 沒有配音 |
| 黑白 | 黑白影像 |
| 單鏡頭 | 單一拍攝角度 |

## 電影放映場所

### 早期場所

1906 年，[電影](https://zh.wikipedia.org/wiki/電影)主要在以下場所放映：

| 場所 | 說明 |
|------|------|
| [音樂廳](https://zh.wikipedia.org/wiki/音樂廳) | 大型放映活動 |
| [ nickelodeon](https://zh.wikipedia.org/wiki/鎳幣影院) | 低價小放映廳 |
| [露天放映](https://zh.wikipedia.org/wiki/露天放映) | 室外放映 |

### Nickelodeon 的興起

1906 年是[nickelodeon](https://zh.wikipedia.org/wiki/鎳幣影院)（低價電影院）快速發展的時期：

| 項目 | 說明 |
|------|------|
| 門票價格 | 5美分（nickel） |
| 座位數 | 20-100個 |
| 屏幕大小 | 較小 |
| 每日放映 | 多場次 |

## 對電影工業的影響

### 電影工業的興起

[電影](https://zh.wikipedia.org/wiki/電影)在 1906 年開始成為一個產業：

| 環節 | 發展 |
|------|------|
| [製作](https://zh.wikipedia.org/wiki/電影製作) | 電影公司興起 |
| [發行](https://zh.wikipedia.org/wiki/電影發行) | 拷貝租賃業務 |
| [放映](https://zh.wikipedia.org/wiki/電影放映) | nickelodeon普及 |

### 好萊塢的萌芽

1906 年前後，電影開始在[好萊塢](https://zh.wikipedia.org/wiki/好萊塢)地區拍攝，利用當地充足的光線和多樣的景觀。

## 技術演進時間線

[電影技術](https://zh.wikipedia.org/wiki/電影技術)的發展：

| 年份 | 里程碑 |
|------|--------|
| 1895 年 | 電影發明 |
| 1900 年代初 | 電影放映普及 |
| 1906 年 | nickelodeon興起 |
| 1910 年代 | 長片出現 |
| 1927 年 | 有聲電影 |
| 1930 年代 | 彩色電影 |

## 對現代電影的影響

### 從無聲到有聲

[電影](https://zh.wikipedia.org/wiki/電影)的發展軌跡：

| 類型 | 年份 | 技術 |
|------|------|------|
| 無聲電影 | 1906 年 | 純視覺體驗 |
| 有聲電影 | 1927 年 | 維他風暴系統 |
| 彩色電影 | 1930 年代 | 特藝彩印 |
| 數位電影 | 1990 年代 | CGI |

### 電影藝術的建立

[電影](https://zh.wikipedia.org/wiki/電影)作為一種藝術形式，在1906年開始建立：

| 元素 | 發展 |
|------|------|
| [剪輯](https://zh.wikipedia.org/wiki/剪輯) | 蒙太奇概念 |
| [攝影機移動](https://zh.wikipedia.org/wiki/攝影機移動) | 各種移動鏡頭 |
| [敘事](https://zh.wikipedia.org/wiki/電影敘事) | 故事講述方式 |

## 結語

1906 年是[電影](https://zh.wikipedia.org/wiki/電影)工業即將爆發的前夕。從早期的實驗到[nickelodeon](https://zh.wikipedia.org/wiki/鎳幣影院)的興起，電影作為一種新興的藝術形式和娛樂方式，正在慢慢走進大眾的生活。

從無聲電影到有聲電影，從黑白到彩色，從膠片到數位，電影技術的發展經歷了革命性的變化。如今，電影已成為最重要的文化產業之一，而1906年正是這個偉大歷程中的重要節點。
