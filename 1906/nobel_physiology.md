# 生理學或醫學獎：卡米洛·高爾基與神經系統

**科學技術年鑑　1906 年號**

編輯：[陳鍾誠](https://csie.nqu.edu.tw/p/404-1038-2377.php?Lang=zh-tw)

---

西元 1906 年，[義大利醫學家卡米洛·高爾基](https://zh.wikipedia.org/wiki/卡米洛·高爾基)（Camillo Golgi，1843-1926）因其對[神經系統結構](https://zh.wikipedia.org/wiki/神經系統)的開創性研究，與[桑地牙哥·拉蒙·卡哈爾](https://zh.wikipedia.org/wiki/桑地牙哥·拉蒙·卡哈爾共同獲得[諾貝爾生理學或醫學獎](https://zh.wikipedia.org/wiki/諾貝爾生理學或醫學獎)。

## 高爾基的科學成就

### 黑色反應技術

[高爾基](https://zh.wikipedia.org/wiki/卡米洛·高爾基)最重要的貢獻是發明了「[黑色反應](https://zh.wikipedia.org/wiki/黑色反應)」染色技術。這種技術能夠將單一神經細胞完整染色，使其在顯微鏡下清晰可見。

| 步驟 | 說明 |
|------|------|
| 浸泡 | 將組織浸泡在硝酸銀溶液中 |
| 沉澱 | 銀鹽在細胞中選擇性沉澱 |
| 顯色 | 形成黑色沉澱物 |
| 透明 | 其他組織保持透明 |

### 網狀學說

基於他的觀察，[高爾基](https://zh.wikipedia.org/wiki/卡米洛·高爾基)提出了「[網狀學說](https://zh.wikipedia.org/wiki/網狀學說)」：

> 神經系統是一個連續的網絡，神經細胞之間沒有間隔，而是直接連接成一個整體。

這個理論與卡哈爾的「[神經元學說](https://zh.wikipedia.org/wiki/神經元學說)」形成了學術上的對立。

## 網狀學說 vs 神經元學說

### 兩種理論的對比

| 理論 | 支持者 | 核心觀點 |
|------|--------|----------|
| [網狀學說](https://zh.wikipedia.org/wiki/網狀學說) | 高爾基 | 神經細胞連成一片 |
| [神經元學說](https://zh.wikipedia.org/wiki/神經元學說) | 卡哈爾 | 神經細胞是獨立單位 |

### 現代驗證

現代科學已經證實：

1. **神經元學說是正確的**：神經元是獨立的功能單位
2. **突觸的存在**：神經元之間通過[突觸](https://zh.wikipedia.org/wiki/突觸)連接
3. **信號單向傳遞**：信息沿特定方向傳遞

然而，兩位科學家共同獲獎，表明他們的工作都對神經科學做出了重要貢獻。

## 高爾基的重大發現

### 高爾基體

除了網狀學說，[高爾基](https://zh.wikipedia.org/wiki/卡米洛·高爾基)還發現了細胞中一種重要的[細胞器](https://zh.wikipedia.org/wiki/細胞器)——[高爾基體](https://zh.wikipedia.org/wiki/高爾基體)。

| 細胞器 | 功能 |
|--------|------|
| [高爾基體](https://zh.wikipedia.org/wiki/高爾基體) | 蛋白質加工、包裝、運輸 |
| [線粒體](https://zh.wikipedia.org/wiki/線粒體) | 能量產生 |
| [細胞核](https://zh.wikipedia.org/wiki/細胞核) | 遺傳信息存儲 |

### 對神經科學的貢獻

儘管網狀學說最終被證明是錯誤的，但高爾基的貢獻仍然巨大：

1. **發明黑色反應**：使神經系統可視化
2. **發現高爾基體**：細胞生物學里程碑
3. **開創神經組織學**：為神經科學奠定基礎

## 對神經科學的影響

### 技術革命

[高爾基](https://zh.wikipedia.org/wiki/卡米洛·高爾基)的「[黑色反應](https://zh.wikipedia.org/wiki/黑色反應)」技術帶來了神經科學的革命：

| 技術 | 影響 |
|------|------|
| 黑色反應 | 使神經細胞可視化 |
| 銀染技術 | 推廣到其他組織染色 |
| 顯微鏡技術 | 促進顯微鏡發展 |

### 現代神經科學

[神經元](https://zh.wikipedia.org/wiki/神經元)是現代[神經科學](https://zh.wikipedia.org/wiki/神經科學)的基石：

| 發展 | 內容 |
|------|------|
| 神經元學說 | 神經系統的基本單位 |
| 突觸傳遞 | 神經信號的傳遞方式 |
| 神經網絡 | 神經元的連接模式 |

## 臨床應用

### 神經疾病研究

對[神經元](https://zh.wikipedia.org/wiki/神經元)結構的了解推動了神經疾病的治療：

| 疾病 | 研究 |
|------|------|
| [阿茲海默症](https://zh.wikipedia.org/wiki/阿茲海默症) | 神經元退化 |
| [帕金森氏症](https://zh.wikipedia.org/wiki/帕金森氏症) | 多巴胺神經元 |
| [癲癇](https://zh.wikipedia.org/wiki/癲癇) | 神經元過度興奮 |

### 神經修復

[神經科學](https://zh.wikipedia.org/wiki/神經科學)的研究正在推動神經修復技術的發展：

- [神經再生](https://zh.wikipedia.org/wiki/神經再生)
- [神經移植](https://zh.wikipedia.org/wiki/神經移植)
- [腦機接口](https://zh.wikipedia.org/wiki/腦機接口)

## 結語

[高爾基](https://zh.wikipedia.org/wiki/卡米洛·高爾基)在 1906 年與卡哈爾共同獲得[諾貝爾獎](https://zh.wikipedia.org/wiki/諾貝爾獎)，是對他開創性研究的極高認可。儘管他的網狀學說最終被證明是錯誤的，但他發明的黑色反應技術和發現的高爾基體，對[神經科學](https://zh.wikipedia.org/wiki/神經科學)和[細胞生物學](https://zh.wikipedia.org/wiki/細胞生物學)的發展做出了不可磨滅的貢獻。

從高爾基的顯微鏡到現代的[功能性磁振造影](https://zh.wikipedia.org/wiki/功能性磁振造影)（fMRI），人類對大腦的認識經歷了革命性的變化。
