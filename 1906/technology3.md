# 早期X光技術的醫學應用：透視人體

**科學技術年鑑　1906 年號**

編輯：[陳鍾誠](https://csie.nqu.edu.tw/p/404-1038-2377.php?Lang=zh-tw)

---

西元 1906 年，[X射線](https://zh.wikipedia.org/wiki/X射線)技術在[醫學診斷](https://zh.wikipedia.org/wiki/醫學影像)領域的應用持續深入。這項於 1895 年被[威廉·倫琴](https://zh.wikipedia.org/wiki/威廉·倫琴)發現的革命性技術，正在改變人類對疾病的診斷方式。

## X射線技術的回顧

### 發現背景

1895 年 11 月，[德國物理學家威廉·倫琴](https://zh.wikipedia.org/wiki/威廉·倫琴)在實驗室中意外發現了[X射線](https://zh.wikipedia.org/wiki/X射線)。這種未知性質的射線能夠穿透固體物質，在熒光屏上顯示出骨骼和物體的輪廓。

### 基本原理

[X射線](https://zh.wikipedia.org/wiki/X射線)的醫學應用基於以下原理：

| 原理 | 說明 |
|------|------|
| [穿透性](https://zh.wikipedia.org/wiki/X射線) | X射線能穿透人體軟組織 |
| [吸收差異](https://zh.wikipedia.org/wiki/X射線) | 不同組織對X射線吸收程度不同 |
| [成像原理](https://zh.wikipedia.org/wiki/成像) | 形成不同灰度的影像 |

## 1906年的技術進步

### 成像技術改進

1906 年，[X射線成像](https://zh.wikipedia.org/wiki/射線照相術)技術取得了重要進步：

| 技術改進 | 具體內容 |
|----------|----------|
| [感光版](https://zh.wikipedia.org/wiki/感光版) | 使用更敏感的感光材料 |
| [曝光時間](https://zh.wikipedia.org/wiki/曝光時間) | 曝光時間縮短 |
| [影像品質](https://zh.wikipedia.org/wiki/影像品質) | 影像清晰度提高 |

### 防護措施

隨著對[X射線](https://zh.wikipedia.org/wiki/輻射防護)危害的認識增加，1906 年開始重視防護：

- 使用[鉛圍裙](https://zh.wikipedia.org/wiki/鉛圍裙)保護
- 限制曝光時間
- 增加工作人員培訓

### 設備小型化

[X光設備](https://zh.wikipedia.org/wiki/X光機)開始朝著更小型化和實用化的方向發展：

| 設備類型 | 1900 年 | 1906 年 |
|----------|---------|---------|
| 尺寸 | 笨重 | 較為輕便 |
| 操作難度 | 複雜 | 簡化 |
| 應用範圍 | 有限 | 擴大 |

## 醫學應用領域

### 骨折診斷

[X射線](https://zh.wikipedia.org/wiki/X射線)在[骨折](https://zh.wikipedia.org/wiki/骨折)診斷中的應用最為廣泛：

| 應用 | 說明 |
|------|------|
| [骨折定位](https://zh.wikipedia.org/wiki/骨折) | 準確確定骨折位置 |
| 骨折類型 | 區分簡單骨折和複雜骨折 |
| 癒合監測 | 追蹤骨折癒合情況 |

### 異物探測

[X射線](https://zh.wikipedia.org/wiki/X射線)還能用於探測體內異物：

- [子彈](https://zh.wikipedia.org/wiki/子彈)和彈片定位
- [吞入異物](https://zh.wikipedia.org/wiki/異物)的檢測
- [誤嚥](https://zh.wikipedia.org/wiki/誤嚥)檢查

### 疾病診斷

1906 年，[X射線](https://zh.wikipedia.org/wiki/射線照相術)開始用於更多疾病的診斷：

| 疾病類型 | 應用 |
|----------|------|
| [肺結核](https://zh.wikipedia.org/wiki/肺結核) | 肺部影像檢查 |
| [肺炎](https://zh.wikipedia.org/wiki/肺炎) | 肺部感染診断 |
| [腫瘤](https://zh.wikipedia.org/wiki/腫瘤) | 異常生長檢測 |

## 對醫學的革命性影響

### 診斷革命

[X射線](https://zh.wikipedia.org/wiki/射線照相術)的發明帶來了醫學診斷的革命：

| 傳統方法 | X射線方法 |
|----------|----------|
| 觸診 | 無創可視化 |
| 猜測 | 準確定位 |
| 手術探查 | 避免不必要的手術 |

### 外科手術的進步

[X射線](https://zh.wikipedia.org/wiki/射線照相術)為[外科手術](https://zh.wikipedia.org/wiki/外科手術)帶來了革命性變化：

1. **術前規劃**：手術前了解病情
2. **術中定位**：準確找到病變位置
3. **術後複查**：確認手術效果

## 技術發展時間線

[X射線](https://zh.wikipedia.org/wiki/X射線)醫學應用的發展：

| 年份 | 里程碑 |
|------|--------|
| 1895 年 | 倫琴發現X射線 |
| 1896 年 | 首次用於醫學診斷 |
| 1906 年 | 技術持續改進 |
| 1913 年 | 開始使用X射線管 |
| 1920 年代 | 普及到各大醫院 |

## 輻射防護的演進

### 早期認識

1906 年，人們對[X射線](https://zh.wikipedia.org/wiki/輻射防護)的危害有了初步認識：

| 危害 | 說明 |
|------|------|
| [皮膚損傷](https://zh.wikipedia.org/wiki/辐射伤害) | 長期暴露導致燒傷 |
| [脫髮](https://zh.wikipedia.org/wiki/脫髮) | 頭部照射導致脫髮 |
| [癌症風險](https://zh.wikipedia.org/wiki/癌症) | 長期暴露增加患癌風險 |

### 防護原則

這些早期認識奠定了現代[輻射防護](https://zh.wikipedia.org/wiki/輻射防護)的基礎：

1. **時間原則**：盡量減少曝光時間
2. **距離原則**：遠離X射線源
3. **屏蔽原則**：使用防護屏障

## 現代X射線技術

從 1906 年的早期應用到現代技術，[X射線](https://zh.wikipedia.org/wiki/射線照相術)經歷了巨大進步：

| 技術 | 說明 |
|------|------|
| [電腦斷層掃描](https://zh.wikipedia.org/wiki/電腦斷層掃描)（CT） | 三維成像 |
| [乳房攝影](https://zh.wikipedia.org/wiki/乳房攝影術) | 乳腺癌篩查 |
| [血管造影](https://zh.wikipedia.org/wiki/血管造影) | 血管成像 |
| [數位X光](https://zh.wikipedia.org/wiki/數位X光) | 數位化處理 |

## 結語

1906 年，[X射線](https://zh.wikipedia.org/wiki/X射線)技術在醫學領域的應用持續深入。這項革命性的技術使人類第一次能夠無創地觀察體內結構，改變了醫學診斷和治療的方式。

從倫琴的偶然發現到現代的電腦斷層掃描，X射線技術經歷了一百多年的發展，成為現代醫學不可或缺的工具。
