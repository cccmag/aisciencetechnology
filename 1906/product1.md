# 早期真空管：電子設備的誕生

**科學技術年鑑　1906 年號**

編輯：[陳鍾誠](https://csie.nqu.edu.tw/p/404-1038-2377.php?Lang=zh-tw)

---

西元 1906 年，[真空管](https://zh.wikipedia.org/wiki/真空管)技術正在快速發展。這一年標誌著電子設備時代的開始，為日後的[廣播](https://zh.wikipedia.org/wiki/廣播)、[通信](https://zh.wikipedia.org/wiki/通信技術)和[計算機](https://zh.wikipedia.org/wiki/計算機)技術奠定了基礎。

## 真空管的發明背景

### 前驅技術

[真空管](https://zh.wikipedia.org/wiki/真空管)的發明基於多項前驅技術的發展：

| 技術 | 時間 | 貢獻 |
|------|------|------|
| [熱離子發射](https://zh.wikipedia.org/wiki/熱離子發射) | 1880 年代 | 電子發射原理 |
| [愛迪生效應](https://zh.wikipedia.org/wiki/愛迪生效應) | 1883 年 | 熱電子發射現象 |
| [真空技術](https://zh.wikipedia.org/wiki/真空技術) | 1900 年代 | 高真空環境 |

### 愛迪生效應

1883 年，[托馬斯·愛迪生](https://zh.wikipedia.org/wiki/托馬斯·愛迪生)在改進電燈時發現了一個奇怪現象：把一片金屬封裝在抽真空的燈泡中，金屬片在加熱時會有電流通過。這就是著名的「[愛迪生效應](https://zh.wikipedia.org/wiki/愛迪生效應)」——熱電子發射的首次觀察。

## 1904年：佛萊明二極管

### 發明

1904 年，[約翰·佛萊明](https://zh.wikipedia.org/wiki/約翰·佛萊明)（John Ambrose Fleming）基於愛迪生效應發明了[二極管](https://zh.wikipedia.org/wiki/二極管)（Vacuum Tube Diode）。

### 結構

[佛萊明二極管](https://zh.wikipedia.org/wiki/佛萊明閥)的結構：

| 組件 | 說明 |
|------|------|
| [陰極](https://zh.wikipedia.org/wiki/陰極) | 加熱後發射電子 |
| [陽極](https://zh.wikipedia.org/wiki/陽極) | 接收電子 |
| 玻璃外殼 | 抽真空，封裝 |

### 功能

[二極管](https://zh.wikipedia.org/wiki/二極管)具有整流功能——只允許電流單向流動。這使其能夠將[交流電](https://zh.wikipedia.org/wiki/交流電)轉換為[直流電](https://zh.wikipedia.org/wiki/直流電)。

## 1906年：德福雷斯特三極管

### 發明

1906 年，[李·德福雷斯特](https://zh.wikipedia.org/wiki/李·德福雷斯特)（Lee De Forest）發明了[三極管](https://zh.wikipedia.org/wiki/三極管)（Audion），這是電子技術史上最重要的發明之一。

### 結構

[三極管](https://zh.wikipedia.org/wiki/三極管)在二極管的基礎上增加了[柵極](https://zh.wikipedia.org/wiki/柵極)：

| 組件 | 位置 | 功能 |
|------|------|------|
| [陰極](https://zh.wikipedia.org/wiki/陰極) | 中央 | 發射電子 |
| [柵極](https://zh.wikipedia.org/wiki/柵極) | 陰極和陽極之間 | 控制電子流 |
| [陽極](https://zh.wikipedia.org/wiki/陽極) | 外層 | 接收電子 |

### 突破性功能

[三極管](https://zh.wikipedia.org/wiki/三極管)的最重要功能是**放大**——微弱的電信號可以被放大到很大的倍數。這是電子學發展的里程碑。

## 早期應用

### 通信領域

1906 年的[真空管](https://zh.wikipedia.org/wiki/真空管)主要應用於：

| 應用 | 說明 |
|------|------|
| [無線電接收](https://zh.wikipedia.org/wiki/無限電接收機) | 放大微弱信號 |
| [電話中繼](https://zh.wikipedia.org/wiki/電話中繼站) | 增強長途電話信號 |
| [電報](https://zh.wikipedia.org/wiki/電報) | 改善電報傳輸 |

### 信號處理

[真空管](https://zh.wikipedia.org/wiki/真空管)能夠實現以下信號處理功能：

| 功能 | 說明 |
|------|------|
| [整流](https://zh.wikipedia.org/wiki/整流器) | 交流變直流 |
| [放大](https://zh.wikipedia.org/wiki/放大器) | 增強信號 |
| [振盪](https://zh.wikipedia.org/wiki/振盪器) | 產生信號 |

## 對電子技術的影響

### 電子學的誕生

[真空管](https://zh.wikipedia.org/wiki/真空管)的發明標誌著[電子學](https://zh.wikipedia.org/wiki/電子學)這一新興學科的誕生。

### 技術發展時間線

| 年份 | 發展 |
|------|------|
| 1904 年 | 二極管發明 |
| 1906 年 | 三極管發明 |
| 1910 年代 | 真空管實用化 |
| 1920 年代 | 廣播時代來臨 |
| 1940 年代 | 電子計算機 |

## 從真空管到電晶體

### 固態電子學

1947 年，[電晶體](https://zh.wikipedia.org/wiki/電晶體)的發明標誌著[固態電子學](https://zh.wikipedia.org/wiki/固態電子學)時代的來臨：

| 設備 | 優勢 | 劣勢 |
|------|------|------|
| [真空管](https://zh.wikipedia.org/wiki/真空管) | 耐高壓，線性好 | 體積大，功耗高，壽命短 |
| [電晶體](https://zh.wikipedia.org/wiki/電晶體) | 體積小，功耗低，壽命長 | 初期成本高 |

### 現代應用

儘管[電晶體](https://zh.wikipedia.org/wiki/電晶體)已經取代了大多數應用中的真空管，但真空管仍在某些特定領域使用：

| 應用領域 | 原因 |
|----------|------|
| [音響設備](https://zh.wikipedia.org/wiki/真空管放大器) | 溫暖的音色 |
| [射頻功率](https://zh.wikipedia.org/wiki/射頻功率放大器) | 高功率需求 |
| [X射線管](https://zh.wikipedia.org/wiki/X射線管) | 高電壓需求 |

## 結語

1906 年是[電子設備](https://zh.wikipedia.org/wiki/電子設備)時代的開始。[真空管](https://zh.wikipedia.org/wiki/真空管)的發明使人類首次能夠控制和放大電信號，為日後的廣播、通信和計算機技術奠定了基礎。

從三極管到積體電路，從真空管計算機到量子計算機，電子技術經歷了翻天覆地的變化，而 1906 年的發明正是這一切的开端。
