# 聯合國的成立：世界和平的新希望

## 一、聯合國的構想

1945年4月，來自50個國家的代表聚集在舊金山，召開聯合國國際組織會議。會議的目的是建立一個新的國際組織，以維護世界和平與安全。

1945年6月26日，50個國家的代表在《聯合國憲章》上簽字。

## 二、聯合國的宗旨

《聯合國憲章》的宗旨是：
- 維護國際和平與安全
- 發展國際間友好關係
- 合作解決國際問題
- 促進對人權的尊重

## 三、成立日期

1945年10月24日，《聯合國憲章》正式生效，聯合國正式成立。這一天被定為"聯合國日"。

## 四、主要機構

1945年成立的聯合國主要機構包括：

| 機構 | 職能 |
|------|------|
| 大會 | 各國代表討論問題 |
| 安全理事會 | 維護國際和平 |
| 經濟及社會理事會 | 協調經濟社會工作 |
| 托管理事會 | 管理託管領土 |
| 國際法院 | 裁決國際爭端 |
| 秘書處 | 行政事務 |

## 五、歷史意義

聯合國的成立是人類追求和平的重要里程碑。雖然聯合國在歷史上經歷了諸多挑戰，但它為國際社會提供了一個對話和合作的重要平台。
