# 青黴素的普及與改進

## 一、青黴素的歷史

1928年，亞歷山大·弗萊明（Alexander Fleming）發現了青黴素。1941年，弗洛里和錢恩改進了培養方法。1943年，青黴素實現工業化生產。1945年，青黴素已經廣泛應用於臨床。

## 二、1945年的產量與應用

### 產量

1945年，青黴素的產量達到了很高的水平：

| 年份 | 月產量（十億單位） |
|------|-------------------|
| 1943 | 數十 |
| 1944 | 數百 |
| 1945 | 數千 |

### 應用範圍

1945年的青黴素應用範圍：

| 適應症 | 效果 |
|--------|------|
| 肺炎 | 顯著降低死亡率 |
| 傷口感染 | 加速愈合 |
| 敗血症 | 有效控制 |
| 梅毒 | 可治愈 |
| 淋病 | 效果顯著 |

## 三、技術改進

1945年，青黴素的生產技術有了進一步改進：
- 深層發酵工藝更加成熟
- 提取和純化方法更加有效
- 質量控制更加嚴格

## 四、歷史意義

青黴素的普及挽救了無數生命，被譽為"20世紀最重要的藥物"。1945年的諾貝爾生理學或醫學獎授予了弗萊明、弗洛里和錢恩，以表彰他們發現青黴素的貢獻。
