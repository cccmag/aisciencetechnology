## 本期內容導覽

1929 年是人類歷史上令人銘記的一年。這一年，[埃德溫·哈勃](https://zh.wikipedia.org/wiki/埃德溫·哈勃)（Edwin Hubble）發表了他對宇宙膨脹的突破性發現，為現代宇宙學奠定了基礎。同時，[路易·德布羅意](https://zh.wikipedia.org/wiki/路易·德布羅意)（Louis de Broglie）因其物質波理論獲得了諾貝爾物理學獎，標誌著量子力學的又一重大勝利。
