# 打字機的普及

## 辦公革命的深化

![打字機](https://upload.wikimedia.org/wikipedia/commons/thumb/9/9d/Remington_Noiseless_portable.jpg/220px-Remington_Noiseless_portable.jpg)

1917 年，打字機已成為大多數辦公室的標準設備。不僅如此，打字機開始進入更多家庭，成為女性的就業工具和個人的寫作設備。

## 一、辦公室普及

**標配設備**：
打字機與電話、文件櫃一起成為辦公室的三大件。

**效率提升**：
打字員可以比手寫快數倍，提高工作效率。

## 二、女性就業

**打字員群體**：
1917 年，打字員幾乎全是女性。她們通常年輕、單身、工資較低。

**職業培訓**：
許多女子學校和職業學校都設有打字課程。

**社會影響**：
打字員職業為女性打開了進入職場的大門，促進了女性經濟獨立。

## 三、技術持續改進

**鍵盤設計**：
QWERTY 鍵盤在 1917 年已完全標準化。

**字體選擇**：
多種字體可選，如 Courier、Elite 等。

**便攜式打字機**：
輕便型打字機開始出現，適用於外出使用。

## 四、社會影響

| 領域 | 影響 |
|------|------|
| 商業 | 提高效率、規範文書 |
| 法律 | 合同和文件的標準化 |
| 新聞 | 提高稿件處理速度 |
| 個人寫作 | 作家開始使用打字機 |
| 教育 | 打字段成為實用技能 |

## 五、戰時應用

**政府和軍事**：
一次大戰期間，大量文件需要處理，打字員需求增加。

**女性就業機會**：
戰爭期間男性參戰，女性填補了大量文書崗位。
