# 原子核物理學的萌芽

## 核物理的先驅研究

![原子核結構](https://upload.wikimedia.org/wikipedia/commons/thumb/b/bc/Atom_diagram.png/220px-Atom_diagram.png)

1917 年，原子核物理學還處於萌芽階段。盧瑟福在 1911 年提出了原子核模型，但對原子核內部結構的認識才刚刚開始。

## 一、盧瑟福模型

1911 年，盧瑟福根據α粒子散射實驗，提出了原子的核式結構模型：

**核心思想**：
原子由一個帶正電的原子核和環繞其外的電子組成。原子核佔據原子質量的絕大部分，但體積極小。

**穩定性問題**：
經典電動力學預言，電子在繞核運動時會輻射能量，最終墜入原子核。這個矛盾直到量子力學誕生才被解決。

## 二、質子與電子

1917 年，人們對原子核組成的認識還很有限：

**質子**：
1919-1920 年，盧瑟福用α粒子轟擊氮原子核，實現了第一次人為核反應。這一發現確認了質子是原子核的組成部分。

**原子量與原子序號的不匹配**：
如果原子核只由質子組成，那麼原子量應該等於原子序號。但實際上，大多數原子的原子量是原子序號的兩倍左右。

**猜測中的中子**：
物理學家開始猜測原子核中可能存在電中性的粒子。直到 1932 年查德威克才真正發現中子。

## 三、人工核反應的先聲

1917 年，盧瑟福開始進行用α粒子轟擊各種元素的實驗。

**實驗設想**：
如果α粒子的能量足夠高，它可能穿透原子核，引發核反應。

**技術條件**：
當時的加速器技術還很落後，只能使用天然放射性物質發射的α粒子作為「炮彈」。

## 四、核物理學的方法論

1917 年的核物理學研究依賴以下方法：

**α粒子散射**：
用α粒子轟擊薄箔，觀察散射圖案，推斷原子核結構。

**能譜分析**：
測量放射性物質發射的α粒子的能量分佈。

**雲霧室**：
威爾遜發明的雲霧室可以讓研究者「看見」帶電粒子的軌跡。

## 五、核能的理論預言

1917 年，愛因斯坦的質能關係式 E=mc² 已經發表了十多年。

**質量虧損**：
當核子組合成原子核時，會有質量虧損。這部分質量轉化為能量。

**核結合能**：
核結合能是組裝原子核所需的能量，也是核反應釋放能量的上限。

這些理論為日後的核能利用奠定了基礎。

## 六、核物理學的未來

1917 年的核物理學雖然還很初步，但已展示了廣闘的前景：

| 年份 | 發展 |
|------|------|
| 1919 | 盧瑟福實現第一次人工核反應 |
| 1932 | 查德威克發現中子 |
| 1938 | 哈恩和斯特拉斯曼發現核裂變 |
| 1942 | 費米建成第一個核反應堆 |
| 1945 | 第一顆原子彈 |
| 1954 | 第一座核電站 |
