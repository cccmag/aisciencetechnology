# 爱迪生电灯系统：城市的光明支柱

**科學技術年鑑　1905 年号**

编辑：[陈钟诚](https://csie.nqu.edu.tw/p/404-1038-2377.php?Lang=zh-tw)

---

## 引言

西元1905年，[电灯](https://zh.wikipedia.org/wiki/电灯)照明技术已经在美国和欧洲的主要城市得到广泛应用。[托马斯·阿尔瓦·爱迪生](https://zh.wikipedia.org/wiki/托马斯·爱迪生)（Thomas Edison）发明的[白炽灯](https://zh.wikipedia.org/wiki/白炽灯)和[直流电力系统](https://zh.wikipedia.org/wiki/直流电)，正在改变人类的生活方式。

## 爱迪生的发明之路

### 碳丝灯泡的突破

1879年，[爱迪生](https://zh.wikipedia.org/wiki/托马斯·爱迪生)成功制作出第一盏实用的[白炽灯](https://zh.wikipedia.org/wiki/白炽灯)：

1. 使用碳化竹丝作为灯丝
2. 寿命可达1200小时
3. 适合商业化生产

### 完整的电力系统

爱迪生不只是一个发明家，更是一个系统工程师：

- **[发电机](https://zh.wikipedia.org/wiki/发电机)**：生产电力
- **[输电线路](https://zh.wikipedia.org/wiki/输电线路)**：输送电力到用户
- **[电灯](https://zh.wikipedia.org/wiki/电灯)**：将电力转化为光
- **[开关插座](https://zh.wikipedia.org/wiki/开关插座)**：控制电路

1882年，纽约[珍珠街发电站](https://zh.wikipedia.org/wiki/珍珠街发电站)开始运营，这是世界上第一个商业[直流电力系统](https://zh.wikipedia.org/wiki/直流电)。

## 1905年的电灯技术

### 白炽灯的改进

到1905年，[白炽灯](https://zh.wikipedia.org/wiki/白炽灯)技术已经有了显著进步：

| 参数 | 1879年 | 1905年 |
|------|--------|--------|
| 光效 | 1.4流明/瓦 | 5-10流明/瓦 |
| 寿命 | 约40小时 | 约1000小时 |
| 灯丝材料 | 碳化竹丝 | 钨丝 |

### 钨丝灯泡的发明

1905年，[威廉·库兴·范孙](https://zh.wikipedia.org/wiki/威廉·库兴·范孙)（William Coolidge）发明了[钨丝灯泡](https://zh.wikipedia.org/wiki/钨)，这标志着[电灯](https://zh.wikipedia.org/wiki/电灯)技术的一个重要里程碑：

- 钨的熔点高达3422°C
- 光效更高
- 寿命更长

## 直流电力系统

### 系统架构

1905年的[爱迪生电力系统](https://zh.wikipedia.org/wiki/直流电)包括：

```
发电站 → 变压器 → 输电线路 → 配电变压器 → 用户
```

### 电压等级

| 应用 | 电压 |
|------|------|
| 家庭用电 | 110伏 |
| 工业用电 | 220-440伏 |
| 输电线路 | 几百伏到几千伏 |

### 珍珠街系统的扩展

到1905年，纽约的[爱迪生电力公司](https://zh.wikipedia.org/wiki/爱迪生电力公司)已经：

- 拥有多个发电站
- 覆盖曼哈顿大部分地区
- 服务数万户家庭

## 电灯照明的社会影响

### 城市夜景的改变

[电灯](https://zh.wikipedia.org/wiki/电灯)的普及彻底改变了城市面貌：

1. **夜间活动时间延长**：商店可以营业到深夜
2. **犯罪率降低**：街道照明减少夜间犯罪
3. **城市扩张**：照明使郊区也成为宜居之地
4. **工业生产增加**：可以实行轮班制生产

### 生活方式的改变

| 领域 | 变化 |
|------|------|
| [家庭生活](https://zh.wikipedia.org/wiki/家庭) | 夜间活动更加丰富 |
| [商业](https://zh.wikipedia.org/wiki/商业) | 营业时间延长 |
| [工业](https://zh.wikipedia.org/wiki/工业) | 两班制、三班制成为可能 |
| [文化](https://zh.wikipedia.org/wiki/文化) | 夜间娱乐场所出现 |

### 城市基础设施的建设

电灯照明系统需要配套基础设施：

- **[发电厂](https://zh.wikipedia.org/wiki/发电厂)**：集中生产电力
- **[电网](https://zh.wikipedia.org/wiki/电网)**：输配电力
- **[电杆线路](https://zh.wikipedia.org/wiki/架空电缆)**：空中布线
- **[路灯](https://zh.wikipedia.org/wiki/路灯)**：城市公共照明

## 与交流电的竞争

### 直流电vs交流电

[爱迪生](https://zh.wikipedia.org/wiki/托马斯·爱迪生)的[直流电系统](https://zh.wikipedia.org/wiki/直流电)与[尼古拉·特斯拉](https://zh.wikipedia.org/wiki/尼古拉·特斯拉)的[交流电系统](https://zh.wikipedia.org/wiki/交流电)展开了激烈竞争：

| 特性 | 直流电 | 交流电 |
|------|--------|--------|
| 电压变换 | 困难 | 容易（变压器） |
| 输电距离 | 短（几英里） | 长（可跨城市） |
| 安全性 | 较高 | 需要防护 |
| 电动机 | 简单 | 需要特殊设计 |

### 交流电的胜利

尽管爱迪生极力推广直流电，但[交流电](https://zh.wikipedia.org/wiki/交流电)凭借以下优势胜出：

1. **长距离输电**：可以使用高电压小电流
2. **变压器**：便于电压转换
3. **电动机**：结构简单可靠

到1905年，[交流电系统](https://zh.wikipedia.org/wiki/交流电)已经开始占据主导地位。

## 历史意义与未来发展

### 电灯照明的技术传承

| 年代 | 技术 | 意义 |
|------|------|------|
| 1879 | [碳丝灯泡](https://zh.wikipedia.org/wiki/白炽灯) | 实用化开始 |
| 1905 | [钨丝灯泡](https://zh.wikipedia.org/wiki/钨) | 效率提升 |
| 1930s | [荧光灯](https://zh.wikipedia.org/wiki/荧光灯) | 新型光源 |
| 1960s | [卤素灯](https://zh.wikipedia.org/wiki/卤素灯) | 高强度光源 |
| 2010s | [LED灯](https://zh.wikipedia.org/wiki/发光二极管) | 固态照明革命 |

### 对后续技术的影响

电灯照明技术的发展推动了：

1. **[电气化](https://zh.wikipedia.org/wiki/电气化)**：家庭和工业用电普及
2. **[家用电器](https://zh.wikipedia.org/wiki/家用电器)**：吸尘器、冰箱等出现
3. **[城市电气化](https://zh.wikipedia.org/wiki/城市电气化)**：基础设施现代化
4. **[电力工业](https://zh.wikipedia.org/wiki/电力工业)**：大型企业形成

## 结语

1905年的[电灯系统](https://zh.wikipedia.org/wiki/电灯)代表了[电气时代](https://zh.wikipedia.org/wiki/电气时代)的到来。[爱迪生](https://zh.wikipedia.org/wiki/托马斯·爱迪生)创造的直流电力系统虽然最终被交流系统取代，但他点燃了人类走向光明之路的第一盏灯。

从蜡烛到[白炽灯](https://zh.wikipedia.org/wiki/白炽灯)，从[直流电](https://zh.wikipedia.org/wiki/直流电)到[交流电](https://zh.wikipedia.org/wiki/交流电)，电灯照明技术的每一步发展都在推动人类文明的进步。
