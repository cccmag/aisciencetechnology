# 早期电话交换机：通信网络的基础

**科學技術年鑑　1905 年号**

编辑：[陈钟诚](https://csie.nqu.edu.tw/p/404-1038-2377.php?Lang=zh-tw)

---

## 引言

西元1905年，[电话](https://zh.wikipedia.org/wiki/电话)通信技术已经快速发展了将近三十年。[亚历山大·格雷厄姆·贝尔](https://zh.wikipedia.org/wiki/亚历山大·格雷厄姆·贝尔)（Alexander Graham Bell）在1876年发明电话后，电话网络开始在全球范围内扩展，而[电话交换机](https://zh.wikipedia.org/wiki/电话交换机)是这一网络的关键组成部分。

## 电话交换机的起源

### 贝尔的发明

1876年，[贝尔](https://zh.wikipedia.org/wiki/亚历山大·格雷厄姆·贝尔)获得了[电话](https://zh.wikipedia.org/wiki/电话)的专利：

> "Mr. Watson, come here, I want to see you."

这是电话发明史上著名的第一句话。

### 人工交换机的诞生

最早的电话交换是由[人工操作](https://zh.wikipedia.org/wiki/人工电话交换机)完成的：

1. 用户摇动电话机的手摇发电机
2. 交换机的话务员应答
3. 话务员用跳线连接双方线路
4. 通话结束后断开连接

## 1905年的交换机技术

### 人工交换机的改进

到1905年，[人工电话交换机](https://zh.wikipedia.org/wiki/人工电话交换机)已经有了显著改进：

| 组件 | 功能 | 改进 |
|------|------|------|
| [交换机](https://zh.wikipedia.org/wiki/电话交换机) | 连接线路 | 容量更大 |
| [话务员座椅](https://zh.wikipedia.org/wiki/话务员) | 人体工程学 | 更舒适 |
| [塞绳](https://zh.wikipedia.org/wiki/塞绳) | 信号连接 | 更耐用 |
| [插孔](https://zh.wikipedia.org/wiki/插孔) | 线路接口 | 可靠性提高 |

### 步进式交换机的发明

1905年左右，[史端乔](https://zh.wikipedia.org/wiki/阿尔蒙·斯特罗格)（Almon Strowger）发明了[步进式交换机](https://zh.wikipedia.org/wiki/步进式电话交换机)，这是世界上最早的[自动电话交换机](https://zh.wikipedia.org/wiki/自动电话交换机)：

- 使用电磁开关代替人工操作
- 用户直接拨号
- 提高了接通效率

### 旋转拨号盘的发明

[旋转拨号盘](https://zh.wikipedia.org/wiki/旋转拨号盘)的出现使得用户可以直接拨号：

1. 用户转动拨号盘
2. 拨号盘产生电脉冲
3. 交换机接收并处理信号
4. 自动建立连接

## 电话网络架构

### 网络层次结构

1905年的[电话网络](https://zh.wikipedia.org/wiki/电话网络)采用分级结构：

```
用户电话机 → 本地交换机 → 长途交换机 → 目的本地交换机 → 对方用户
```

### 线路类型

| 类型 | 用途 | 距离 |
|------|------|------|
| 用户线 | 连接用户与交换机 | 几公里 |
| 中继线 | 连接不同交换机 | 几十公里 |
| 长途线 | 城市间通信 | 上百公里 |

### 信号系统

早期电话使用[磁铃](https://zh.wikipedia.org/wiki/电铃)和[手摇发电机](https://zh.wikipedia.org/wiki/手摇发电机)：

- **振铃信号**：呼叫对方
- **摘机信号**：表示应答
- **挂机信号**：表示通话结束

## 技术局限性

### 1905年的技术限制

当时电话系统面临诸多局限：

- **容量有限**：每个交换机能处理的线路数量有限
- **音质不佳**：噪音和失真问题
- **覆盖范围**：长途通话质量差
- **成本高昂**：安装和维护费用高

### 人工操作的效率问题

[人工交换机](https://zh.wikipedia.org/wiki/人工电话交换机)的主要问题：

1. 需要大量话务员
2. 连接速度慢
3. 人为错误多
4. 无法24小时服务

## 对社会的影响

### 商业通信革命

[电话](https://zh.wikipedia.org/wiki/电话)改变了商业运作方式：

1. **即时通信**：无需等待信件
2. **客户服务**：可以电话咨询
3. **商业谈判**：可以电话会议
4. **订单处理**：可以电话下单

### 社交方式的改变

| 领域 | 变化 |
|------|------|
| [家庭](https://zh.wikipedia.org/wiki/家庭) | 亲友联系更便捷 |
| [医疗](https://zh.wikipedia.org/wiki/医疗) | 可以电话问诊 |
| [政府](https://zh.wikipedia.org/wiki/政府) | 行政效率提高 |
| [新闻](https://zh.wikipedia.org/wiki/新闻) | 信息传递更快 |

### 城市发展

电话网络促进了[城市化](https://zh.wikipedia.org/wiki/城市化)：

- 商业中心需要通信设施
- 城市规划要考虑线路
- 电信基础设施投资增加

## 历史意义与未来发展

### 电话交换机的技术演进

| 年代 | 技术 | 意义 |
|------|------|------|
| 1878 | [人工交换机](https://zh.wikipedia.org/wiki/人工电话交换机) | 电话网络基础 |
| 1892 | [史端乔交换机](https://zh.wikipedia.org/wiki/步进式电话交换机) | 自动交换开始 |
| 1930s | [纵横制交换机](https://zh.wikipedia.org/wiki/纵横制电话交换机) | 更高效可靠 |
| 1960s | [程控交换机](https://zh.wikipedia.org/wiki/程控电话交换机) | 电子化控制 |
| 1990s | [数字交换机](https://zh.wikipedia.org/wiki/数字电话交换机) | 全数字化 |
| 2000s | [VoIP](https://zh.wikipedia.org/wiki/网络电话) | 互联网电话 |

### 对后续技术的影响

电话交换机技术的发展为以下技术奠定了基础：

1. **[数据通信](https://zh.wikipedia.org/wiki/数据通信)**：网络协议发展
2. **[互联网](https://zh.wikipedia.org/wiki/互联网)**：网络基础设施
3. **[移动通信](https://zh.wikipedia.org/wiki/移动通信)**：蜂窝网络
4. **[云计算](https://zh.wikipedia.org/wiki/云计算)**：网络服务

## 结语

1905年的[电话交换机](https://zh.wikipedia.org/wiki/电话交换机)技术代表了[通信网络](https://zh.wikipedia.org/wiki/通信网络)的开始。从人工交换到自动交换，从模拟信号到数字信号，电话技术走过了漫长的道路。

[步进式交换机](https://zh.wikipedia.org/wiki/步进式电话交换机)的发明标志着自动化的开始，这为后来[程控交换机](https://zh.wikipedia.org/wiki/程控电话交换机)和[数字交换机](https://zh.wikipedia.org/wiki/数字电话交换机)的发展铺平了道路。
