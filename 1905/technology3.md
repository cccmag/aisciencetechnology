# 早期飞机的研发：莱特兄弟的持续实验

**科學技術年鑑　1905 年号**

编辑：[陈钟诚](https://csie.nqu.edu.tw/p/404-1038-2377.php?Lang=zh-tw)

---

## 引言

西元1905年，是人类追求飞行梦想的关键一年。[莱特兄弟](https://zh.wikipedia.org/wiki/莱特兄弟）——[威尔伯·莱特](https://zh.wikipedia.org/wiki/威尔伯·莱特）和[奥维尔·莱特](https://zh.wikipedia.org/wiki/奥维尔·莱特）在这一年进行了持续的[飞机](https://zh.wikipedia.org/wiki/飞机)实验，为两年后的历史性飞行奠定了坚实的基础。

## 莱特兄弟的飞行探索

### 从滑翔机到动力飞机

莱特兄弟从1899年开始从事[航空研究](https://zh.wikipedia.org/wiki/航空学)。他们首先研究[滑翔机](https://zh.wikipedia.org/wiki/滑翔机)，系统地进行风洞实验，积累了大量空气动力学数据。

莱特兄弟的研究方法极为严谨：

1. **风洞实验**：制作小型风洞，测试不同翼型的升力和阻力特性
2. **飞行记录**：用相机记录每次飞行的姿态和结果
3. **系统分析**：详细记录风速、翼面积、飞行速度等参数
4. **持续改进**：根据实验结果不断优化设计

### 1903年的历史性突破

1903年12月17日，[莱特兄弟](https://zh.wikipedia.org/wiki/莱特兄弟）在北卡罗来纳州基蒂霍克进行了人类历史上首次有动力、可控制的[飞机飞行](https://zh.wikipedia.org/wiki/莱特兄弟)。

当天进行的飞行试验：

| 飞行次序 | 飞行员 | 距离 | 时间 |
|----------|--------|------|------|
| 第一次 | 奥维尔 | 36米 | 12秒 |
| 第二次 | 威尔伯 | 53米 | 15秒 |
| 第三次 | 奥维尔 | 61米 | 15秒 |
| 第四次 | 威尔伯 | 260米 | 59秒 |

最后一次飞行由威尔伯驾驶，飞行了260米，持续约59秒。这是人类历史上首次实现可操控的动力飞机飞行。

## 1905年的持续改进

### 莱特飞行者三号

1905年，莱特兄弟研制出了[莱特飞行者三号](https://zh.wikipedia.org/wiki/莱特飞行者三号)，这是世界上第一架实用化的动力飞机。

飞行者三号的特点：

- **机翼上反角**：增加横向稳定性
- **双翼结构**：提供更大的升力
- **可操控的副翼**：实现精确的滚转控制
- **尾翼设计**：改善俯仰和偏航稳定性
- **强化的起落架**：适应各种地面条件

### 1905年的飞行试验

1905年10月5日，莱特兄弟进行了最为成功的飞行试验：

- 地点：俄亥俄州霍毅草原
- 飞行员：奥维尔·莱特
- 飞行距离：38.6公里
- 飞行时间：39分23秒
- 最高时速：约61公里

这次飞行证明了莱特兄弟的飞机已经具备实用性。

### 技术突破的关键

莱特兄弟能够成功的关键在于：

1. **系统性的研究方法**：不像其他飞行先驱那样盲目尝试
2. **精确的风洞实验**：获得可靠的空气动力学数据
3. **三轴控制系统**：解决了飞机在三维空间中的控制问题
4. **实际试飞经验**：从多次失败中学习和改进

## 其他飞行先驱

### 竞争对手

在莱特兄弟进行实验的同时，世界上还有其他飞行先驱也在努力：

- **奥古斯特·桑德尔·弗莱明**：德国航空先驱
- **路易·布莱里奥**：法国航空先驱，1909年首次飞越英吉利海峡
- **格伦·柯蒂斯**：美国航空先驱，后来成为著名飞机设计师

### 莱特兄弟的低调

与其他航空先驱不同，莱特兄弟在成功後保持了相当低调的态度。他们：

- 没有立即公开表演
- 对自己的技术保密
- 等待合适的商业时机

这种做法虽然延迟了飞机技术的传播，但也保护了他们的知识产权。

## 飞机研发的科学技术基础

### 空气动力学原理

莱特兄弟的成功，建立在对[空气动力学](https://zh.wikipedia.org/wiki/空气动力学)的深入理解之上：

| 原理 | 应用 |
|------|------|
| [升力原理](https://zh.wikipedia.org/wiki/升力) | 机翼设计 |
| [阻力原理](https://zh.wikipedia.org/wiki/阻力) | 减少空气阻力 |
| [伯努利原理](https://zh.wikipedia.org/wiki/伯努利定律) | 理解压力分布 |
| [俯仰/滚转/偏航](https://zh.wikipedia.org/wiki/飞行力学) | 飞机控制 |

### 工程技术的进步

莱特兄弟的飞机融合了多项工程技术：

- **轻量化结构**：使用木头和布料减少重量
- **高功率发动机**：专门为飞机设计的汽油发动机
- **精密制造**：确保各部件的配合精度
- **材料科学**：选择合适的布料和胶水

## 对未来的深远影响

莱特兄弟的工作，为现代[航空工业](https://zh.wikipedia.org/wiki/航空工业)奠定了基础：

| 时间 | 发展 |
|------|------|
| 1905年 | 莱特兄弟持续改进飞机技术 |
| 1908年 | 莱特飞机获美军采用 |
| 1910年 | 商业航空的萌芽 |
| 1914年 | 第一次世界大战促进航空工业发展 |
| 1927年 | 查尔斯·林德伯格首次单人不着陆跨越大西洋 |
| 1947年 | 查克·叶格突破音障 |
| 1969年 | 阿波罗11号登月 |
| 至今 | 商业航空、太空探索、无人机技术 |

## 结语

1905年，[莱特兄弟](https://zh.wikipedia.org/wiki/莱特兄弟）的工作标志着人类航空事业进入了崭新的阶段。从1903年的历史性首飞，到1905年的持续改进和成功实验，莱特兄弟用系统性的科学方法和不懈的努力，证明了人类飞行梦想的可行性。

莱特兄弟的成功，不仅是工程技术的胜利，更是科学方法的胜利。他们通过风洞实验、系统记录、持续改进的方法，为后世的航空研究树立了典范。

如今，从巨型客机到太空梭，从无人机到电动垂直起降航空器，航空技术的发展可谓日新月异。这一切都可以追溯到一百二十多年前莱特兄弟在北卡罗来纳海滩上的那一次次尝试。
