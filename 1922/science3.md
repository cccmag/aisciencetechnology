# 光譜線細微結構的發現

## 一、光譜學的興起

光譜學是19世紀末至20世紀初物理學最重要的研究領域之一。當元素被加熱或通電時，會發出特定波長的光，形成獨特的光譜線圖案，如同元素的「指紋」。通過分析光譜，科學家可以確定未知物質的化學成分。

19世紀中期，本生和基爾霍夫建立了光譜分析的基本方法，發現了銫、銣等多種新元素。此後，光譜學成為天文學、化學、物理學的重要工具。1868年，太陽元素氦就是通過光譜分析發現的。

## 二、氫原子光譜的規律

1885年，巴爾末發現氫原子可見光譜的波长可以用一個簡單的公式表示：λ = B n²/(n² - 4)，其中 n = 3, 4, 5, ...。這就是著名的巴爾末公式。1889年，黎德堡推廣了這個公式，提出了更一般的光譜線公式：ν = R(1/m² - 1/n²)。

這些經驗公式的存在表明，氫原子的能量狀態是量子化的。玻爾模型成功從理論上推導出這些公式，解釋了光譜線的離散性。巴爾末系對應電子從 n>2 的軌道躍遷到 n=2 的軌道，萊曼系對應躍遷到 n=1。

## 三、精細結構的發現

然而，隨著測量技術的進步，物理學家發現光譜線並非完全單一。事實上，每條光譜線都由幾條非常靠近的線組成，這就是光譜線的「精細結構」。這種現象是玻爾模型無法解釋的。

1892年，迈克耳孫通過精確干涉測量發現鈉雙線（589.0納米和589.6納米）實際上並非單一譜線，而是靠得非常近的兩條線。這標誌著精細結構研究的開始。此後，用更高分辨率的光譜儀，越來越多的精細結構被發現。

## 四、索莫菲的貢獻

1916年，索莫菲對精細結構提出了第一個成功的理論解釋。他認識到電子在原子核周圍的橢圓軌道運動會產生相對論效應，導致軌道能量發生細微變化。索莫菲引入了兩個量子數——主量子數 n 和角量子數 l，來描述電子的運動狀態。

根據索莫菲的理論，由於相對論效應和橢圓軌道的存在，即使對於相同的主量子數 n，不同角量子數 l 的軌道也具有略微不同的能量。這就解釋了光譜線的精細結構。

## 五、對量子力學的推動

精細結構的研究推動了量子理論的進一步發展。索莫菲的模型雖然成功解釋了觀測到的精細結構，但它仍然是一個半經典的理論。真正的突破要等到量子力學的建立。

1926年，薛定諤的波動力學能夠自動產生出相對論效應，無需額外的假設。狄拉克進一步將量子力學與狹義相對論結合，建立了相對論性量子力學，完美地解釋了光譜線的精細結構和蘭姆位移等現象。
