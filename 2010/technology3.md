# 雲端運算持續普及：AWS 領導地位鞏固

2010 年，[雲端運算](https://zh.wikipedia.org/wiki/雲端運算)市場持續增長，Amazon Web Services (AWS) 鞏固市場領導地位，雲端服務成為企業 IT 的主流選擇。

## AWS 發展

### 主要服務

[AWS](https://zh.wikipedia.org/wiki/Amazon_Web_Services) 2010 年服務：
| 服務 | 類型 | 功能 |
|------|------|------|
| EC2 | IaaS | 雲端伺服器 |
| S3 | 存儲 | 雲端存儲 |
| RDS | 資料庫 | 關係資料庫 |
| VPC | 網路 | 虛擬私有雲 |

### 市場地位

AWS 的領導地位：
- 雲端先驅
- 持續創新
- 價格持續下降
- 企業採用

雲端市場：

雲端採用：
