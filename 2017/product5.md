# DJI Spark：掌心無人機

## 概述

2017年5月，大疆創新發布了DJI Spark「曉」無人機，這款號稱「會飛的自拍神器」的迷你無人機，專為入門用戶設計。Spark以其小巧的尺寸、手勢控制和親民的價格，讓航拍從專業領域走進了大眾消費市場。

![DJI Spark](https://upload.wikimedia.org/wikipedia/commons/thumb/e/e9/DJI_Spark.jpg/800px-DJI_Spark.jpg)

## 一、產品設計

### 1.1 小巧機身

Spark的尺寸與重量：

| 參數 | 數值 |
|------|------|
| 尺寸 | 143×143×55 mm |
| 起飛重量 | 300g |
| 折疊後 | 巴掌大小 |
| 槳葉直徑 | 7.6 cm |

### 1.2 機身配色

Spark提供多種顏色選擇：
- 雪山白
- 檯石青
- 草地綠
- 天空藍
- 向陽黃

### 1.3 快速起飛

Spark的起飛方式：
- **掌上升空**：放置掌心，識別人臉後起飛
- **手勢控制**：無需手機或遙控器
- **一鍵起飛**：點擊按鈕，自動起飛

## 二、相機系統

### 2.1 拍攝能力

Spark的相機參數：

| 參數 | 數值 |
|------|------|
| 感測器 | 1/2.3" CMOS |
| 有效像素 | 12MP |
| 鏡頭視角 | 81.9° |
| 光圈 | f/2.6 |
| 視頻 | 1080p@30fps |

### 2.2 拍攝模式

Spark提供的智能拍攝模式：
- **一鍵短片**：漸遠、螺旋、衝天、環繞
- **掌聲控制**：拍手機身控制拍攝
- **跟蹤**：自動跟隨目標
- **興趣點環繞**：繞指定點飛行

### 2.3 穩定性

Spark的飛行穩定性：
- **GPS定位**：精確定位
- **視覺定位**：室內也能穩定懸停
- **風抗能力**：4級風以下穩定飛行

## 三、手勢控制

### 3.1 創新的交互方式

Spark的手勢控制功能：
- **起飛/降落**：掌心升空，手掌降落
- **控制移動**：手掌左右移動控制方向
- **拍照**：比「拍照」手勢，2秒延時拍攝
- **召回**：張開雙臂，無人機飛回

### 3.2 技術原理

手勢控制的實現：
- **前視障礙感知**：雙目視覺感應
- **紅外感知**：下方紅外感應
- **晶片處理**：快速識別手勢

### 3.3 用戶體驗

手勢控制的優勢：
- 無需學習遙控器
- 適合社交場合
- 簡單直觀

## 四、續航與飛行

### 4.1 飛行時間

Spark的續航能力：
| 參數 | 數值 |
|------|------|
| 最長飛行時間 | 16分鐘 |
| 最長懸停時間 | 15分鐘 |
| 最大水準速度 | 50 km/h |

### 4.2 飛行控制距離

Spark的控制距離：
- **WiFi控制**：100米（手機）
- **遙控器控制**：2公里（另購）
- **姿態模式**：室內無GPS環境

### 4.3 安全功能

Spark的安全保護：
- **地理圍欄**：禁飛區限制
- **自動返航**：低電量或失控時自動返航
- **FlightAutonomy系統**：精確定位和障礙感知

## 五、對無人機市場的影響

### 5.1 入門級無人機的標杆

Spark重新定義了入門級無人機：
- **價格親民**：預售价3,299元
- **容易上手**：無需培訓
- **便攜**：隨身攜帶

### 5.2 擴大用戶群

Spark吸引的新用戶：
- **普通消費者**：而非專業飛手
- **女性用戶**：輕便、時尚
- **年輕群體**：社交分享為主

### 5.3 後續產品

Spark的成功催生了更多型號：
- **DJI Mavic Air**（2018）：更強的便攜性
- **DJI Mavic Mini**（2019）：更輕的249g
- **DJI Mini 2**（2020）：性能增強
