# 特斯拉Semi卡車與Roadster跑車

## 概述

2017年11月，特斯拉發布了Semi全電動卡車和第二代Roadster跑車。Semi展示了電動汽車在商用車領域的潛力，而新Roadster則以2秒的0-100km/h加速時間創下了量產車的世界紀錄。

![特斯拉卡車](https://upload.wikimedia.org/wikipedia/commons/thumb/a/a0/Tesla_Semi.jpg/800px-Tesla_Semi.jpg)

## 一、Tesla Semi卡車

### 1.1 發布與規格

Tesla Semi於2017年11月在美國加州發布：

| 參數 | 數值 |
|------|------|
| 續航里程 | 500英里（800公里）或300英里 |
| 0-60mph（滿載） | 20秒 |
| 拖運重量 | 36,000公斤 |
| 風阻係數 | 0.36 Cd |
| 售價 | 15-18萬美元（預估） |

### 1.2 技術特點

Semi的創新技術：
- **增強型Autopilot**：自動緊急制動、車道保持
- **車隊模式**：多輛Semi自動跟隨行駛
- **超級快充**：30分鐘充滿80%
- **無齒輪箱設計**：直接驅動

### 1.3 商業意義

Semi對貨運行業的影響：
- 降低運營成本（電費 vs 柴油）
- 減少維護需求（電動機几乎不需維護）
- 提升司機工作環境

## 二、第二代Roadster跑車

### 2.1 規格與性能

第二代Roadster的震撼規格：

| 參數 | 數值 |
|------|------|
| 0-60 mph | 1.9秒 |
| 0-100 mph | 4.2秒 |
| 最高時速 | 超過250 mph（400+ km/h） |
| 續航里程 | 620英里（1000公里） |
| 扭矩 | 10,000 Nm |

### 2.2 設計亮點

Roadster的設計特點：
- 敞篷跑車設計
- 四人座（後座空間有限）
- 可拆卸車頂
- 極致的空氣動力學

### 2.3 預售情況

Roadster的預售：
- 定金：5萬美元
- 創始人版本：25萬美元
- 訂單金額：早期就已超過1億美元

## 三、技術創新

### 3.1 电池技術

特斯拉在电池方面的突破：
- **2170电池**：比之前的18650更大、更高效
- **Gigafactory**：超級工廠規模化生產
- **BMS系統**：精密的電池管理

### 3.2 自動駕駛

兩款車型都配備自動駕駛硬體：
- 8個攝像頭
- 12個超聲波感測器
- 前向雷達
- 完全自動駕駛能力（待監管批准）

### 3.3 充電網絡

特斯拉的充電基礎設施：
- **超級充電站網路**：全球數千個
- **Semi專用充電**：Megacharger網路
- **家用充電**：Wall Connector

## 四、對汽車產業的影響

### 4.1 商用車電動化

Semi加速了商用車的電動化進程：
- 沃爾瑪、UPS等大客戶下單
- 傳統卡車廠商加速電動卡車研發
- 戴姆勒、福斯等紛紛跟進

### 4.2 超跑市場的電動化

Roadster改變了超跑的遊戲規則：
- 加速性能超越傳統燃油超跑
- 環保與性能可以兼顧
- 為電動超跑設立標杆

### 4.3 產業生態

特斯拉的產品矩陣：
- **家用轎車**：Model S、Model 3
- **家用SUV**：Model X
- **商用卡車**：Semi
- **跑車**：Roadster
- **皮卡**：Cybertruck（後來發布）

## 五、交付與挑戰

### 5.1 交付延遲

特斯拉一貫的交付延遲問題：
- **Semi**：原定2019年交付，延遲至2022年
- **Roadster**：原定2020年交付，多次延遲
- 原因包括生產挑戰和供應鏈問題

### 5.2 生產挑戰

特斯拉面臨的挑戰：
- **產能瓶頸**：難以快速擴大產量
- **供應鏈**：部分零件供應不穩定
- **現金流**：持續虧損

### 5.3 市場反應

市場對兩款車型的反應：
- **Semi**：獲得大量訂單，商用前景看好
- **Roadster**：被視為電動汽車的技術展示
