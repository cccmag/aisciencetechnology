# AlphaGo Zero：無師自通的圍棋大師

## 概述

2017年10月19日，Google DeepMind在《自然》雜誌發表論文，宣布推出新一代AlphaGo系統——AlphaGo Zero。與之前的版本不同，AlphaGo Zero完全不需要人類棋譜，從空白狀態開始，通過純強化學習（Reinforcement Learning）在短短40天內達到了超越所有之前版本的水平。

![AlphaGo Zero](https://upload.wikimedia.org/wikipedia/commons/thumb/4/4f/AlphaGo_logo.svg/200px-AlphaGo_logo.svg.png)

## 一、從監督學習到無師自通

### 1.1 之前版本的局限

之前的AlphaGo版本（包括戰勝李世石的版本）的訓練分為兩個階段：

| 階段 | 數據來源 | 作用 |
|------|----------|------|
| 監督學習 | 人類職業棋譜 | 學習人類下法 |
| 強化學習 | 自我對弈 | 進一步提升 |

這種方法雖然成功，但存在根本缺陷：
- 需要大量人類棋譜數據
- 學習受人類知識限制
- 難以超越人類水平

### 1.2 AlphaGo Zero的創新

AlphaGo Zero從一開始就完全依靠強化學習：

| 特性 | AlphaGo Lee（對陣李世石版本） | AlphaGo Zero |
|------|------------------------------|--------------|
| 學習方式 | 人類棋譜 + 強化學習 | 純強化學習 |
| 網路結構 | 策略網路 + 價值網路 + 樹搜索 | 單一深度網路 |
| 訓練時間 | 數月 | 40天 |
| 對弈結果 | — | 100:0 |

### 1.3 為何稱為「Zero」？

Zero代表「從零開始」：
- 從空白狀態出發
- 不使用任何人類知識
- 完全從自我對弈中學習

## 二、技術原理

### 2.1 強化學習基礎

強化學習是一種通過「試錯」學習的範式：

```
智能體 → 行動 → 環境 → 觀察 → 獎勵 → 智能體
```

在圍棋中：
- 智能體：AlphaGo Zero
- 行動：選擇下一步
- 環境：圍棋規則
- 獎勵：最終獲勝（+1）或失敗（-1）

### 2.2 深度強化學習

AlphaGo Zero的核心是深度強化學習：

**單一神經網路**：
- 輸入：19×19的棋盤狀態
- 輸出：每個位置的概率分佈 + 獲勝機率
- 融合了之前策略網路和價值網路的功能

**自我對弈訓練**：
1. 網路當前版本與自己對弈
2. 根據棋局結果更新網路
3. 重复這個過程

### 2.3 蒙特卡羅樹搜索的簡化

AlphaGo Zero保留了MCTS，但进行了簡化：

| 版本 | MCTS特點 |
|------|----------|
| AlphaGo Lee | 使用策略網路指導搜索 |
| AlphaGo Zero | 使用神經網路直接指導搜索 |

## 三、驚人的結果

### 3.1 實力對比

AlphaGo Master（對陣柯潔版本）對比AlphaGo Zero：

| 對比項目 | AlphaGo Zero vs AlphaGo Lee |
|----------|----------------------------|
| 對弈比分 | 100:0 |
| 訓練效率 | 40天 vs 數月 |
| 所需的計算 | 更少 |
| 所需的人類知識 | 零 |

### 3.2 超越人類的「創意」

AlphaGo Zero不僅在實力上超越人類，還展示了新的圍棋「創意」：
- 發現了人類數百年來未曾注意的下法
- 走出了被認為是業餘水平的开局
- 證明人類的圍棋理論可能並非最優

### 3.3 學習曲線

AlphaGo Zero的學習曲線顯示：
- **第1天**：水平接近業餘高手
- **第3天**：超越最初的AlphaGo版本
- **第21天**：達到Master版本水平
- **第40天**：超越所有之前版本

## 四、通用性的開端

### 4.1 AlphaZero的誕生

受到AlphaGo Zero成功的啟發，DeepMind將同樣的方法推廣到其他遊戲：

**AlphaZero**：
- 國際象棋
- 將棋
- 圍棋

AlphaZero在24小時內達到了超越頂級程式的水平。

### 4.2 從圍棋到其他領域

AlphaGo Zero的方法論有潛力應用於：
- **科學研究**：發現新的數學定理
- **藥物發現**：設計新分子
- **遊戲設計**：創造新遊戲
- **優化問題**：物流、路徑規劃

### 4.3 通用人工智慧的曙光？

AlphaGo Zero展示了「無監督」學習的可能性：
- 不需要人工標註的數據
- 可以在完全陌生的環境中學習
- 這種能力被認為是邁向通用AI的重要步驟

## 五、深遠的影響

### 5.1 對AI研究的影響

AlphaGo Zero的方法正在被廣泛借鑒：
- 純強化學習成為研究熱點
- 自我對弈的方法被應用於其他領域
- 對「監督學習vs強化學習」的討論深入

### 5.2 對人類知識的反思

AlphaGo Zero的成果引發了深刻問題：
- 人類積累的知識是否都是最優的？
- AI是否會發現人類無法理解的新知識？
- 人類應如何與超越自己的AI相處？

### 5.3 開源的影響

2018年，DeepMind開源了AlphaGo Zero的技術細節，促進了AI社區的發展。
