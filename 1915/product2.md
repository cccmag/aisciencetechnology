# 收音機的興起

## 廣播時代的萌芽

![早期收音機](https://upload.wikimedia.org/wikipedia/commons/thumb/8/8e/Crystal_radio185.jpg/220px-Crystal_radio185.jpg)

1915 年，收音機技術正在從實驗室走向家庭。雖然商業廣播還要再等十年才會出現，但業餘愛好者已經開始用簡陋的設備接收信號。這項技術的發展，為日後的信息革命奠定了重要基礎。

## 一、收音機技術的發展

**真空管的改進**：
1904 年弗萊明髮明的整流二極管和 1906 年德弗雷斯特髮明的三極管，使收音機能夠放大微弱的信號。1915 年，這些真空管技術已經比較成熟。

**調諧技術**：
如何從眾多的廣播電台中選擇想听的節目是個技術難題。1915 年，可變電容器和線圈組成的調諧電路已能實現較好的選擇性。

**晶體收音機**：
簡易的晶體收音機、無需電源，適合業餘愛好者自制。

## 二、業餘收音的流行

1915 年，收音機主要是業餘愛好者的玩具。

**執餘電台**：
许多業餘愛好者設立了自己的電台，用摩爾斯電碼互相通訊。這些人被稱為「執餘電台」（Amateur Radio）。

**接收設備**：
有興趣的人可以購買零件自建收音機，或購買成套的「收音機套件」。

**戰時的限制**：
由於第一次世界大戰，美國政府於 1917 年禁止了業餘電台活動，直到 1919 年才恢復。

## 三、無線電通訊的商業應用

除了家庭收音機，1915 年的無線電技術主要用於商業和軍事通訊：

**船舶通訊**：
大型船舶普遍配備了無線電設備，可以用於緊急呼救和日常通訊。1912 年的泰坦尼克號救援行動促進了這一技術的普及。

**新聞服務**：
報紙開始使用無線電接收來自海外的新聞，提高新聞時效性。

**港口通信**：
港口與進港船舶之間建立了常規的無線電聯繫。

## 四、技術原理

收音機的工作原理涉及以下基本概念：

**調制**：
音頻信號需要「加載」到高頻載波上才能傳播。分為調幅（AM）和調頻（FM）。

**解調**：
接收端從載波中恢復原始的音頻信號。

**放大**：
利用真空管放大微弱的射頻信號，使其能被聽見。

```
音頻信號 → 調制 → 發射 → 傳播 → 接收 → 解調 → 放大 → 聲音
```

## 五、走向大眾

1915 年的收音機距離走入千家萬戶還有距離，但技術儲備正在快速積累。

**元器件的標準化**：
收音機零件開始標準化生產，降低了製造成本。

**技術文獻的普及**：
業餘愛好者雜誌和書籍開始出版，推廣收音機技術。

**技術人才的培養**：
大量的業餘愛好者成為了未來廣播電台的技術骨幹。

1920 年，美國首個商業廣播電台 KDKA 開播，標誌著廣播時代的正式開始。從 1915 年的技術儲備到 1920 年的商業繁榮，收音機經歷了漫長的孕育期。
