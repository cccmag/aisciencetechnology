# 歷史穿梭：1995 年的科學、技術與未來

**科學技術年鑑　1995 年號**

編輯：[陳鍾誠](https://csie.nqu.edu.tw/p/404-1038-2377.php?Lang=zh-tw)

---

1995 年是 [Windows 95](#) 時代和 [Amazon](#) 崛起的轉捩點。這一年標誌著個人電腦大眾化和電子商務時代的來臨。

## 1995 與 2026：我們有何不同？

| 1995 年的人 | 2026 年的我們 |
|------------|--------------|
| Windows 95 是最新系統 | 雲端和 AI 是主流 |
| Amazon 才成立 | Amazon 為全球最大電商之一 |
| 互聯網用戶約 4000 萬 | 互聯網用戶超過 50 億 |
| 網路購物剛興起 | 電子商務無處不在 |
| DVD 剛開始普及 | 串流媒體主宰市場 |

## 對未來的影響

| 1995 年 | 影響領域 | 對未來的影響 |
|--------|----------|--------------|
| Windows 95 | [個人電腦](https://zh.wikipedia.org/wiki/個人電腦) | [作業系統](https://zh.wikipedia.org/wiki/作業系統)、[雲端運算](https://zh.wikipedia.org/wiki/雲端運算) |
| Amazon | [電子商務](https://zh.wikipedia.org/wiki/電子商務) | [雲端服務](https://zh.wikipedia.org/wiki/雲端運算)、[AI](https://zh.wikipedia.org/wiki/人工智慧) |
| DVD 標準 | [家庭娛樂](https://zh.wikipedia.org/wiki/家庭娛樂) | [串流媒體](https://zh.wikipedia.org/wiki/串流媒體) |
| JavaScript | [網頁開發](https://zh.wikipedia.org/wiki/網頁設計) | [Web 應用](https://zh.wikipedia.org/wiki/網頁應用) |
| 頂夸克發現 | [粒子物理學](https://zh.wikipedia.org/wiki/粒子物理學) | [希格斯玻色子](https://zh.wikipedia.org/wiki/希格斯玻色子) |

## 結論

1995 年開啟了個人電腦大眾化和電子商務時代。這些趨勢在後續三十農間深刻改變了我們的社會和生活。
