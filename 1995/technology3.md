# JavaScript 語言發布：互動式網頁的誕生

1995 年，網景公司在 Navigator 瀏覽器中引入了 [JavaScript](https://zh.wikipedia.org/wiki/JavaScript) 腳本語言。這是一種可以在網頁中運行的程式語言，使網頁獲得了互動能力。

## JavaScript 的意義

[JavaScript](https://zh.wikipedia.org/wiki/JavaScript) 使網頁從靜態內容轉變為動態應用。這是 Web 開發史上的重大突破，為後續的 Ajax、jQuery、React 等技術奠定了基礎。

## 對網頁開發的影響

如今，JavaScript 是全球最流行的程式語言之一，几乎所有網站都使用 JavaScript。
