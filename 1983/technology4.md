# Compact Disc 音樂標準：數位音頻的革命

西元 1982 年，[索尼](https://zh.wikipedia.org/wiki/索尼)和[飛利蒲](https://zh.wikipedia.org/wiki/飛利浦)共同制定了 [CD](https://zh.wikipedia.org/wiki/CD)（Compact Disc，緊湊光碟）音頻標準。1983 年，CD 播放機和唱片開始大規模上市，這一革命性的音頻載體迅速改變了人們聆聽音樂的方式。

## CD 標準的制定

CD 標準的制定經歷了多年的研發和協調：

### 1979 年
飛利蒲在漢諾威博覽會上展示了 CD 原型。

### 1980 年
索尼和飛利浦共同發布「紅皮書」標準，定義了 CD 音頻格式。

### 1982 年
第一批 CD 唱片和播放器上市。

### 1983 年
CD 在市場上開始大規模普及。

## 技術規格

CD 音頻格式的技術規格：

| 項目 | 規格 |
|------|------|
| 音頻格式 | 16 位元 PCM |
| 採樣率 | 44.1 kHz |
| 頻率響應 | 20 Hz - 20 kHz |
| 動態範圍 | 96 dB |
| 播放時間 | 約 80 分鐘 |
| 光碟直徑 | 12 cm |
| 光碟厚度 | 1.2 mm |

## 技術創新

CD 相比黑膠唱片和卡帶具有多項創新：

### 數位錄製
從錄音到播放全程數位化，確保極高的聲音品質。

### 無接觸讀取
雷射光束非接觸讀取，不會磨損光碟。

### 隨機存取
可以立即跳转到任何曲目，無需倒帶或尋找。

### 零噪音
播放過程中沒有類比載體的底噪和爆音。

## 市場影響

CD 的普及對音樂產業产生了深远影響：

### 銷售成績
1983 年 CD 播放器销量突破 100 万台。

### 唱片轉換
大量唱片公司開始將其目錄轉換為 CD 格式。

### 價格下降
隨著產量增加，CD 和播放器的價格持續下降。

### 音樂愛好者
發燒友追捧 CD 的高品質聲音。

## 對後世的影響

CD 的發明對後續音頻技術產生了深遠影響：

### 數位音頻革命
CD 開創了數位音頻時代，為日後的 MP3、串流媒體奠定基礎。

### 音樂產業
從實體唱片到數位下載，CD 是過渡階段的關鍵產品。

### 存儲介質
CD 的技術被擴展用於數據存儲（CD-ROM）。

### 消費電子
CD 播放機成為家庭音響系統的標準配置。

## 歷史意義

CD 的發明是音頻技術從類比時代邁入數位時代的標誌。從 1983 年到 2000 年代，CD 统治了音樂市場近二十年，至今仍是音頻品質的參考標準。
