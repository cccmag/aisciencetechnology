# 蘋果 Lisa 電腦：圖形介面的先驅

西元 1983 年，[蘋果公司](https://zh.wikipedia.org/wiki/蘋果公司)推出 [Lisa](https://zh.wikipedia.org/wiki/Lisa_(電腦))，這是世界上第一款配備[圖形使用者介面](https://zh.wikipedia.org/wiki/圖形使用者介面)（GUI）的商業化個人電腦。Lisa 雖然在市場上並不成功，但它的技術創新為日後的 Macintosh 和整個 GUI 產業奠定了基礎。

## 產品的誕生

Lisa 的開發是蘋果公司的一項重大投資：

### 開發團隊
蘋果從施樂 PARC 挖來了多名工程師，包括比爾·艾奇森（Bill Atkinson）和蘇珊·卡恩（Susan Kare）。

### 開發時間
長達三年的開發週期，投入了約 5,000 萬美元。

### 產品定位
Lisa 定位於企業和專業用戶，售價 9,995 美元。

## 技術規格

Lisa 的硬體配置：

| 組件 | 規格 |
|------|------|
| 處理器 | Motorola 68000，5 MHz |
| 記憶體 | 1 MB（最大 2 MB） |
| 顯示器 | 12 吋 CRT，720×360 |
| 儲存 | 5 MB 硬碟 + 軟碟 |
| 作業系統 | Lisa OS |

## 技術創新

Lisa 帶來了多項重要的技術創新：

### 圖形使用者介面
- 視窗（Windows）
- 圖標（Icons）
- 功能表（Menus）
- 指標（Pointer）

### 桌面比喻
Lisa 的介面採用「桌面」（Desktop）的比喻，檔案和資料夾圖標直接擺放在桌面上。

### 先進功能
- 重疊式視窗
- 下拉式功能表
- 軌跡球控制指標
- 所見即所得（WYSIWYG）

## 市場表現

Lisa 的市場表現並不成功：

### 價格過高
9,995 美元的價格讓普通消費者望而卻步。

### 與 Macintosh 的競爭
1984 年推出的 Macintosh 售價更低，更受歡迎。

### 蘋果的策略失誤
蘋果後來決定將 Lisa 的部分技術應用於 Macintosh。

## 歷史意義

Lisa 雖然商業失敗，但具有重要的歷史意義：

### GUI 先驅
Lisa 是第一個商業化的 GUI 系統，為日後的 Windows、Mac OS 和所有 GUI 系統奠定了基礎。

### 施樂 PARC 的傳承
Lisa 繼承並發揚了施樂 PARC 的 GUI 創新。

### 培養人才
Lisa 團隊培養了眾多傑出的軟體工程師，他們日後參與了 Macintosh 和其他重要專案。

## 歷史意義

Lisa 是蘋果公司乃至整個電腦產業的重要里程碑。它的技術創新為日後所有圖形介面作業系統樹立了標準，改變了人機交互的方式。
