# 盤尼西林針劑：士兵的救命稻草

## 一、盤尼西林的意義

盤尼西林（Penicillin，青黴素）是人類歷史上第一種抗生素。它的發現開創了抗生素時代，挽救了無數生命。在二戰期間，盤尼西林針劑成為士兵們的"救命稻草"。

## 二、1943年的供應情況

1943年，盤尼西林的供應仍然有限，但相比1942年有了顯著改善：

| 項目 | 1942年 | 1943年 |
|------|--------|--------|
| 月產量 | 數千萬單位 | 數十億單位 |
| 價格 | 極昂貴 | 大幅下降 |
| 供應範圍 | 優先傷員 | 擴大到前線 |

## 三、盤尼西林針劑的特點

盤尼西林針劑具有以下特點：

| 特點 | 描述 |
|------|------|
| 給藥方式 | 肌肉注射 |
| 起效速度 | 快速 |
| 抗菌範圍 | 廣泛 |
| 安全性 | 較高 |
| 副作用 | 少數過敏 |

## 四、戰場應用

1943年，盤尼西林針劑在以下情況下廣泛使用：

### 主要適應症
- 傷口感染
- 肺炎
- 敗血症
- 腦膜炎
- 淋病

### 使用情況
- 每個野戰醫院配備盤尼西林
- 傷員優先使用
- 珍貴的醫療物資

## 五、士兵的反應

士兵們將盤尼西林視為"神奇藥物"。據老兵回憶，盤尼西林針劑是他們在戰場上收到的最珍貴的禮物。

## 六、歷史影響

盤尼西林針劑的成功應用：
- 開創了抗生素治療的先河
- 降低了傷口感染死亡率
- 為日後民用醫療的普及奠定了基礎
