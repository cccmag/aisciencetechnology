# 化學獎：赫斯與同位素示蹤技術

## 獲獎者簡介

1943年諾貝爾化學獎授予了匈牙利裔化學家**喬治·赫斯**（Georg de Hevesy，1885-1966），以表彰他使用同位素作為示蹤劑進行生物化學研究。

赫斯出生於匈牙利，1926年移民美國，在佛羅里達大學和斯德哥爾摩從事研究工作。

## 同位素示蹤技術

### 原理

同位素示蹤技術的基本原理是：
1. 將穩定或放射性同位素摻入化合物中
2. 追蹤該化合物在生物體或化學反應中的路徑
3. 通過檢測同位素的放射性或質量來確定其位置

### 優勢

| 特性 | 優點 |
|------|------|
| 靈敏度 | 極高，可檢測極少量 |
| 專一性 | 不會干擾正常生理過程 |
| 可追蹤 | 實時觀察化學反應過程 |

## 應用領域

赫斯的方法在以下領域得到了廣泛應用：

| 領域 | 應用 |
|------|------|
| 生物化學 | 研究代謝途徑 |
| 醫學 | 診斷和治療甲狀腺疾病 |
| 農業 | 研究植物營養吸收 |
| 化學 | 追蹤化學反應機制 |

## 歷史影響

同位素示蹤技術的發明為生物學和醫學研究提供了強大的工具。這項技術至今仍是實驗室和臨床診斷的重要手段。
