# 前言：新冠疫情與百年未有之大變局

## 2020年——人類歷史的分水嶺

2020年，一種名為新型冠狀病毒（SARS-CoV-2）引起的疾病——新冠肺炎（COVID-19）——從中國武漢迅速蔓延至全球各地，至年底感染人數超過8000萬，死亡人數逾170萬。這是人類進入21世紀以來最嚴重的全球公共衛生危機，深刻改變了人類的社會運作方式、工作模式、教育形態和國際秩序。

## 年度重大事件回顧

### 新冠疫情全球大流行

2019年12月，武漢市衛健委首次通報不明原因肺炎病例。2020年1月，專家確認病原為新型冠狀病毒。1月30日，世界衛生組織宣佈將疫情列為「國際關注的突發公共衛生事件」。3月11日，WHO正式宣佈新冠疫情構成全球大流行。

| 日期 | 事件 |
|------|------|
| 2019年12月 | 武漢首批病例被發現 |
| 2020年1月 | 病毒基因序列公佈 |
| 2020年3月 | WHO宣佈為全球大流行 |
| 2020年12月 | mRNA疫苗開始大規模接种 |

### 居家隔離與遠程辦公

各國相繼採取封鎖措施，人類社會首次大規模轉向遠程辦公與線上教育。Zoom等視訊會議軟體用戶數量爆炸性增長，從2019年底的1000萬增至2020年底的3億多人。

### mRNA疫苗的歷史性突破

2020年11月，輝瑞-拜恩泰科（Pfizer-BioNTech）和莫德納（Moderna）相繼宣佈其mRNA疫苗三期臨床試驗有效率達95%左右，人類首次使用mRNA疫苗技術。這項技術從立項到疫苗問世僅用了不到一年時間，創造了疫苗髮展史上的奇蹟。

### 諾貝爾獎與疫情的交匯

2020年的諾貝爾獎同樣備受矚目。生理醫學獎授予了發現丙型肝炎病毒的三位科學家，他們的工作為全球數億丙肝患者帶來了治愈希望，也為日後抗擊新冠疫情提供了重要基礎。

## 從危機到轉機

2020年的種種事件正在深刻重塑人類社會：

- mRNA疫苗技術的成熟將開啟疫苗學的新時代
- 遠程辦公與線上教育將成為常態
- 公共衛生體系得到前所未有的重視
- 數位化轉型全面加速

讓我們回顧這一年的重要科學發現、技術發明和產品創新。
