# 布拉格父子與 X 射線晶體分析

## 一、X 射線的發現與早期研究

1895 年，德國物理學家威廉·康拉德·倫琴（Wilhelm Röntgen）發現了 X 射線，這一發現開創了醫學影像診斷的新紀元。倫琴因此獲得第一個諾貝爾物理學獎（1901 年）。

X 射線發現後，科學家們開始思考：這種高能電磁輻射能否用於研究物質的微觀結構？

## 二、布拉格父子的研究

威廉·亨利·布拉格（William Henry Bragg，1862-1942）是英國物理學家，長期在劍橋大學任教。他的兒子威廉·勞倫斯·布拉格（William Lawrence Bragg，1890-1971）曾在劍橋大學攻讀物理學。

1912 年，威廉·勞倫斯·布拉格在研究中發現了一個重要的規律：X 射線在晶體中的衍射圖案可以用晶體的原子排列來解釋。他提出了著名的「布拉格定律」（Bragg's Law）：

**nλ = 2d sin θ**

其中：
- n 是衍射級數（整數）
- λ 是 X 射線波長
- d 是晶面間距
- θ 是入射角

這個公式描述了 X 射線被晶體衍射的條件，是 X 射線晶體學的基礎。

## 三、X 射線晶體分析的原理

布拉格父子利用 X 射線研究晶體結構，開創了一個全新的研究領域。其基本原理是：

**X 射線與晶體的相互作用：**
- X 射線的波長與原子間距相當
- 當 X 射線穿過晶體時，會發生衍射
- 衍射圖案反映了晶體的原子排列

**分析方法：**
1. 用單色 X 射線照射晶體樣品
2. 測量不同角度下的衍射強度
3. 根據布拉格定律計算晶面間距
4. 重構晶體的原子結構

## 四、歷史意義與影響

布拉格父子的工作對科學發展產生了深遠的影響：

| 應用領域 | 具體貢獻 |
|---------|---------|
| 礦物學 | 確定了多種礦物的晶體結構 |
| 化學 | 為結構化學的建立奠定基礎 |
| 生物學 | 為了解生物大分子的結構提供了工具 |
| 醫學 | 促進了 X 射線診斷技術的發展 |

1915 年，布拉格父子共同獲得諾貝爾物理學獎。值得一提的是，威廉·勞倫斯·布拉格獲獎時年僅 25 歲，是迄今為止最年輕的諾貝爾物理學獎得主。

## 五、對後世科學的影響

布拉格父子開創的 X 射線晶體學方法，為後續的科學發現提供了強大的工具：

| 年份 | 重要發現 | 意義 |
|------|---------|------|
| 1913 年 | 氯化鈉晶體結構 | 首次證實離子晶體的排列 |
| 1953 年 | DNA 雙螺旋結構 | 分子生物學的里程碑 |
| 1958 年 | 血紅蛋白結構 | 結構生物學的開始 |
| 2012 年 | 大型蛋白質結構資料庫 | 結構生物學的大數據時代 |

X 射線晶體學的方法至今仍是確定物質原子結構的主要手段之一。在藥物設計、材料科學、催化學等領域，X 射線晶體學都發揮著不可替代的作用。
