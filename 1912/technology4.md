# 早期雷達探測概念

## 一、無綫電技術的興起

1895 年，義大利發明家古列爾莫·馬可尼（Guglielmo Marconi）成功進行了第一次無綫電通信實驗。此後，無綫電技術經歷了快速發展，應用範圍不斷擴大。

第一次世界大戰前，各國已經開始軍事應用無綫電技術，包括通信和探測。

## 二、1912 年的早期探測技術

1912 年，出現了利用無綫電技術進行探測的概念和初步實驗：

### 無綫電測距原理

科學家們開始探索利用無綫電波探測遠處物體的可能性：

**基本原理：**
- 發射無綫電脈衝
- 測量脈衝到達目標並返回的時間
- 根據時間計算目標的距離

### 專利申請

1912 年，德國工程師胡爾斯特·克裏斯蒂安·許林（Christian Hülsmeyer）申請了一項利用無綫電探測物體的專利，名為「 Telemobiloscope」：

- 這是被認為是最早的雷達專利之一
- 專利描述了利用反射波探測遠處物體的裝置
- 這個裝置可以探測到約 2 公里外的船隻

## 三、早期探測系統的特點

1912 年的早期無綫電探測系統具有以下特點：

| 方面 | 描述 |
|------|------|
| 工作波長 | 長波（當時技術限制） |
| 探測距離 | 非常有限（約 2-5 公里） |
| 解析度 | 非常粗糙 |
| 實用性 | 缺乏實用價值 |

## 四、軍事需求的推動

早期無綫電探測技術的發展主要受到軍事需求的推動：

**海上導航：**
- 需要探測冰山和其他船隻
- 為船舶提供導航輔助
- 避免碰撞事故

**邊境監視：**
- 監測入侵的飛機或船隻
- 提供早期預警
- 補充目視觀察的不足

## 五、歷史意義與後續發展

1912 年的無綫電探測概念雖然距離實用還有很長的路要走，但它標誌著現代雷達技術的起點。

| 年份 | 雷達技術里程碑 |
|------|--------------|
| 1912 年 | 無綫電探測概念提出 |
| 1930 年代 | 實用化無綫電探測裝置 |
| 1935 年 | 英國建立第一個雷達站 |
| 1940 年代 | 微波雷達發展 |
| 1950 年代 | 平面陣列雷達 |
| 1990 年代 | 相控陣雷達普及 |
