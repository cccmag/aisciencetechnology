# 早期飛機導航系統

## 一、飛機導航的興起

1903 年萊特兄弟成功飛行後，飛機技術經歷了快速發展。然而，早期飛機的導航主要依靠飛行員的視覺參考和地標識別，這在惡劣天氣或海洋上空飛行時變得極為困難甚至危險。

隨著航空事業的發展，如何在看不見地面的情況下確定飛機的位置和航向，成為一個迫切需要解決的問題。

## 二、1912 年的導航技術

1912 年出現了幾種早期飛機導航輔助設備：

### 陀螺儀

1912 年，美國工程師勞倫斯·斯佩里（Lawrence Sperry）發明了陀螺穩定器，並將其應用於飛機。陀螺儀可以幫助飛行員保持飞机的姿態和航向：

**原理：**
- 高速旋轉的陀螺儀保持其在空間中的方向
- 飛機姿態變化時，陀螺儀可以指示出來
- 這對於在看不見地面的情況下飛行非常重要

### 人工地平線

1912 年，出現了第一個人工地平線指示器（Artificial Horizon）：
- 使用陀螺儀保持水平參考
- 顯示飛機的俯仰和滾轉姿態
- 幫助飛行員判斷飛機是否水平飛行

### 定向陀螺儀

定向陀螺儀（Directional Gyro）是另一項重要發明：
- 保持一個固定的航向參考
- 顯示飛機相對於北方的航向
- 雖然不是真正的磁羅盤，但可以在短時間內提供航向參考

## 三、導航設備的類型

1912 年的早期飛機導航設備主要包括：

| 設備 | 功能 | 局限性 |
|------|------|--------|
| 陀螺儀 | 保持姿態穩定 | 需要定期校正 |
| 人工地平線 | 顯示飛機姿態 | 不能指示方向 |
| 定向陀螺儀 | 指示航向 | 會飄移，需要校正 |
| 磁力羅盤 | 指示北方 | 受飛機磁場干擾 |

## 四、對航空安全的影響

早期導航設備的發明對航空安全產生了重要影響：

**減少事故：**
- 幫助飛行員在惡劣天氣下保持控制
- 降低了可控飛行撞地（CFIT）的風險
- 提高了夜間飛行的安全性

**擴大應用範圍：**
- 使得長距離飛行成為可能
- 促进了商業航空的發展
- 為後續的儀表著陸系統奠定了基礎

## 五、技術演進

1912 年的導航技術雖然還很初步，但為後續的發展奠定了基礎：

| 年份 | 導航技術 |
|------|---------|
| 1912 年 | 陀螺儀應用於飛機 |
| 1920 年代 | 陀螺羅盤成熟 |
| 1930 年代 | 儀表著陸系統（ILS） |
| 1940 年代 | 無綫電導航 |
| 1950 年代 | 雷達導航 |
| 1970 年代 | 慣性導航系統 |
| 1990 年代 | GPS 導航 |
