# C++ 程式語言：物件導向程式設計的普及

西元 1983 年，[比雅尼·斯特勞斯特魯普](https://zh.wikipedia.org/wiki/比雅尼·斯特勞斯特魯普)（Bjarne Stroustrup）在貝爾實驗室發明了 [C++](https://zh.wikipedia.org/wiki/C%2B%2B) 程式語言。1984 年，C++ 開始被廣泛使用，成為物件導向程式設計的主流語言。

## 語言的起源

[C++](https://zh.wikipedia.org/wiki/C%2B%2B) 的前身是「C with Classes」，是斯特勞斯特魯普於 1979 年開始開發的：

### 設計目標
結合 C 語言的效率和低層控制能力，加上物件導向的特性。

### 名字來源
「C++」中的「++」代表 C 的遞增運算，暗示這是 C 的擴展版本。

### 標準化
經過多年發展，C++ 於 1998 年正式標準化（ISO/IEC 14882）。

## 核心特性

C++ 語言提供了豐富的程式設計能力：

### 物件導向
- 類別（Class）
- 繼承（Inheritance）
- 多型（Polymorphism）
- 封裝（Encapsulation）

### 高效能
- 直接記憶體操作
- 指標運算
- 編譯器優化

### 泛型程式設計
- 模板（Template）
- 標準模板庫（STL）

### 異常處理
- try-catch 結構
- 異常規格

## 與 C 的比較

C++ 與 C 語言的主要區別：

| 特性 | C | C++ |
|------|---|-----|
| 物件導向 | 不支援 | 完全支援 |
| 類別 | 無 | 有 |
| 模板 | 無 | 有 |
| 例外處理 | 無 | 有 |
| 命名空間 | 無 | 有 |
| 向後相容 | - | 完全相容 |

## 應用領域

C++ 在 1984 年後迅速成為主流程式語言，應用於：

### 系統程式設計
作業系統、驅動程式、編譯器等。

### 遊戲開發
大多數 AAA 級遊戲引擎（如 Unreal、Unity）使用 C++。

### 金融交易
高頻交易系統需要高效能。

### 嵌入式系統
資源受限的嵌入式設備離不開 C++。

### 大型軟體
Adobe、Microsoft Office 等大型軟體使用 C++ 開發。

## 歷史意義

C++ 是物件導向程式設計時代最具影響力的語言之一。它不僅延續了 C 語言的高效能，同時引入了物件導向的程式設計範式，深刻影響了後續的程式語言和軟體開發實踐。
