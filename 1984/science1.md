# 蘋果 Macintosh 發布：圖形介面革命的巔峰

西元 1984 年 1 月 24 日，[蘋果公司](https://zh.wikipedia.org/wiki/蘋果公司)正式發布 [Macintosh](https://zh.wikipedia.org/wiki/Macintosh) 電腦。這款配備圖形使用者介面和滑鼠的個人電腦，被譽為「1970 年以來最重要的事件」，開創了個人計算的新時代。

## 產品的誕生

[Macintosh](https://zh.wikipedia.org/wiki/Macintosh) 的開發是史蒂夫·喬布斯（Steve Jobs）最偉大的成就之一：

### Lisa 的教訓
Macintosh 的開發借鑑了 Lisa 的經驗，採用了更先進的技術和更低的價格。

### 開發團隊
由比爾·艾奇森（Bill Atkinson）、安迪·赫茲菲爾德（Andy Hertzfeld）等天才工程師組成的團隊。

### 設計理念
追求極致的用戶體驗，讓電腦「為大眾設計」。

## 技術規格

第一代 Macintosh 的配置：

| 組件 | 規格 |
|------|------|
| 處理器 | Motorola 68000，8 MHz |
| 記憶體 | 128 KB（最大 512 KB） |
| 顯示器 | 9 吋內建 CRT，512×342 |
| 儲存 | 400 KB 軟碟 |
| 作業系統 | Mac OS（System 1） |
| 輸入設備 | 滑鼠 |
| 價格 | 2,495 美元 |

## 技術創新

Macintosh 帶來了多項革命性創新：

### 圖形使用者介面
- 視窗（Windows）
- 圖標（Icons）
- 桌面（Desktop）
- 下拉式功能表

### 滑鼠輸入
Macintosh 是第一款大量市場化的使用滑鼠的個人電腦。

### 所見即所得
文字和圖形在螢幕上的顯示與列印結果一致。

### 桌面隱喻
採用「桌面」的比喻，讓電腦操作直觀易懂。

## 市場影響

Macintosh 的發布在市場上引起了巨大反響：

### 首日銷售
Macintosh 上市當天就售出了數萬台。

### 市場定位
迅速成為圖形設計和出版行業的首選工具。

### 產業影響
催生了桌面出版革命，徹底改變了出版業。

### 跟隨者
微軟的 Windows 和其他 GUI 系統都受到了 Macintosh 的啟發。

## 歷史意義

Macintosh 不僅是一款產品，更是一種變革的力量。它將複雜的電腦操作變得直觀易懂，讓「每個人都能使用電腦」的願景成為可能。
