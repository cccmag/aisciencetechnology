# 光纖區域網路標準：FDDI 標準確立

西元 1984 年，[光纖分散式資料介面](https://zh.wikipedia.org/wiki/光纖分散式資料介面)（FDDI，Fiber Distributed Data Interface）標準的制定接近完成。這是第一個針對光纖設計的 100 Mbps 區域網路標準。

## FDDI 的背景

FDDI 的開發是為了滿足高速區域網路的需求：

### 速度需求
隨著電腦網路的發展，10 Mbps 的乙太網路已經不能滿足需求。

### 光纖技術
光纖技術的進步使得高速傳輸成為可能。

### 標準化需求
業界需要一個統一的高速網路標準。

## 技術規格

FDDI 標準的技術規格：

| 項目 | 規格 |
|------|------|
| 傳輸速率 | 100 Mbps |
| 介質類型 | 光纖（雙環） |
| 最大節點數 | 500 個 |
| 最大環路長度 | 200 公里 |
| 標準年份 | 1988（正式） |
| 開發階段 | 1980 年代中期 |

## 技術特點

FDDI 具有以下技術特點：

### 雙環結構
使用雙環拓撲，提供冗餘和容錯能力。

### 光纖傳輸
使用光纖作為傳輸介質，抗干擾能力強。

### 確定性介質訪問
使用令牌傳遞機制，確保確定性延遲。

### 長距離傳輸
最大傳輸距離可達 200 公里。

## 應用場景

FDDI 在 1980 年代中期主要應用於：

### 骨幹網路
連接大樓或校園內多個乙太網路區段。

### 數據中心
連接伺服器和儲存設備。

### 高速互聯
需要高速傳輸的科研和工程應用。

### 金融機構
銀行和證券交易所需要高速可靠的網路。

## 對後世的影響

FDDI 的標準化對網路技術的發展产生了深远影響：

### 光纖網路
推動了光纖區域網路的普及和發展。

### 高速標準
為後續的 Fast Ethernet、Gigabit Ethernet 奠定了基礎。

### 網路設計
FDDI 的雙環冗餘設計成為網路設計的典範。

### 標準化經驗
FDDI 的標準化過程為後續網路標準的制定提供了經驗。

## 歷史意義

FDDI 是第一個實用化的高速光纖區域網路標準。它的出現滿足了 1980 年代對高速網路的需求，並為日後的網路升級奠定了基礎。
