# Game Boy Color 掌上遊戲機

1998 年，任天堂發布了 Game Boy Color，這是 Game Boy 系列的重要升級版本。這款彩色掌上遊戲機的發布，進一步延長了 Game Boy 家族的市場生命。

## Game Boy 的市場統治

1998 年，Game Boy 已經統治掌上遊戲機市場近十年。1989 年發布的原版 Game Boy，以其實惠的價格和出色的遊戲陣容，創造了掌上遊戲的奇蹟。

到 1998 年，Game Boy 的全球銷量已經超過 6000 萬台，是有史以來最暢銷的掌上遊戲機。這個記錄在此後很長時間內都沒有被打破。

Game Boy 的成功有多個因素：實惠的價格（最初僅為 49.95 美元）、長時間的電池續航、豐富的遊戲選擇、以及創新的卡帶設計讓用戶可以玩到多種類型的遊戲。

## Game Boy Color 的新功能

Game Boy Color 於 1998 年 10 月在日本發布，1998 年 11 月在北美上市。這款新主機最大的亮點是配備了彩色螢幕。

彩色 TFT LCD 螢幕可以顯示 32,768 種顏色（15 位元色），雖然與當時的家用遊戲機相比並不算高，但對於掌上遊戲機來說是一個重大飛躍。

除了彩色螢幕外，Game Boy Color 還配備了更快的處理器（比原版快約 50%）、更大的記憶體、以及與原版 Game Boy 的向後相容性。

## 與競爭對手的比較

1998 年的掌上遊戲機市場，除了 Game Boy 還有 AT&T 的 Game Player 和其他一些產品。但這些產品都沒有對 Game Boy 的市場地位造成實質威脅。

同年，世嘉推出了 WonderSwan，這是另一款掌上遊戲機，由於 Bandai 在日本銷售。但 WonderSwan 沒有在北美市場取得成功。

最終，Game Boy Color 延續了 Game Boy 系列的市場統治地位。這款產品在 1998 年聖誕購物季期間成為最受歡迎的禮物之一。

## 遊戲陣容

Game Boy Color 的成功離不開豐富的遊戲陣容。任天堂和第三方開發商為這款主機開發了大量遊戲。

一些遊戲利用了彩色螢幕的優勢，推出了彩色版本或專門設計的彩色遊戲。經典作品如《精靈寶可夢》的彩色版本在玩家中非常受歡迎。

與此同時，原版 Game Boy 的遊戲在 Game Boy Color 上仍然可以正常運行。這種向後相容性保護了玩家的遊戲投資。

## 技術規格

Game Boy Color 的技術規格如下：
- 處理器：Sharp LR35902，時脈 8 MHz
- 記憶體：128 KB 系統記憶體，256 KB 視訊記憶體
- 螢幕：2.9 英寸 TFT 彩色 LCD，解析度 160x144 像素
- 顏色：32,768 色（最多同時顯示 56 色）
- 電源：2 顆 AA 電池，約可運行 10 小時

## 歷史地位

Game Boy Color 在 Game Boy 系列的演進中扮演了重要角色。它為玩家提供了更好的視覺體驗，同時保持了合理的价格。

這款產品也為 2001 年的 Game Boy Advance 奠定了基礎。Game Boy Advance 在很多方面都是 Game Boy Color 的直接後繼者。

回顧歷史，Game Boy 系列的持久成功証明了，有時候產品的簡單性和實用性比特斯拉這樣的華而不實更為重要。
