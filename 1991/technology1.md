# 萬維網（World Wide Web）的普及：資訊革命的引爆點

1991 年，[提姆·伯納斯-李](https://zh.wikipedia.org/wiki/提姆·伯納斯-李)在[歐洲核子研究組織](https://zh.wikipedia.org/wiki/歐洲核子研究組織)（CERN）正式對外開放了[萬維網](https://zh.wikipedia.org/wiki/萬維網)（World Wide Web）的技術。這個決定徹底改變了人類獲取和分享資訊的方式，被認為是[資訊革命](https://zh.wikipedia.org/wiki/資訊革命)的引爆點。

## 萬維網的誕生背景

[提姆·伯納斯-李](https://zh.wikipedia.org/wiki/提姆·伯納斯-李)是英國物理學家，1989 年在 CERN 工作時，基於已有的網路技術（[TCP/IP](https://zh.wikipedia.org/wiki/TCP/IP协议) 和 [DNS](https://zh.wikipedia.org/wiki/域名系統)），發明了萬維網。

伯納斯-李的設計理念是：建立一個全球資訊網路，人們可以通過「超連結」的方式將各種資訊連接在一起。他設計了三項核心技術：[超文本標記語言](https://zh.wikipedia.org/wiki/HTML)（HTML）用於編寫網頁；[統一資源定位符](https://zh.wikipedia.org/wiki/URL)（URL）用於標識網路上的資源；[超文本傳輸協定](https://zh.wikipedia.org/wiki/HTTP)（HTTP）用於在網路設備之間傳輸資料。

1990 年，伯納斯-李開發了第一個網頁瀏覽器和伺服器軟體。這套系統在 CERN 內部使用了一年多，1991 年正式對外開放。

## 開放的决定

1991 年，伯納斯-李做出了一個改變歷史的決定：將萬維網的技術免费向全世界開放。他放棄了專利和版稅，讓任何人都可以免費使用萬維網的技術來建立網站。

這個決定極具遠見。如果伯納斯-李像當時的許多發明家一樣，為萬維網申請專利並收取版稅，網路的普及可能會延緩數年甚至數十年。正是這個開放的決定，使得萬維網得以迅速在全球普及。

[維基百科](https://zh.wikipedia.org/wiki/維基百科)引用伯納斯-李的話說：「如果當初我為萬維網申請專利並收取費用，我現在可能比爾·蓋茲更富有。但這不是我想做的事。我希望萬維網能夠惠及所有人。」

## 萬維網的工作原理

[萬維網](https://zh.wikipedia.org/wiki/萬維網)建立在一系列協定和技術之上：

當使用者在瀏覽器中輸入一個網址（如 https://www.example.com）時，瀏覽器會向目標伺服器發送一個 HTTP 請求。伺服器收到請求後，會返回對應的 HTML 文檔。瀏覽器讀取 HTML 文檔，解釋其中的標籤和內容，最終將網頁渲染在螢幕上。

網頁中的文字、圖片、視頻等多種內容通過 URL 來標識。這些 URL 可以指向同一個伺服器上的其他資源，也可以指向世界上任何角落的其他伺服器。通過這種方式，全世界的網路資源被連接成一個巨大的資訊網。

## 1991 年的推廣

1991 年 8 月，伯納斯-李向公眾發布了萬維網的詳細說明。這份說明文檔被稱為「URL 文件」，詳細解釋了如何使用萬維網的各項技術。

同樣在 1991 年，世界上第一個網站 info.cern.ch 上線。這個網站介紹了萬維網的技術和使用方法。雖然當時全球只有大約 200 個網站，但這標誌著網路大眾化的開始。

伯納斯-李還建立了一個名為「alt.hypertext」的網路論壇，供對萬維網技術感興趣的人們討論和交流。

## 對社會的深遠影響

[萬維網](https://zh.wikipedia.org/wiki/萬維網)的普及對人類社會產生了變革性的影響。

在資訊傳播方面，萬維網使人們可以即時獲取全球各地的資訊。傳統的圖書館、新聞機構的功能被重新定義，任何人都可以在網上發布資訊。

在商業模式方面，電子商務、線上服務、數位行銷等新業態層出不窮。Amazon、eBay、Alibaba 等電子商務巨頭的崛起，都是建立在萬維網技術之上的。

在社會互動方面，社群媒體、部落格、論壇等平台改变了人與人之間的交流方式。遠程辦公、線上教育、數位醫療等服務也因為萬維網而成為可能。

在政治參與方面，社交媒體成為了政治動員和公眾參與的新工具。從某種程度上說，萬維網改变了現代民主運作的方式。

## 從 Web 1.0 到未來

1991 年的萬維網還處於非常原始的階段。當時的網站主要是靜態的 HTML 頁面，互動功能極為有限。論壇、電子商務、社交媒體等我們今天熟悉的應用，那時還沒有出現。

但正是這些原始的技術和服務，為日後的發展奠定了基礎。從 1990 年代的 [Web 1.0](https://zh.wikipedia.org/wiki/Web+1.0) 到 2000 年代的 [Web 2.0](https://zh.wikipedia.org/wiki/Web+2.0)，再到未來的 [Web 3.0](https://zh.wikipedia.org/wiki/Web+3.0)，萬維網經歷了持續的演進和創新。

如今，我們正在見證[區塊鏈](https://zh.wikipedia.org/wiki/區塊鏈)、[人工智慧](https://zh.wikipedia.org/wiki/人工智慧)和萬維網的融合，這些新技術正在開創網路的新時代。
