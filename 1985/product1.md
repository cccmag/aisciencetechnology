# Microsoft Windows 1.0：作業系統的革新

1985 年 11 月 20 日，[微軟](https://zh.wikipedia.org/wiki/微軟)發布 [Windows 1.0](https://zh.wikipedia.org/wiki/Windows_1.0)，這款售價 99 美元的作業系統環境開創了微軟圖形介面的新時代。

## 產品特點

Windows 1.0 的核心特點：

### 圖形介面
視窗、功能表、圖標的 GUI 設計。

### 預設應用
包括 Write、Paint、Calculator 等工具。

### 記憶體限制
需要至少 256 KB 記憶體。

## 歷史意義

Windows 1.0 雖然並不完美，但為日後 Windows 95 的巨大成功奠定了基礎。
