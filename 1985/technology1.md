# Microsoft Windows：作業系統的圖形化

西元 1985 年 11 月 20 日，[微軟](https://zh.wikipedia.org/wiki/微軟)發布 [Windows 1.0](https://zh.wikipedia.org/wiki/Windows_1.0)，這是第一款成功的[圖形化作業系統](https://zh.wikipedia.org/wiki/圖形使用者介面)環境。

## 技術架構

Windows 1.0 的技術特點：

### 基於 MS-DOS
Windows 1.0 並非獨立作業系統，而是運行在 MS-DOS 之上的圖形環境。

### 協作式多工
不同於現代的優先級多工，Windows 1.0 使用協作式多工。

### 記憶體限制
受限於 DOS 的 640 KB 基本記憶體限制。

### 圖形設備介面
使用 GDI（圖形設備介面）進行繪圖。

## 核心功能

Windows 1.0 提供了多項核心功能：

### 視窗管理
支援視窗的移動、調整大小和最小化。

### 功能表系統
統一的 Windows 功能表設計。

### 對話框
標準化的對話框控制項。

### 圖標支援
桌面和程式管理器的圖標。

## 歷史意義

Windows 1.0 的發布標誌著微軟圖形介面時代的開始，為日後 Windows 95 的巨大成功奠定了基礎。
