# Windows 3.11：為 Windows 95 熱身

1993 年，微軟發布了 [Windows 3.11](https://zh.wikipedia.org/wiki/Windows_3.11)。這是 Windows 3.1 的最後一個版本，增加了網路功能和錯誤修復。

## Windows 3.11 的意義

[Windows 3.11](https://zh.wikipedia.org/wiki/Windows_3.11) 為即將發布的 Windows 95 奠定了基礎，預示著新一代 Windows 作業系統的來臨。
