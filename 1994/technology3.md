# JavaScript 發布：互動式網頁的誕生

1994 年，網景公司在 Navigator 瀏覽器中引入了 [JavaScript](https://zh.wikipedia.org/wiki/JavaScript)。這是一種可以在網頁中運行的腳本語言，使網頁獲得了互動能力。

## JavaScript 的意義

[JavaScript](https://zh.wikipedia.org/wiki/JavaScript) 使靜態網頁轉變為動態應用，這是 Web 開發史上的重大突破。

如今，JavaScript 是全球最流行的程式語言之一。
