# 歷史穿梭：1994 年的科學、技術與未來

**科學技術年鑑　1994 年號**

編輯：[陳鍾誠](https://csie.nqu.edu.tw/p/404-1038-2377.php?Lang=zh-tw)

---

1994 年是[網景 Navigator](#) 時代來臨和[北美自由貿易協定](#) 生效的轉捩點。這些事件開啟了互聯網普及和全球化的新階段。

## 1994 與 2026：我們有何不同？

| 1994 年的人 | 2026 年的我們 |
|------------|--------------|
| 網景 Navigator 佔據 80% 市場 | 瀏覽器市場多元化 |
| NAFTA 剛生效 | USMCA 取代 NAFTA |
| JavaScript 剛發布 | Web 應用無處不在 |
| 互聯網用戶約 3000 萬 | 互聯網用戶超過 50 億 |
| 類比手機黃金時代 | 5G/6G 移動網路 |

## 對未來的影響

| 1994 年 | 影響領域 | 對未來的影響 |
|--------|----------|--------------|
| 網景 Navigator | [互聯網](https://zh.wikipedia.org/wiki/互聯網) | [電子商務](https://zh.wikipedia.org/wiki/電子商務)、[社交媒體](https://zh.wikipedia.org/wiki/社交媒體) |
| NAFTA | [經濟](https://zh.wikipedia.org/wiki/經濟學) | [全球供應鏈](https://zh.wikipedia.org/wiki/供應鏈) |
| JavaScript | [網頁開發](https://zh.wikipedia.org/wiki/網頁設計) | [前端框架](https://zh.wikipedia.org/wiki/前端框架)、[Node.js](https://zh.wikipedia.org/wiki/Node.js) |
| PlayStation | [遊戲](https://zh.wikipedia.org/wiki/電子遊戲) | [雲端遊戲](https://zh.wikipedia.org/wiki/雲端遊戲) |
| Pentium Pro | [半導體](https://zh.wikipedia.org/wiki/半導體) | [摩爾定律](https://zh.wikipedia.org/wiki/摩爾定律) |

## 結論

1994 年開啟了互聯網普及和全球化的新時代。這些趨勢在後續三十年間深刻改變了我們的社會和生活。
