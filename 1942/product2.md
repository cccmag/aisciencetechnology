# 玻璃紙包裝的商業化

## 一、玻璃紙的發明

玻璃紙（Cellophane）是一種由纖維素製成的透明薄膜。1908年，瑞士化學家雅克·布蘭德伯格（Jacques Brandenberger）發明了玻璃紙。這種材料最初是為了保護酒瓶標籤而設計的。

1942年，玻璃紙包裝在美國和歐洲得到廣泛商業化應用。

## 二、玻璃紙的特點

玻璃紙具有諸多優異性能：

| 特性 | 描述 |
|------|------|
| 透明度 | 如玻璃般透明 |
| 防潮性 | 對水蒸氣有一定的阻隔性 |
| 強度 | 堅韌不易撕裂 |
| 環保 | 可生物降解 |

## 三、1942年的商業應用

1942年，玻璃紙包裝在以下領域得到廣泛應用：

### 食品包裝
- 糖果包裝
- 餅乾包裝
- 肉類包裝

### 菸草包裝
- 香菸盒內層包裝
- 菸草保鮮包裝

### 醫藥包裝
- 藥品防潮包裝
- 醫療器械包裝

## 四、歷史影響

玻璃紙的商業化對包裝行業產生了深遠影響：
1. 推動了透明包裝的普及
2. 促進了食品保鮮技術的發展
3. 為日後塑料包裝的興起奠定了基礎
