# 便攜式收音機的流行

## 一、收音機的普及

廣播作為一種新興媒體，在1920年代開始普及。1942年，收音機已成為美國家庭必備的電器。全美約有5000萬台收音機，平均每三個家庭就有一台。

## 二、1942年的收音機技術

1942年的收音機技術相比十年前有了顯著進步：

| 技術環節 | 1942年的水平 |
|----------|--------------|
| 電子管 | 可靠性提高 |
| 功耗 | 降低約30% |
| 體積 | 更緊湊 |
| 價格 | 持續下降 |

## 三、戰時收音機的作用

1942年，收音機成為戰時最重要的信息傳播工具：

### 新聞傳播
- 及時報導戰爭進展
- 政府發布重要信息
- 鼓舞民心士氣

### 娛樂
- 廣播劇
- 音樂節目
- 訪談節目

### 教育和文化
- 成人教育節目
- 外語學習節目
- 新聞評論

## 四、便攜式收音機

1942年，便攜式收音機開始流行。這些收音機使用電池供電，可以隨身攜帶，成為士兵和民眾的重要伴侶。

## 五、歷史影響

收音機的普及對社會產生了深遠影響：
1. 開創了大眾傳媒時代
2. 改变了政治动员方式
3. 丰富了战时文化生活
4. 為日後電視的興起奠定了基礎
