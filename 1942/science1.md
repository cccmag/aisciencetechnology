# 芝加哥一號堆：人類首次觸及原子能

## 一、歷史背景

1942年是人類歷史上具有深遠意義的一年。這一年，美國的曼哈頓計畫進入了關鍵階段。8月，物理學家恩里科·費米（Enrico Fermi）被任命為世界上第一個核反應堆項目的負責人。

費米是意大利裔美國物理學家，曾經在羅馬大學和哥倫比亞大學工作。他對核物理學的深刻理解和卓越的工程能力，使他成為建造反應堆的不二人選。

## 二、芝加哥一號堆的建造

### 選址與設計

芝加哥大學的斯塔格田徑場（Stagg Field）的西看台下，被選為反應堆的建造地點。這個位置足夠隱蔽，又靠近大學的實驗室設施。

費米設計的反應堆採用：
- **燃料**：天然鈾（氧化鈾）
- **緩衝劑**：超高純度石墨
- **控制材料**：鎘吸收棒

### 建設過程

1942年11月16日，反應堆的建設正式開始。來自不同領域的科學家和工程師日以繼夜地工作，將一塊塊石墨堆積成一個巨大的球形結構。

反應堆的建造面臨諸多挑戰：
1. 石墨純度要求極高（雜質會吸收中子）
2. 鈾燃料需要精心加工
3. 安全措施必須萬無一失

## 三、歷史性的一刻

1942年12月2日，下午3點25分，歷史性的實驗開始了。

費米站在控制台前，緩慢地拔出鎘控制棒。工作人員緊張地觀察著儀器讀數：中子計數率開始緩慢上升，然後急劇攀升——

**鏈式反應成功了！**

費米立即下令插回控制棒，反應堆在38分鐘後安全關閉。人類第一次成功控制了核能量的釋放。

## 四、科學原理

核反應堆的工作原理是：
1. 鈾-235核吸收中子後發生裂變
2. 裂變釋放能量和新中子
3. 新中子引發更多裂變——鏈式反應
4. 控制棒吸收多餘中子，維持可控狀態

### 關鍵數據

| 參數 | 數值 |
|------|------|
| 設計功率 | 200千瓦（熱） |
| 臨界質量 | 約40噸天然鈾 |
| 石墨用量 | 385噸 |
| 工作人員 | 約50人 |

## 五、歷史影響

芝加哥一號堆的成功具有深遠的歷史意義：

1. **開啟原子能時代**：人類第一次從原子核中獲取可控制的能量
2. **驗證核能理論**：證明了費米等人的核反應理論是正確的
3. **推動曼哈頓計畫**：為原子彈的製造奠定了技術基礎
4. **促進核能發展**：為日後核電站的建設提供了範例

費米後來被稱為"原子能之父"。他的這項成就永遠改變了人類的能源格局。
