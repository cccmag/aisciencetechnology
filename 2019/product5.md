# PlayStation 5與Xbox Series X：主機大戰

## 概述

2019年是次世代遊戲主機大戰的前哨戰。索尼和微軟先後公佈了PlayStation 5和Xbox Series X的詳細規格，雙方在硬體性能上的競爭趨於白熱化。

![PS5](https://upload.wikimedia.org/wikipedia/commons/thumb/4/4f/PS5_console_and_controller.png/800px-PS5_console_and_controller.png)

## 一、硬體規格

### 1.1 PS5規格

PlayStation 5的主要規格：

| 參數 | 數值 |
|------|------|
| CPU | AMD Zen 2（8核3.5GHz） |
| GPU | RDNA 2（10.28 TFLOPS） |
| 記憶體 | 16GB GDDR6 |
| 儲存 | 825GB SSD |
| 光碟 | 4K藍光 |

### 1.2 Xbox Series X規格

Xbox Series X的主要規格：

| 參數 | 數值 |
|------|------|
| CPU | AMD Zen 2（8核3.8GHz） |
| GPU | RDNA 2（12 TFLOPS） |
| 記憶體 | 16GB GDDR6 |
| 儲存 | 1TB SSD |
| 光碟 | 4K藍光 |

### 1.3 性能對比

兩家主機的性能比較：

| 指標 | PS5 | Xbox Series X |
|------|-----|---------------|
| GPU性能 | 10.28 TF | 12 TF |
| 處理器 | 3.5 GHz | 3.8 GHz |
| SSD速度 | 5.5 GB/s | 2.4 GB/s |
