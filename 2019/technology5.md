# 自動駕駛的L4時代

## 概述

2019年，自動駕駛技術持續進步，L4級自動駕駛（即在特定區域內完全自動駕駛）在越來越多的場景中變為現實。自動駕駛貨運、無人計程車、無人配送等場景相繼亮相，自動駕駛商業化進程明顯加速。

![自動駕駛](https://upload.wikimedia.org/wikipedia/commons/thumb/8/8e/Google_Self-Driving_Car_%28stack_exchange%29.jpg/800px-Google_Self-Driving_Car_%28stack_exchange%29.jpg)

## 一、等級標準

### 1.1 L0-L5分級

SAE自動駕駛分級：

| 等級 | 名稱 | 駕駛主體 | 場景限制 |
|------|------|----------|----------|
| L0 | 無自動化 | 人類 | 無 |
| L1 | 駕駛輔助 | 人類為主 | 簡單 |
| L2 | 部分自動化 | 人類為主 | 高速 |
| L3 | 條件自動化 | 系統為主 | 特定 |
| L4 | 高度自動化 | 系統 | 限定區域 |
| L5 | 完全自動化 | 系統 | 無限制 |

### 1.2 L4的定義

L4自動駕駛的特點：
- 在定義的運行區域內（ODD）
- 不需要人類干預
- 可以沒有方向盤和刹車
- ，但仍需要緊急情況下的安全措施

### 1.3 與L5的區別

L4與L5的主要區別：
- L4限於特定區域
- L5可在任何道路條件下運行
- L4技術更易實現

## 二、主要進展

### 2.1 Waymo

Waymo的商業化進展：
- Waymo One付費服務在鳳凰城運行
- 累積超過1000萬英里測試里程
- 推出自動駕駛貨運服務Waymo Via

### 2.2 Cruise

通用Cruise的進展：
- 舊金山測試
- 微軟投資
- 計劃2020年商業化

### 2.3 中國企業

中國自動駕駛企業：
- 百度Apollo：在多個城市測試
- 小馬智行：Robotaxi服務
- 文遠知行：廣州Robotaxi

## 三、商業化場景

### 3.1 Robotaxi

自動駕駛出租車服務：

| 公司 | 地點 | 進展 |
|------|------|------|
| Waymo | 鳳凰城 | 付費服務 |
| Cruise | 舊金山 | 測試中 |
| 百度Apollo | 北京、長沙 | Robotaxi服務 |
| 小馬智行 | 廣州 | 載客測試 |

### 3.2 自動駕駛貨運

自動駕駛卡車：
- Waymo Via：貨運服務
- Tesla Semi：電動+自動駕駛
- Otto：自動駕駛卡車

### 3.3 限定場景

限定場景的商業化：
- **礦區**：自動駕駛礦車
- **港口**：集裝箱運輸
- **園區**：低速接駁

## 四、技術進展

### 4.1 感知系統

感知技術的進步：
- 固態LiDAR成本下降
- 感測器融合更加精確
- 算力持續提升

### 4.2 預測規劃

預測和規劃能力提升：
- 行為預測更準確
- 規劃算法更成熟
- 複雜場景處理

### 4.3 高清地圖

高清地圖的支撐：
- 厘米級精度
- 即時更新
- 語義信息

## 五、面臨的挑戰

### 5.1 技術挑戰

自動駕駛仍面臨的問題：
-惡劣天氣性能
- 複雜城市環境
- 邊緣情況處理

### 5.2 法規挑戰

法規方面的問題：
- 責任劃分不明
- 安全標準缺失
- 測試許可複雜

### 5.3 商業化挑戰

商業化的困難：
- 成本仍然較高
- 盈利能力不明
- 公眾接受度
