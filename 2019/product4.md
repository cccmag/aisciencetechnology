# AirPods Pro：主動降噪普及

## 概述

2019年10月，蘋果發布了AirPods Pro，這是AirPods系列首次配備主動降噪功能。AirPods Pro的推出，標誌著真無線耳機進入「降噪時代」，為用戶提供了更好的聆聽體驗。

![AirPods Pro](https://upload.wikimedia.org/wikipedia/commons/thumb/f/f3/AirPods.svg/800px-AirPods.svg.png)

## 一、產品特點

### 1.1 降噪功能

AirPods Pro的核心功能：
| 功能 | 說明 |
|------|------|
| 主動降噪 |每秒調整200次 |
| 通透模式 | 聽到周圍環境音 |
| 適應性均衡 | 根據貼合度調整聲音 |

### 1.2 設計變化

AirPods Pro的設計：
- 入耳式設計
- 可更換耳塞
- 更短的耳機柄
- 全新充電盒

### 1.3 技術規格

主要規格：

| 參數 | 數值 |
|------|------|
| 晶片 | H1 |
| 藍牙 | 5.0 |
| 防水 | IPX4 |
| ANC | 主動降噪 |
| 空間音頻 | 支持 |

## 三、核心技術

### 3.1 主動降噪原理

降噪原理：
1. 麥克風偵測外界噪音
2. 晶片產生反向聲波
3. 相互抵消，消除噪音
4. 根據環境調整降噪程度

### 3.2 通透模式

通透模式的實現：
- 外部麥克風收音
- 傳入耳內
- 用戶可聽到環境聲音
- 自然對話體驗

### 3.3 空間音頻

空間音頻功能：
- 動態頭部追蹤
- 環繞音效
- 杜比全景聲支持
