## 重要歷史背景

1918 年是人類歷史上極為重要的一年。這一年，第一次世界大戰終於結束，但代價極為慘重——超過 1700 萬人死亡。同時，一場史無前例的流感大流行（西班牙流感）席捲全球，估計造成 5000 萬至 1 億人死亡。在這樣的黑暗時期，諾貝爾獎在停發三年後恢歸頒發，為科學界帶來一絲希望。
