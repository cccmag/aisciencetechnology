## 撰寫規則

### 歷史穿梭的撰寫重點

歷史穿梭的重點是「連結過去與未來的科學、技術、產品」，放在整個歷史上去思考：

1. **從過去到未來的連結**：說明這年的科學發現如何影響未來的科技發展
   - 例：1902 年的洛倫茲變換 → 1905 年相對論 → 今日 GPS

2. **從產品到產品的連結**：追蹤同類產品的演進軌跡
   - 例：1902 年福特 A 型車 → 1908 年 T 型車 → 今日電動汽車

3. **跨領域的連結**：指出不同領域之間的相互影響
   - 例：量子力學 → 半導體 → 電腦 → 網際網路

4. **技術發展的長鏈**：展示技術如何在數十年間逐步演進
   - 例：電話交換機 → 自動交換 → 數位交換 → 網路電話 → 雲端通信
