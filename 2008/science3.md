# 宇宙學精密測量：宇宙結構的深入理解

2008 年，[宇宙微波背景輻射](https://zh.wikipedia.org/wiki/宇宙微波背景輻射)的精密測量持續取得進展，來自 WMAP 和其他實驗的數據為宇宙學模型提供了更精確的約束。

## 宇宙微波背景輻射

### WMAP 數據

[威爾金森微波各向異性探測器](https://zh.wikipedia.org/wiki/威爾金森微波各向異性探測器)（WMAP）：
- 2001 年發射
- 持續觀測宇宙
- 2006 年發布第一年數據
- 2008 年發布三年數據

### 測量結果

WMAP 三年數據的宇宙參數：
| 參數 | 數值 |
|------|------|
| 宇宙年齡 | 137 億年 |
| 暗能量比例 | 72% |
| 暗物質比例 | 23% |
| 普通物質比例 | 4.6% |

## 宇宙組成

### 暗能量

[暗能量](https://zh.wikipedia.org/wiki/暗能量)的證據：
- 宇宙加速膨脹
- 佔宇宙總量 72%
- 本質未知
- 宇宙學最大謎題

### 暗物質

[暗物質](https://zh.wikipedia.org/wiki/暗物質)的證據：
- 星系旋轉曲線
- 引力透鏡
- 星系團質量
- 宇宙結構形成

## 宇宙結構

### 原初密度起伏

宇宙結構的種子：
- 暴漲理論預言
- 量子漲落放大
- WMAP 精密測量
- 驗證理論

### 宇宙幾何

宇宙的形狀：
- 測量結果：平坦
- 與暴漲理論一致
- 臨界密度
- 歐幾里得幾何

## 與 LHC 的關聯

### 宇宙早期

[LHC](https://zh.wikipedia.org/wiki/大型強子對撞機)和宇宙學的聯繫：
- 宇宙早期能量等於 LHC 碰撞能量
- 粒子物理和宇宙學交匯
- 相互驗證
- 共同揭示基本規律

### 暗物質候選

暗物質粒子：
- 超對稱粒子
- 軸子
- 暗光子
- LHC 可能發現

## 歷史意義

### 精密宇宙學

宇宙學測量的歷史意義：
- **宇宙圖景**：全新的宇宙圖像
- **精確測量**：從猜測到科學
- **新物理**：暗能量和暗物質
- **跨學科**：粒子物理和宇宙學融合

### 未解之謎

仍待解決的問題：
- 暗能量的本質
- 暗物質是什麼
- 暴漲的詳細機制
- 宇宙的最終命運
