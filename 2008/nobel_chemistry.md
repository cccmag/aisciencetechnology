# 化學獎：綠色熒光蛋白

2008 年諾貝爾化學獎授予[下村修](https://zh.wikipedia.org/wiki/下村修)，表彰他發現和開發[綠色熒光蛋白](https://zh.wikipedia.org/wiki/綠色熒光蛋白)（GFP），這項發現為生物學研究帶來革命性工具。

## 獲獎者簡介

### 下村修

[下村修](https://zh.wikipedia.org/wiki/下村修)（Osamu Shimomura）：
- 1928 年出生於日本
- 1960 年獲博士學位
- 伍茲霍爾海洋研究所
- 2008 年獲諾貝爾化學獎

## 綠色熒光蛋白

### 發現歷史

[綠色熒光蛋白](https://zh.wikipedia.org/wiki/綠色熒光蛋白)（GFP）的發現：
- 1962 年：下村修從水母中分離 GFP
- 發現 GFP 的熒光特性
- 1979 年：確定 GFP 結構
- 1992 年：首次用於標記生物

### 蛋白質結構

GFP 的特點：
- 237 個氨基酸
- 自發熒光
- 無需輔因子
- 分子量約 27 kDa

## 科學應用

### 生物學研究

GFP 在生物學中的應用：
| 應用 | 說明 |
|------|------|
| 蛋白質定位 | 追蹤蛋白質在細胞中的位置 |
| 基因表達 | 報告基因 |
| 蛋白質相互作用 | BiFC |
| 細胞追蹤 | 追蹤細胞發育 |

### 轉基因生物

轉基因 GFP 應用：
- GFP 小鼠
- 斑馬魚
- 植物
- 微生物

## 技術價值

### 標記工具

GFP 作為標記工具：
- 可見熒光
- 非破壞性
- 操作簡單
- 適用範圍廣

### 諾貝爾獎後續

馬丁·查爾菲和錢永健因將 GFP 應用於生物研究而獲得 2008 年諾貝爾化學獎。

## 歷史意義

### 生物學革命

GFP 的歷史意義：
- **革命性工具**：生命科學
- **可視化**：看不見的過程可見
- **追蹤**：細胞發育過程
- **諾貝爾獎**：遲來的承認

### 持續影響

長遠的影響：
-熒光蛋白多色系
- 超分辨顯微鏡
- 光遺傳學
- 合成生物學
