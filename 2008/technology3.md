# 雲端運算持續發展：AWS 服務持續擴展

2008 年，[雲端運算](https://zh.wikipedia.org/wiki/雲端運算)市場持續發展，[Amazon Web Services](https://zh.wikipedia.org/wiki/Amazon_Web_Services)（AWS）持續擴展服務範圍，為企業和開發者提供更多雲端解決方案。

## AWS 服務發展

### 主要服務

[AWS](https://zh.wikipedia.org/wiki/Amazon_Web_Services)的核心服務：
| 服務 | 功能 | 發布時間 |
|------|------|----------|
| S3 | 雲端存儲 | 2006 |
| EC2 | 雲端計算 | 2006 |
| SQS | 消息隊列 | 2004 |
| SimpleDB | 資料庫 | 2007 |

### 2008 年新增

2008 年 AWS 新服務：
- 更多實例類型
- 區域擴展
- 管理工具
- 數據庫服務

## 雲端運算概念

### 服務模式

[雲端運算](https://zh.wikipedia.org/wiki/雲端運算)的層次：
| 層次 | 說明 | 例子 |

| IaaS | 基礎設施 | AWS EC2 |
