## 本期編輯語

1947 年是冷戰開始的年份。杜魯門主義的提出、馬歇爾計畫的啟動、東西方集團的對峙，這一切都預示著一個新的時代的來臨。然而，即便在這樣的政治氣候下，科學研究仍然沒有停滯。

這一年，貝爾實驗室正式確認了電晶體的發明，標誌著半導體時代的真正開始。同時，B-47 噴射轟炸機的研發成功，預示著軍事航空的重大變革。這些事件將深刻影響未來數十年的世界格局和科技發展。

讓我們走進 1947 年，看看這個冷戰初期的年份，如何為未來的科技繁榮奠定基礎。
