# 胰島素結構測定：蛋白質結構的重大突破

## 一、胰島素研究的背景

[胰島素](https://zh.wikipedia.org/wiki/胰島素)是由胰臟β細胞分泌的蛋白質激素，是調節血糖的關鍵分子。[1921年](https://zh.wikipedia.org/wiki/1921年)，[弗雷德里克·班廷](https://zh.wikipedia.org/wiki/弗雷德里克·班廷)和[查爾斯·貝斯特](https://zh.wikipedia.org/wiki/查爾斯·貝斯特)首次從胰腺中提取出胰島素，用於治療糖尿病。

## 二、桑格的蛋白質測序研究

[1954年](https://zh.wikipedia.org/wiki/1954年），[弗雷德里克·桑格](https://zh.wikipedia.org/wiki/弗雷德里克·桑格)在蛋白質測序方面取得了重大突破，確定了[胰島素](https://zh.wikipedia.org/wiki/胰島素)的完整胺基酸序列。

[桑格](https://zh.wikipedia.org/wiki/弗雷德里克·桑格)使用的測序方法：
1. **部分水解**：使用酶將胰島素切成較小的肽段
2. **層析分離**：利用層析技術分離不同肽段
3. **末端分析**：確定每個肽段的N端和C端胺基酸
4. **重疊拼接**：通過重疊區域拼接完整的序列

## 三、胰島素的結構

[胰島素](https://zh.wikipedia.org/wiki/胰島素)的結構特點：

1. **雙鏈結構**：由A鏈和B鏈兩條多肽鏈組成
2. **二硫鍵連接**：兩條鏈通過兩個二硫鍵連接
3. **A鏈**：21個胺基酸
4. **B鏈**：30個胺基酸

## 四、歷史意義

[胰島素結構測定](https://zh.wikipedia.org/wiki/胰島素#結構)的成功具有重大意義：

1. **方法創新**：建立了蛋白質測序的標準方法
2. **理解功能**：有助於理解胰島素的作用機制
3. **藥物開發**：為日後重組胰島素的生產奠定基礎

## 五、對後世的影響

[胰島素測序](https://zh.wikipedia.org/wiki/弗雷德里克·桑格)工作對後世产生了深远影响：

1. **蛋白質科學**：推動了蛋白質結構與功能研究
2. **生物技術**：為基因工程和蛋白質工程提供基礎
3. **糖尿病治療**：為更好的胰島素製劑開發創造條件
