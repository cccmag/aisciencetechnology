# 靜電複印機：辦公室自動化

## 一、複印技術的發展

[1950年代初期](https://zh.wikipedia.org/wiki/1950年代初期），辦公室對文件複印的需求日益增長。[切斯特·卡爾森](https://zh.wikipedia.org/wiki/切斯特·卡爾森)在[1938年](https://zh.wikipedia.org/wiki/1938年)發明了靜電複印技術，但這種技術直到1950年代才真正商業化。

## 二、施樂公司的崛起

[1954年](https://zh.wikipedia.org/wiki/1954年），[施樂公司](https://zh.wikipedia.org/wiki/施樂公司)（Haloid Company）開始大規模推廣基於靜電複印技術的複印機。

[施樂公司](https://zh.wikipedia.org/wiki/施樂公司)的發展：
- 採用卡爾森的靜電複印技術
- 改名為Xerox（施樂）
- 成為辦公室設備的領導品牌

## 三、靜電複印原理

[靜電複印](https://zh.wikipedia.org/wiki/靜電複印)的工作原理：

1. **充電**：光導體表面均勻充電
2. **曝光**：將原稿圖像投射到光導體上
3. **顯影**：帶電墨粉附著在圖像區域
4. **轉印**：將墨粉圖像轉印到紙張上
5. **定影**：加熱使墨粉永久附著

## 四、對辦公室的影響

[靜電複印機](https://zh.wikipedia.org/wiki/靜電複印機)的發明對辦公室產生了革命性影響：

1. **效率提升**：快速複印文件
2. **信息傳播**：便於文件分享
3. **行政簡化**：減少人工抄寫工作
4. **知識普及**：促進知識的廣泛傳播

## 五、對未來的影響

[靜電複印技術](https://zh.wikipedia.org/wiki/靜電複印)的發明標誌著辦公室自動化的開始：

1. **辦公室革命**：催生現代辦公室設備
2. **信息時代**：為信息時代的來臨奠定基礎
3. **複製技術**：推動後續數位複印技術的發展
