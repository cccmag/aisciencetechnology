# 尼龍絲襪：工業奇蹟的時尚象徵

## 一、尼龍的發明

1935年，杜邦公司（DuPont）的化學家**華萊士·卡羅瑟斯**（Wallace Carothers）發明了尼龍（Nylon）。這是人類歷史上第一種完全由人工合成的纖維。尼龍的強度、彈性和耐磨性都遠超天然纖維。

1941年，尼龍絲襪開始大規模量產，成為當時最時尚的女性用品。

## 二、尼龍的特性

尼龍具有諸多優異性能：

| 特性 | 描述 |
|------|------|
| 強度 | 比鋼絲還要強韌 |
| 彈性 | 伸長後能恢復原狀 |
| 耐磨 | 使用壽命長 |
| 輕盈 | 密度低 |
| 易洗快乾 | 打理方便 |

## 三、戰時的"尼龍狂熱"

1941年，美國女性為購買尼龍絲襪排起長龍。尼龍絲襪被認為是"比絲綢更優雅，比棉布更舒適"的存在。

二戰期間，尼龍被列為軍需物資，大量用於製作降落傘、繩索和帐篷。民間的尼龍絲襪供應量大幅減少，黑市價格飛漲。據說有人甚至用珍貴的尼龍長襪進行以物易物交易。

## 四、歷史影響

尼龍的發明標誌著合成纖維時代的來臨。此後，聚酯（Polyester）、丙烯酸（Acrylic）等合成纖維相繼問世，徹底改變了紡織工業的面貌。

今天，合成纖維在服裝、產業用紡織品和家用紡織品中的應用無處不在，這一切都始於1935年尼龍的發明和1941年尼龍絲襪的普及。
