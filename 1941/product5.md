# 速溶咖啡：戰爭催生的便利饮品

## 一、速溶咖啡的發明

速溶咖啡的發明可以追溯到1906年，當時日本化學家**佳多·加利**（Kato Gentaro）發明了第一種速溶咖啡工藝。但這種工藝過於複雜，難以工業化推廣。

1930年代，瑞士咖啡進口商**馬蒂斯·塞弗恩**（Martinus Seeger）改進了速溶咖啡的生產工藝。1941年，雀巢公司（Nescafé）開始在美國大規模推廣速溶咖啡。

## 二、生產工藝

1941年的速溶咖啡生產工藝主要包括：

| 工藝步驟 | 描述 |
|----------|------|
| 萃取 | 用熱水從咖啡豆中提取可溶物質 |
| 濃縮 | 低溫減壓濃縮咖啡液 |
| 噴霧乾燥 | 將濃縮液噴成霧狀，熱風乾燥成粉末 |
| 包裝 | 密封保存，即衝即飲 |

## 三、戰時意義

二戰期間，速溶咖啡成為軍隊的重要物資。士兵們可以在前線便捷地飲用熱咖啡，這對提振士氣起到重要作用。同時，速溶咖啡也成為普通民眾日常生活的便利飲品。

## 四、歷史影響

速溶咖啡的普及改變了人類的飲茶（咖啡）習慣：
- 簡化了製作過程
- 延長了保存期限
- 降低了運輸成本
- 使咖啡消費大眾化

戰後，速溶咖啡繼續發展，衍生出咖啡膠囊、冷萃咖啡等新產品形態。
