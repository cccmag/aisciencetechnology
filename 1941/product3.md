# 越野吉普車：戰場上的機械驢

## 一、吉普車的由來

Jeep（吉普）這個名稱的起源有多種說法。一說來自於美國陸軍的代號"GP"（General Purpose，通用目的）。1941年，美國班塔姆汽車公司（Bantam）、威利斯-奧夫蘭多汽車公司（Willys-Overland）和福特汽車公司（Ford）開始為美軍生產這種輕型越野車。

同年，美國陸軍開始裝備Willys MB吉普車，這款車成為二戰中最具標誌性的軍用車輛之一。

## 二、技術特點

吉普車的設計簡潔實用：

| 配置 | 說明 |
|------|------|
| 引擎 | 2.2升四缸發動機 |
| 功率 | 60馬力 |
| 驅動 | 四輪驅動 |
| 重量 | 約1噸 |
| 極速 | 約105公里/小時 |

吉普車採用簡單的框架式車身、配備越野胎、可拆式軟頂，必要時甚至可以無頂駕駛。

## 三、戰場應用

吉普車在二戰中發揮了重要作用：
- 偵察巡邏
- 人員運輸
- 物資運輸
- 拖曳火炮
- 戰地通信

它被士兵們親切地稱為"工作馬"或"機械驢"，成為盟軍勝利的象徵之一。

## 四、歷史影響

吉普車開創了輕型越野車的先河，啟發了日後悍馬（HMMWV）、路虎（Land Rover）等越野車的設計。今天的吉普牧馬人（Jeep Wrangler）仍保留著1941年吉普車的設計元素。
