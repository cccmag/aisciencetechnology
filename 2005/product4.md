# 智慧手機萌芽：iPhone 前的時代

2005 年是[智慧手機](https://zh.wikipedia.org/wiki/智慧型手機)爆發前夜。[Symbian](https://zh.wikipedia.org/wiki/Symbian)、[Windows Mobile](https://zh.wikipedia.org/wiki/Windows_Mobile)和 [BlackBerry](https://zh.wikipedia.org/wiki/BlackBerry) 持續演化，PDA 與手機的功能持續融合。iPhone 在兩年後的 2007 年發布，將徹底改變這個行業。

## 2005 年的智慧手機市場

### 主要平台

2005 年智慧手機作業系統：
| 平台 | 廠商 | 特點 |
|------|------|------|
| Symbian | Nokia | 市場領導者 |
| Windows Mobile | 多廠商 | PC 整合 |
| BlackBerry | RIM | 企業市場 |
| Palm OS | Palm | 簡單易用 |

### 市場份額

2005 年智慧手機市場：
- Nokia  Symbian：約 50% 市場份額
- BlackBerry：約 20%
- Windows Mobile：約 15%
- 其他：15%

## 主要產品

### Nokia N 系列

2005 年 Nokia 的旗艦：
- **N70**：N 系列第一款
- **N90**：可旋轉螢幕設計
- **N91**：內建硬碟，音樂手機
- **N93**：光學變焦

### Windows Mobile

2005 年 Windows Mobile 設備：
- **HTC TyTN**：側滑式 QWERTY
- **Palm Treo 650**：QWERTY 鍵盤
- **Mio A700**：GPS 整合

## 功能特點

### 與現在智慧手機的比較

| 功能 | 2005 年 | 現在 |
|------|---------|------|
| 觸控 | 較少，電阻式 | 電容式多點觸控 |
| 網路 | 2G/EDGE | 4G/5G |
| 應用 | 有限 | 百萬應用 |
| 相機 | 1-3 MP | 數十到百 MP |
| GPS | 外接或罕見 | 內建 |

### 當時的技術限制

智慧手機的局限：
- 應用程式匱乏
- 網路速度慢
- 使用不方便
- 價格昂貴

## iPhone 前的準備

### 技術準備

為 iPhone 奠基的技術：
- **觸控螢幕**：多點觸控技術成熟
- **行動網絡**：3G 網絡普及
- **軟體生態**：App Store 概念萌芽
- **用戶習慣**：智慧手機用戶增加

### 市場準備

為 iPhone 奠基的市場：
- 用戶對智慧手機的認知提高
- 對更好體驗的期待
- 行動網絡需求增長
- 應用下載需求出現

## 歷史意義

智慧手機萌芽的歷史意義：
- **技術積累**：為革命性產品奠基
- **用戶教育**：培養智慧手機使用習慣
- **市場培育**：創造智慧手機需求
- **平台之爭**：為 iPhone 和 Android 預熱
