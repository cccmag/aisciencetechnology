# Web 2.0 時代全面來臨

2005 年，[Web 2.0](https://zh.wikipedia.org/wiki/Web_2.0) 概念全面流行，[用戶生成內容](https://zh.wikipedia.org/wiki/用戶生成內容)（UGC）和[社交媒體](https://zh.wikipedia.org/wiki/社交媒體)開始爆發。這一年被認為是 Web 2.0 時代的元年，互聯網從「閱讀」為主轉變為「閱讀+創作」為主。

## Web 2.0 的概念

### 什麼是 Web 2.0

[Web 2.0](https://zh.wikipedia.org/wiki/Web_2.0) 的核心特點：
- **用戶生成內容**：任何人都可以創作和發布
- **社交互動**：用戶之間的連接和互動
- **開放分享**：知識和信息共享
- **平台效應**：網絡效應創造價值

### 與 Web 1.0 的對比

| 方面 | Web 1.0 | Web 2.0 |
|------|----------|---------|
| 主體 | 機構內容 | 用戶內容 |
| 互動 | 有限 | 豐富 |
| 參與 | 閱讀為主 | 閱讀+創作 |
| 技術 | 靜態網頁 | 動態應用 |

## 2005 年的主要平台

### 社交媒體

2005 年的主要社交媒體：
| 平台 | 類型 | 用戶規模 |
|------|------|----------|
| Facebook | 社交網絡 | 開始普及 |
| YouTube | 視頻分享 | 迅速增長 |
| MySpace | 社交網絡 | 快速增長 |
| Flickr | 照片分享 | 持續增長 |

### 部落格和 Wiki

用戶生成內容的平台：
- **WordPress**：部落格平台興起
- **Blogger**：Google 旗下部落格服務
- **維基百科**：知識共享的代表
- **RSS**：內容訂閱技術普及

## 用戶生成內容

### UGC 的定義

[用戶生成內容](https://zh.wikipedia.org/wiki/用戶生成內容)（User-Generated Content）的類型：
- **文字**：部落格、論壇、評論
- **圖片**：照片、表情包、截圖
- **視頻**：自拍、教程、遊戲
- **音頻**：播客、音樂

### 創作門檻降低

Web 2.0 降低的創作障礙：
- **工具簡單**：無需專業技能
- **發布容易**：一鍵發布
- **成本為零**：免費使用
- **傳播廣泛**：互聯網分享

## 對媒體的影響

### 媒體形態變化

 Web 2.0 對媒體的影響：
| 方面 | 傳統媒體 | Web 2.0 媒體 |
|------|----------|-------------|
| 話語權 | 機構掌控 | 個人也有話語權 |
| 發布門檻 | 高 | 低 |
| 反饋機制 | 延遲 | 即時 |
| 覆蓋範圍 | 有限 | 全球 |

### 公民新聞

 Web 2.0 催生的公民新聞：
- **突發事件**：普通人記錄新聞
- **監督作用**：對權力的監督
- **多元視角**：不同角度的報道
- **事實核查**：公眾參與事實核查

## 社交媒體的興起

### 社交圖譜

[社交媒體](https://zh.wikipedia.org/wiki/社交媒體)建立的連接：
- **朋友關係**：與朋友和家人保持聯繫
- **興趣小組**：加入志同道合的群體
- **關注機制**：關注名人、品牌
- **位置分享**：分享所在位置

### 社交化電商

社交媒體與商業的結合：
- **口碑傳播**：通過社交分享推廣
- **社交電商**：直接在社交媒體購買
- **網紅行銷**：影響者行銷
- **社群團購**：組團購買

## 歷史意義

Web 2.0 時代的歷史意義：
- **話語權民主化**：任何人都有發聲平台
- **內容創作平民化**：創作不再是少數人的特權
- **社交互聯網**：人與人的連接
- **平台經濟**：建立在大規模用戶參與基礎上的商業模式
