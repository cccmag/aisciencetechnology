# 大數據分析技術成熟

## 引言

西元 2013 年，[大數據](https://zh.wikipedia.org/wiki/大數據)分析技術日益成熟，各種開源和商業工具開始在企業中廣泛部署。

## 主要技術框架

### Apache Hadoop
[Hadoop](https://zh.wikipedia.org/wiki/Apache_Hadoop) 生態系統持續完善：

- **HDFS**：分散式檔案系統
- **MapReduce**：分散式計算框架
- **Hive**：數據倉庫
- **HBase**：NoSQL 資料庫

### Apache Spark
[Spark](https://zh.wikipedia.org/wiki/Apache_Spark) 作為 Hadoop 的補充，開始獲得廣泛關注，提供更快的記憶體內運算。

### NoSQL 資料庫
MongoDB、Cassandra、Redis 等 NoSQL 資料庫成為處理多樣化數據的主流選擇。

## 雲端大數據服務

主要雲端提供商開始推出托管大數據服務：

### AWS EMR
Amazon Web Services 的彈性 MapReduce（EMR）服務，簡化了 Hadoop 集群的部署。

### Azure HDInsight
微軟的 Azure 平台提供托管 Hadoop 和 Spark 服務。

## 數據科學家

大數據時代催生了新的職業——[數據科學家](https://zh.wikipedia.org/wiki/數據科學家)：

- 結合統計學、程式設計和領域知識
- 從海量數據中提取洞見
- 機器學習和預測分析的專家
