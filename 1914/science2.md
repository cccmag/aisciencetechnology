# X射線晶體學的發展：揭示物質微觀結構

## 概述

1912 年，[馬克斯·馮·勞厄](https://zh.wikipedia.org/wiki/馬克斯·馮·勞厄)在慕尼黑大學發現了 X 射線通過晶體時的衍射現象。1914 年，勞厄因此發現榮獲諾貝爾物理學獎。這一發現開創了[X射線晶體學](https://zh.wikipedia.org/wiki/X射線晶體學)這一新學科。

## 一、勞厄的發現

### 1.1 實驗設計

1912 年，勞厄提出一個大膽的假設：如果 X 射線是一種電磁波，而且波長與晶體中原子間距相當，那麼晶體應該能夠作為 X 射線的天然衍射光柵。

### 1.2 實驗驗證

在研究生弗里德里希和尼平的協助下，勞厄用硫酸銅晶體進行實驗，成功觀察到了 X 射線的衍射圖案。

### 1.3 諾貝爾獎

1914 年，勞厄因其 X 射線晶體衍射的發現榮獲諾貝爾物理學獎。

## 二、布拉格父子的貢獻

### 2.1 布拉格定律

[威廉·亨利·布拉格](https://zh.wikipedia.org/wiki/威廉·亨利·布拉格)和[威廉·勞倫斯·布拉格](https://zh.wikipedia.org/wiki/威廉·勞倫斯·布拉格)父子在 1912-1913 年發展了 X 射線晶體結構分析的方法。

$$n\lambda = 2d\sin\theta$$

這個公式被稱為[布拉格定律](https://zh.wikipedia.org/wiki/布拉格定律)，是 X 射線晶體學的基礎。

### 2.2 晶體結構分析

1913 年，布拉格父子用 X 射線衍射方法確定了氯化鈉的晶體結構。

### 2.3 父子共獲諾貝爾獎

1915 年，布拉格父子共同獲得諾貝爾物理學獎，這在諾貝爾獎歷史上是極為罕見的。

## 三、1914 年的發展

### 3.1 研究隊伍擴大

1914 年，越來越多的實驗室開始從事 X 射線晶體學研究。

### 3.2 新晶體結構

科學家們確定了越來越多的晶體結構，包括金屬、鹽類和礦物。

### 3.3 理論完善

X 射線晶體學的理論基礎不斷完善。

## 四、科學意義

### 4.1 微觀結構的直接觀察

X 射線晶體學使人類能夠「看到」原子在晶體中的排列方式。

### 4.2 化學結構的確定

X 射線晶體學為確定分子的三維結構提供了強有力的工具。

### 4.3 跨學科應用

X 射線晶體學迅速應用於物理學、化學、礦物學、生物學等多個學科。

## 五、歷史影響

### 5.1 晶體學的建立

X 射線晶體學的建立標誌著現代晶體學的開始。

### 5.2 DNA 結構的揭示

1953 年，沃森和克里克正是基於 X 射線晶體學數據，才成功確定了 DNA 的雙螺旋結構。

### 5.3 結構生物學的興起

X 射線晶體學在蛋白質和其他生物大分子的結構研究中發揮了關鍵作用。
