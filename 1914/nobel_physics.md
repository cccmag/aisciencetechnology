# 物理學獎：勞厄與X射線衍射

## 得獎人

**[馬克斯·馮·勞厄](https://zh.wikipedia.org/wiki/馬克斯·馮·勞厄)**（Max von Laue，1879-1960）
德國物理學家，慕尼黑大學教授

## 獲獎原因

「表彰他發現了X射線通過晶體時的衍射現象」

## 科學生涯

### 早年經歷

勞厄 1879 年出生於德國法爾肯費爾德，在斯特拉斯堡和慕尼黑學習物理。1912 年成為慕尼黑大學教授。

### 研究方向

勞厄的研究方向包括熱力學、光學和 X 射線物理學。他對晶體的光學性質有深入研究。

## X射線衍射的發現

### 1912 年的突破

1912 年，勞厄提出了一個大膽的假設：如果 X 射線是一種電磁波，而且波長與晶體中原子間距相當（約 1  Å），那麼晶體應該能夠作為 X 射線的天然衍射光柵。

### 實驗驗證

在研究生弗里德里希（Friedrich）和尼平（Knipping）的協助下，勞厄用硫酸銅（CuSO₄）晶體進行實驗。實驗結果超出了所有人的預期：不僅觀察到了 X 射線的衍射，而且形成了清晰的衍射斑點圖案。

### 歷史意義

這一發現證明瞭 X 射線是一種波動，為 X 射線晶體學的建立奠定了基礎。

## 物理學的影響

### X射線晶體學的建立

勞厄的發現開創了[X射線晶體學](https://zh.wikipedia.org/wiki/X射線晶體學)這一新學科。這項技術使人類能夠「看到」晶體中原子的排列方式。

### 晶體結構分析

X 射線晶體學迅速成為確定分子和晶體結構的強有力工具。

### 諾貝爾獎

1914 年，勞厄因其這一重要發現榮獲諾貝爾物理學獎。

## 後續發展

### 布拉格定律

1913 年，[布拉格父子](https://zh.wikipedia.org/wiki/布拉格定律)提出了著名的布拉格定律，為 X 射線晶體學提供了數學工具。

### 結構化學

X 射線晶體學的發展促進了結構化學的建立，使人們能夠理解分子的三維結構。

### 生物學應用

後來，X 射線晶體學被用於確定 DNA 和蛋白質的結構，開創了結構生物學。

## 歷史意義

勞厄發現 X 射線衍射是物理學史上的一個重要里程碑。這一發現不僅解決了 X 射線本質的問題，還為整個晶體學和結構化學的發展開闢了新途徑。
