# 火星地震監測：洞察號登陸火星

## 概述

2018年5月5日，NASA的洞察號（InSight）探測器發射升空，經過為期六個月的星際旅行，於11月26日在火星表面成功著陸。洞察號是人類首個專門用於研究火星內部結構的探測器，攜帶了地震儀等儀器，試圖「聆聽」這顆紅色星球的「心跳」。

![洞察號著陸示意圖](https://upload.wikimedia.org/wikipedia/commons/thumb/8/8a/Mars_InSight_illustration_2018.jpg/800px-Mars_InSight_illustration_2018.jpg)

## 一、任務概況

### 1.1 任務目標

洞察號的科學目標：

| 目標 | 說明 |
|------|------|
| 了解火星內部結構 | 研究火星殼、地幔、地核的分層 |
| 測量地熱流 | 了解火星的熱演化 |
| 監測火星地震 | 記錄火星上的「火星地震」 |
| 研究隕石撞擊 | 記錄撞擊事件 |

### 1.2 著陸地點

洞察號降落在了火星的「埃律西昂平原」（Elysium Planitia）：
- 這裡被科學家稱為「火星上最大的停車場」
- 地形平坦，適合放置儀器
- 陽光充足，適合太陽能發電

### 1.3 著陸過程

洞察號的著陸過程被稱為「恐怖七分鐘」：
- 進入大氣層：隔熱盾保護
- 減速傘打開：進一步減速
- 拋棄隔熱罩：露出雷達
- 反推火箭點火：最後減速
- 懸停著陸：軟著陸

## 二、科學儀器

### 2.1 地震儀（SEIS）

洞察號搭載的最重要儀器：

| 項目 | 規格 |
|------|------|
| 名稱 | 內部結構地震實驗（SEIS） |
| 設計 | 由法國航天局（CNES）提供 |
| 靈敏度 | 可探測比原子直徑還小的移動 |
| 放置 | 需要機械臂放置在地面 |

SEIS的主要任務是測量火星地震波，這些波可以揭示火星內部的結構。

### 2.2 熱流探測儀（HP3）

熱流探測儀用於測量火星內部的熱量傳遞：
- 由德國航天局（DLR）提供
- 將在火星地表下5米處測量溫度
- 幫助了解火星的熱歷史

### 2.3 旋轉和內部結構實驗（RISE）

RISE儀器通過通信跟蹤火星的自轉：
- 測量火星旋轉的微小變化
- 推斷火星內核的大小和狀態

## 三、火星地震的發現

### 3.1 首次探測

2019年4月6日，洞察號記錄到了首次確認的火星地震：
- 信號被命名為「Sol 128」
- 這是自1960年代以來首次在火星上探測到地震

### 3.2 火星地震的特點

與地球地震的比較：

| 特性 | 火星地震 | 地球地震 |
|------|----------|----------|
| 頻率 | 以低頻為主 | 頻率範圍廣 |
| 強度 | 較弱 | 差異大 |
| 原因 | 隕石撞擊、冷卻收縮 | 板塊運動 |

### 3.3 科學意義

火星地震數據的價值：
- 確定火星內部結構
- 了解火星是否仍有地質活動
- 幫助理解類地行星的形成

## 四、火星地質結構

### 4.1 火星的內部結構

根據現有數據繪製的火星內部結構：

| 層次 | 深度 | 特點 |
|------|------|------|
| 地殼 | 0-50公里 | 厚度不均，磁性殘留 |
| 地幔 | 50-1500公里 | 固體岩石 |
| 外核 | 1500-3000公里 | 液態鐵合金 |
| 內核 | 3000-6800公里 | 固態鐵 |

### 4.2 與地球的比較

火星與地球的結構差異：

| 特性 | 火星 | 地球 |
|------|------|------|
| 地核大小 | 較大（半徑~1800km） | 半徑~3400km |
| 狀態 | 仍未確定 | 外核液態、內核固態 |
| 磁場 | 微弱 | 強大 |
| 地質活動 | 微弱 | 活躍 |

### 4.3 科學發現

洞察號已經獲得的重要發現：
- 火星地殼厚度約20-70公里
- 火星內核可能部分液態
- 火星內部的地震活動比預期少

## 五、對行星科學的影響

### 5.1 比較行星學

洞察號數據對比較行星學的貢獻：

| 研究領域 | 洞察號的作用 |
|----------|--------------|
| 類地行星演化 | 理解地球、火星、金星差異 |
| 行星形成 | 了解行星形成的普遍規律 |
| 生命存在條件 | 評估火星過去或現在是否存在生命 |

### 5.2 未來探測

洞察號為未來火星探測奠定基礎：
- 驗證火星著陸技術
- 建立火星天氣數據庫
- 為樣本返回做準備

### 5.3 國際合作

洞察號任務展示了國際合作的力量：
- NASA負責發射和著陸
- CNES提供地震儀
- DLR提供熱流儀
- 多國科學家共同分析數據
