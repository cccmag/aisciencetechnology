# 折疊螢幕手機的興起

## 概述

2018年，折疊螢幕手機從概念走向現實。柔宇科技發布了全球首款消費級折疊手機柔派（FlexPai），三星、華為等大廠也展示了自家的折疊螢幕原型機。這項柔性顯示技術的商業化，預示著智慧手機形態的革命性變化。

![折疊螢幕示意圖](https://upload.wikimedia.org/wikipedia/commons/thumb/e/eb/Samsung_Galaxy_Fold.jpg/800px-Samsung_Galaxy_Fold.jpg)

## 一、柔性顯示技術

### 1.1 OLED的優勢

柔性顯示的技術基礎是有機發光二極體（OLED）：

| 特性 | 說明 |
|------|------|
| 自發光 | 無需背光，螢幕可以更薄 |
| 可撓性 | 可以在柔性基板上製造 |
| 高對比度 | 完美的黑色顯示 |
| 廣視角 | 各個角度看起來一致 |

### 1.2 柔性基板

柔性螢幕使用的材料：
- **塑膠基板**：如聚醯亞胺（PI）
- **金屬箔**：可折疊但成本高
- **超薄玻璃**：結合玻璃穩定性和柔性

### 1.3 彎曲 vs 折疊

柔性螢幕的彎曲程度：

| 類型 | 彎曲程度 | 應用 |
|------|----------|------|
| 曲面螢幕 | 輕度彎曲 | 電視、手機邊緣 |
| 可撓式 | 中度彎曲 | 概念設備 |
| 可折疊式 | 180度折疊 | 折疊手機 |

## 二、主要產品

### 2.1 柔宇 FlexPai

2018年10月，柔宇科技發布了首款消費級折疊手機：

| 參數 | 規格 |
|------|------|
| 螢幕尺寸 | 7.8英寸（展開）/4.3英寸（折疊） |
| 解析度 | 1920×1440 |
| 售價 | 約8999元起 |
| 特點 | 向外折疊，蟬翼螢幕 |

### 2.2 三星 Infinity Flex

三星在2018年11月的開發者大會上展示了折疊螢幕原型：
- 向內折疊
- 外部有第二螢幕用於折疊狀態
- 2019年正式發布Galaxy Fold

### 2.3 華為 Mate X

華為的折疊手機方案：
- 向外折疊
- 5G支持
- 鉸鏈設計

## 三、技術挑戰

### 3.1 鉸鏈設計

折疊機構的設計難題：
- 多次折疊後的耐久性
- 折疊處的平整度
- 整機厚度控制

### 3.2 螢幕保護

折疊區域的保護：
- 超硬塗層保護螢幕
- 防刮傷處理
- 折疊次數測試

### 3.3 良率問題

折疊螢幕的量產難題：
- 初期良率低
- 生產成本高
- 產能受限

## 四、應用場景

### 4.1 手機-平板合一

折疊手機帶來的新體驗：
- 折疊時是手機，便於攜帶
- 展開後是平板，適合觀看視頻、辦公
- 多工處理，提高效率

### 4.2 軟體適配

折疊螢幕的軟體支援：
- 谷歌的折疊螢幕API
- 應用在大/小螢幕間無縫切換
- 多視窗功能的支援

### 4.3 價格定位

折疊手機的價格：
- Galaxy Fold：約15000元
- 華為 Mate X：約17000元
- 初期定位超高端市場

## 五、未來展望

### 5.1 技術進步

折疊螢幕技術的進步方向：
- 更輕薄的機身
- 更耐久的鉸鏈
- 更便宜的價格
- 多種折疊形態

### 5.2 市場預測

折疊手機市場的預測：
| 年份 | 全球出貨量預測 |
|------|----------------|
| 2019 | 約50萬部 |
| 2020 | 約400萬部 |
| 2022 | 預計1800萬部 |

### 5.3 產品演進

折疊手機的演進方向：
- 卷軸式手機
- 多折疊設計
- 折疊平板
- 折疊筆電
