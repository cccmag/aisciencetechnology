# 小行星採樣返回：隼鳥2號與OSIRIS-REx

## 概述

2018年是人類小行星探測史上重要的一年。日本的隼鳥2號（Hayabusa2）和美國的OSIRIS-REx兩艘小行星探測器，分別對龍宮小行星和 Bennu 小行星進行了採樣工作。這些珍貴的樣本將幫助科學家研究太陽系的起源和生命的起源。

![隼鳥2號](https://upload.wikimedia.org/wikipedia/commons/thumb/1/1e/Hayabusa2_artist_concept.png/800px-Hayabusa2_artist_concept.png)

## 一、隼鳥2號任務

### 1.1 任務概況

隼鳥2號是日本宇宙航空研究開發機構（JAXA）的小行星探測器：

| 項目 | 參數 |
|------|------|
| 發射時間 | 2014年12月 |
| 目標小行星 | 龍宮（162173 Ryugu） |
| 採樣時間 | 2018年2-10月 |
| 返回時間 | 2020年12月 |
| 採樣量 | 約5.4克 |

### 1.2 龍宮小行星

龍宮是一顆近地小行星：
- 直徑約900米
- 旋轉周期約7.6小時
- 表面覆蓋著碎石

### 1.3 採樣過程

隼鳥2號的採樣方式：

| 步驟 | 操作 |
|------|------|
| 接近 | 緩慢下降至小行星表面 |
| 發射彈丸 | 擊中表面，揚起碎石 |
| 採集 | 收集彈出的物質 |
| 撤離 | 上升返回軌道 |

隼鳥2號進行了三次採樣：
- 第一次：表面樣本
- 第二次：表面樣本
- 第三次：使用撞擊器製造的人工隕石坑内部樣本

## 二、OSIRIS-REx任務

### 2.1 任務概況

OSIRIS-REx是NASA的小行星採樣返回任務：

| 項目 | 參數 |
|------|------|
| 發射時間 | 2016年9月 |
| 目標小行星 | 貝努（101955 Bennu） |
| 採樣時間 | 2018年10月-2020年10月 |
| 預計返回 | 2023年9月 |
| 採樣量 | 60克-2公斤 |

### 2.2 貝努小行星

貝努是一顆有潛在威脅的近地小行星：
- 直徑約490米
- 富含碳元素
- 可能含有有機物和水

### 2.3 觸碰-and-離開

OSIRIS-REx使用了「觸碰-and-離開」（TAG）方式：
1. 緩慢下降至小行星表面
2. 接觸表面，噴射氮氣
3. 揚起的顆粒被收集
4. 快速撤離

## 三、中國的小行星探測

### 3.1 發展規劃

中國也在積極發展小行星探測能力：
- 規劃中的小行星探測任務
- 计划採樣返回多個小行星
- 瞄準資源開發和科學研究

### 3.2 技術積累

中國小行星探測的技術基礎：
- 「嫦娥」月球探測任務
- 「天問」火星探測任務
- 返回式衛星技術

## 四、科學價值

### 4.1 太陽系起源

小行星樣本的科學價值：
- 記錄了太陽系形成之初的物質
- 幫助理解行星形成過程
- 揭示水的來源和有機物的分布

### 4.2 生命起源

與生命起源的關聯：
- 小行星可能攜帶了地球早期水和有機物
- 研究隕石中的氨基酸
- 探索生命的宇宙起源

### 4.3 行星防御

近地小行星研究的實際意義：
- 評估貝努的撞擊風險
- 發展行星防御技術
- 為未來偏轉任務做準備

## 五、歷史意義

### 5.1 日本的成就

隼鳥2號延續了日本在小行星探測領域的領先地位：
- 隼鳥1號：首個小行星樣本返回（2010年）
- 隼鳥2號：更高難度的採樣

### 5.2 美國的實力

OSIRIS-REx展示了NASA的技術實力：
- 先進的導航和控制系統
- 精確的採樣技術
- 長期任務規劃能力

### 5.3 人類探索的腳步

這些任務標誌著人類深空探測能力的進步：
- 小行星採樣返回
- 載人登陸小行星的可行性
- 未來資源利用
