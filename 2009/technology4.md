# HTML5 標準化：Web 技術持續進步

2009 年，[HTML5](https://zh.wikipedia.org/wiki/HTML5) 標準化工作持續推進，Web 技術邁向新的里程碑，為更豐富的 Web 應用奠定基礎。

## HTML5 發展

### 標準化進程

[HTML5](https://zh.wikipedia.org/wiki/HTML5) 的標準化：
- WHATWG 和 W3C 同時推進
- 2008 年：首個草案
- 2009 年：持續開發
- 2014 年：正式標準

### 核心特性

HTML5 的新特性：
| 特性 | 說明 |
|------|------|
| 視頻/音頻 | 原生支持 |
| Canvas | 2D 繪圖 |
| 離線存儲 | 應用緩存 |
| 地理定位 | API |

## 歷史意義

### Web 標準

HTML5 的歷史意義：
- **跨平台**：一次開發多平台
- **富應用**：桌面和移動
- **離線應用**：無需網絡
- **標準化**：開放網絡
