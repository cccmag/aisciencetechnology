# 雲端運算爆發：AWS 服務快速擴展

2009 年是[雲端運算](https://zh.wikipedia.org/wiki/雲端運算)爆發增長年。Amazon Web Services (AWS) 服務快速擴展，Google App Engine 持續發展，雲端運算成為 IT 產業的主流。

## AWS 發展

### 主要服務

[AWS](https://zh.wikipedia.org/wiki/Amazon_Web_Services) 2009 年服務：
| 服務 | 類型 | 功能 |
|------|------|------|
| EC2 | IaaS | 雲端伺服器 |
| S3 | 存儲 | 雲端存儲 |
| RDS | 資料庫 | 關聯式資料庫 |
| VPC | 網路 | 虛擬私有雲 |

### 市場地位

AWS 的市場領導：
- 雲端運算先驅
- 持續創新
- 價格下降
- 企業採用
