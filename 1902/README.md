### 本年度科學

1. [洛倫茲與洛倫茲變換：相對論的理論先驅](science1.md) -- 洛倫茲提出著名變換方程，為愛因斯坦相對論奠定基礎
2. [塞曼效應：光譜線分裂的神秘現象](science2.md) -- 彼得·塞曼發現光譜線在磁場中的分裂現象
3. [催化理論：化學反應的加速器](science3.md) -- 威廉·奧斯特瓦爾德系統性研究催化理論，為化學工業奠定基礎
4. [齊柏林飛艇：空中旅行的新紀元](science4.md) -- 斐迪南·馮·齊柏林伯爵設計建造人類史上第一艘實用化大型飛船
5. [氬原子量之謎：同位素發現的前奏](science5.md) -- 瑞利勳爵研究氬原子量，發現惰性氣體的存在異常

### 本年度技術

1. [福特 A 型車：汽車大眾化的先驅](technology1.md) -- 亨利·福特設計出可靠且價格親民的福特 A 型車
2. [電話交換機自動化：通信網絡的升級](technology2.md) -- 自動電話交換技術開始推廣，改變人類通信方式
3. [電氣鐵路：城市交通的革命](technology3.md) -- 電氣化鐵路技術成熟，開始在全球大規模推廣
4. [冷藏火車：長途食品運輸的突破](technology4.md) -- 冷藏火車技術成熟，徹底改變食品保存和運輸方式
5. [水下隧道：工程技術的奇蹟](technology5.md) -- 多項水下隧道工程正在進行，代表人類工程技術的巔峰
