## 重要歷史背景

1920 年是第一次世界大戰結束後的第一個完整年份。這一年，世界各國開始從戰爭的創傷中恢復，科學技術事業也開始重建。諾貝爾獎在這一年頒發了物理學獎和化學獎，但生理學獎因故未頒發。

這一年的科學發展體現了戰後科技復蘇的特點：基礎研究繼續深入，測量技術不斷提高，材料科學開始興起。這些發展為後世的科技進步奠定了重要基礎。
