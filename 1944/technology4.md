# V-2火箭的實戰部署

## 一、V-2火箭的發展歷程

V-2火箭（V-2 Rocket）是德國在二戰期間研製的世界上第一種實用彈道導彈。它的研製工作始於1936年，由沃爾特·多恩伯格（Walter Dornberger）和沃納·馮·布勞恩（Wernher von Braun）領導。

1944年，V-2火箭終於開始實戰部署。

## 二、1944年的部署情況

### 首次實戰使用

1944年9月8日，德國從荷蘭發射了第一枚V-2火箭，攻擊了巴黎。這是人類歷史上第一次使用彈道導彈。

### 攻擊統計

截至1945年3月，德國共發射了約3000枚V-2火箭：

| 目標 | 攻擊次數 |
|------|----------|
| 倫敦 | 約1400枚 |
| 安特衛普 | 約1200枚 |
| 其他目標 | 約400枚 |

## 三、技術特點

V-2火箭是當時最先進的武器系統：

| 參數 | 數值 |
|------|------|
| 全長 | 14米 |
| 直徑 | 1.65米 |
| 起飛重量 | 12,800公斤 |
| 推力 | 27,000公斤力 |
| 最大射程 | 320公里 |
| 最大射高 | 100公里 |
| 戰鬥部 | 高爆彈頭，約1000公斤 |

## 四、對戰爭的影響

V-2火箭的實戰使用雖然沒有改變戰爭的結果，但帶來了深刻的影響：

1. **心理影響**：對平民造成恐慌
2. **技術影響**：證明了導彈武器的可行性
3. **戰後影響**：大量技術人員和設備被戰勝國俘獲

## 五、歷史意義：通往太空的階梯

V-2火箭的技術遺產影響深遠：

| V-2技術 | 戰後應用 |
|----------|----------|
| 液體燃料火箭 | 美蘇太空競爭 |
| 慣性導航 | 洲際導彈 |
| 結構材料 | 飛機和飛船 |
| 研製團隊 | 美蘇火箭專家 |

戰後，馮·布勞恩和團隊被美國俘虜，他們的技術直接促成了美國太空計劃的發展。1958年，美國發射了第一顆人造衛星"探索者1號"；1969年，人類首次登月。這一切，都始於V-2火箭。
