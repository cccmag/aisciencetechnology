# 維生素B12的發現

## 一、惡性貧血的歷史

惡性貧血（Pernicious Anemia）是一種嚴重的貧血病，過去被認為是不治之症。1926年，美國醫生喬治·米諾特（George Minot）和威廉·墨菲（William Murphy）發現肝臟可以治療惡性貧血，他們因此獲得1934年諾貝爾獎。

但肝臟中的有效成分究竟是什麼？這個問題困擾了科學家多年。

## 二、1948年的發現（提前說明）

維生素B12（又稱鈷胺素）於1948年被美國和英國的科學家分別發現。但這項發現的工作在1944年就已經開始了。

## 三、1944年的研究進展

### 濃縮肝臟提取物

1944年，科學家開始系統地濃縮肝臟提取物中的有效成分。他們使用各種化學方法，希望能分離出這種神秘的物質。

### 放射性標記研究

1944年，科學家使用放射性鈷來追蹤肝臟中的有效成分。這些實驗為日後維生素B12的發現奠定了基礎。

## 四、維生素B12的特性

維生素B12具有以下特性：

| 特性 | 描述 |
|------|------|
| 化學結構 | 含鈷的咕啉化合物 |
| 分子量 | 1355 |
| 穩定性 | 在酸性環境中穩定 |
| 缺乏症 | 惡性貧血、神經系統損傷 |

## 五、歷史意義

維生素B12的發現是營養學史上的重要里程碑。它不仅揭示了恶性贫血的病因，也为日后的营养学研究提供了重要方法。
