## 本期編輯語

1948 年是半導體歷史上重要的一年。貝爾實驗室正式對外展示了電晶體，標誌著這項發明開始走向世界。這一年，布萊克特因為在宇宙射線和雲霧室研究方面的貢獻，獲得了諾貝爾物理學獎。

本期雜誌將帶您回顧 1948 年的科學發現、技術進展和重要產品，以及這些事件如何連接過去與未來。
