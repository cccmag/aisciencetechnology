## 重要年份標誌

1928 年是醫學史上具有里程碑意義的一年。英國微生物學家亞歷山大·弗萊明在實驗室中意外發現了青黴素，這一發現開創了抗生素時代，拯救了數以百萬計的生命。這是人類對抗細菌感染的重大突破。
